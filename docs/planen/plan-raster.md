# Das Plan-Raster: Ihre interaktive Zeit-Matrix

Das Herz der täglichen Arbeit: eine leistungsfähige Zeit-Matrix, in der Zeilen für Mitarbeitende oder Kundschaft stehen und Spalten für die Tage.

## So funktioniert es

- **Drag & Drop:** Schichten verschieben Sie direkt mit der Maus von einer Person zur nächsten; über den Ausfüllgriff lässt sich ein Eintrag zusätzlich auf weitere Tage derselben Zeile kopieren.
- **Echtzeit:** Änderungen erscheinen sofort bei allen, die denselben Plan geöffnet haben. Zwei Planende können gleichzeitig arbeiten — Konflikte werden erkannt, statt Änderungen unbemerkt zu überschreiben.
- **Alles im Blick:** Qualifikationen und Sperren werden direkt im Raster markiert, Konflikte meldet Klacks zusätzlich in einer eigenen Übersicht — Sie sehen sofort, wenn eine Zuordnung nicht zulässig ist.
- **Schnell:** Das Raster ist für grosse Pläne gebaut; auch hunderte Mitarbeitende über einen ganzen Monat bleiben flüssig bedienbar.

## Gut zu wissen

- Zellen ausserhalb der Gruppenzugehörigkeit oder des Vertragszeitraums einer Person sind ausgegraut und nicht beplanbar — so entstehen keine versehentlichen Einsätze.
- Das Raster und die Auto-Planung arbeiten zusammen: erst automatisch füllen lassen, dann von Hand feinjustieren.
- Je nach Freigabestufe der jeweiligen Einträge ist die Bearbeitung gesperrt — siehe [Freigabe-Workflow: Vom Entwurf zum verbindlichen Plan](../planen/freigabe-workflow.md).
