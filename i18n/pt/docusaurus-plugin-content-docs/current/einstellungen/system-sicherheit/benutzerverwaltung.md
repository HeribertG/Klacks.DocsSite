# Gestão de utilizadores

Em Definições > Gestão de utilizadores, gere as **contas de acesso** (utilizadores do sistema) com as quais as pessoas iniciam sessão no Klacks — isto não é o mesmo que a gestão de colaboradores ou de endereços, pois uma conta de acesso pode, mas não tem de, estar associada a um cartão de dados mestre de uma pessoa.

## Como funciona

**Adicionar utilizador:** O diálogo "+ Adicionar utilizador" gera automaticamente, a partir do nome próprio e do apelido (cada um com pelo menos 2 caracteres), uma sugestão para o nome de utilizador. O endereço de e-mail é obrigatório e tem de conter um caráter @; só é possível gravar quando todos os campos estiverem preenchidos de forma válida. Ao criar, é gerada automaticamente uma palavra-passe — para que a pessoa possa definir uma própria, utilize depois "Repor palavra-passe" na respetiva linha.

**Editar utilizador:** Um duplo clique na linha do nome abre o diálogo de edição. Aí é possível alterar posteriormente o nome de utilizador — na criação, este é apenas uma sugestão.

**Repor palavra-passe:** O link "Repor palavra-passe" na linha de um utilizador envia um e-mail para o endereço registado, através do qual a pessoa define ela própria uma nova palavra-passe.

**Eliminar utilizador:** O botão do caixote do lixo remove definitivamente uma conta de acesso, após confirmação no diálogo de eliminação.

**Atribuir permissões:** Através de uma lista pendente na linha, atribui **Supervisor** ou **Admin**; a alteração tem efeito imediato, sem necessidade de um clique de gravação separado. O que estas duas funções podem fazer em detalhe está descrito em "Funções e permissões" em Primeiros passos — aqui trata-se apenas da gestão das próprias contas.

## Bom saber

- O utilizador atualmente com sessão iniciada também é apresentado na lista, mas não pode ser editado, ter as suas permissões alteradas ou ser eliminado por si próprio aí.
- Uma conta de acesso é independente da gestão de pessoas/colaboradores — pode, por exemplo, criar uma conta para uma empresa de informática externa sem ter de criar para isso um cartão de colaborador.
- Não existe uma definição direta de palavra-passe pelo administrador: as contas novas recebem uma palavra-passe gerada automaticamente, e qualquer alteração posterior é feita através do link de e-mail "Repor palavra-passe".
