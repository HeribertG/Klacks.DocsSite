# Plugins de Funcionalidades

Os plugins de funcionalidades ampliam o Klacks com áreas funcionais opcionais que não fazem parte do produto principal — por exemplo, Mensagens (envio e receção via Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat e Zalo) ou Floor Plan (editor visual de planta de espaço com marcadores de postos de trabalho e atribuição de turnos).

## Como funciona

**Gerir plugins instalados:** Em Definições > Plugins de Funcionalidades vê uma lista dos plugins já instalados. Em cada linha, ativa ou desativa o plugin através de um interruptor (ativado/desativado), sem o desinstalar, ou remove-o completamente através da ação Desinstalar.

**Encontrar novos plugins:** Através do botão da lupa abre-se a caixa de diálogo do Marketplace, na qual pode procurar outros plugins disponíveis e instalá-los diretamente.

## Bom saber

- Um plugin desativado permanece instalado, mas fica bloqueado para todos os utilizadores: as páginas associadas não estão acessíveis e os endpoints de API associados devolvem uma resposta 404, como se não existissem.
- Após ativar/desativar ou instalar/desinstalar, o Klacks recarrega automaticamente as traduções, para que os novos textos do plugin fiquem imediatamente visíveis, sem ser necessário recarregar a página.
- Alguns plugins trazem os seus próprios cartões de definições (por exemplo, configuração de fornecedores de mensagens) — estes só aparecem, noutro local das definições, depois da instalação.
