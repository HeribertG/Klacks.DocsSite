# Wtyczki funkcjonalne

Wtyczki funkcjonalne rozszerzają Klacks o opcjonalne obszary funkcjonalności, które nie należą do produktu podstawowego — np. Messaging (wysyłanie i odbieranie wiadomości przez Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat i Zalo) lub Floor Plan (wizualny edytor planu pomieszczeń ze znacznikami stanowisk pracy i przypisaniem dyżurów).

## Jak to działa

**Zarządzanie zainstalowanymi wtyczkami:** W sekcji Ustawienia > Wtyczki funkcjonalne widzą Państwo listę już zainstalowanych wtyczek. W każdym wierszu można włączyć lub wyłączyć wtyczkę za pomocą przełącznika (aktywna/nieaktywna) bez jej odinstalowywania, lub usunąć ją całkowicie za pomocą akcji Odinstaluj.

**Wyszukiwanie nowych wtyczek:** Przycisk lupy otwiera okno dialogowe Marketplace, w którym można wyszukać kolejne dostępne wtyczki i od razu je zainstalować.

## Warto wiedzieć

- Wyłączona wtyczka pozostaje zainstalowana, ale jest zablokowana dla wszystkich użytkowników: powiązane strony są niedostępne, a powiązane punkty końcowe API zwracają odpowiedź 404, tak jakby w ogóle nie istniały.
- Po włączeniu/wyłączeniu lub instalacji/odinstalowaniu Klacks automatycznie przeładowuje tłumaczenia, dzięki czemu nowe teksty wtyczki są od razu widoczne, bez konieczności odświeżania strony.
- Niektóre wtyczki wprowadzają własne karty ustawień (np. konfigurację dostawcy Messaging) — pojawiają się one dopiero po instalacji, w innym miejscu ustawień.
