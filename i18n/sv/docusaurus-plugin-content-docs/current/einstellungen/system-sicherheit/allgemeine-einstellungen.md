# Allmänna inställningar

Under Inställningar > Allmänt anger du grundidentiteten för din Klacks-installation: applikationsnamn, ikon och logotyp — samt, som ett eget avsnitt på samma nivå av inställningarna, färgschemat för schemarutnätet.

## Så fungerar det

**Applikationsnamn:** Namnet du anger här används omedelbart som webbläsarens fliktitel för applikationen och visas även på andra ställen i gränssnittet, till exempel i automatiskt utskickade e-postmeddelanden.

**Ikon:** En egen favicon (`.ico` eller `.png`) laddar du upp via dra-och-släpp eller filval; den ersätter standardikonen i webbläsarfliken. Via "Ta bort" återställer du den till `favicon.ico`.

**Logotyp:** En egen logotyp (`.jpeg`, `.png` eller `.gif`) laddar du på samma sätt upp via dra-och-släpp eller filval; Klacks skalar den proportionellt för visning. Via "Ta bort" tas den bort igen.

### Färgschema för schemarutnätet

Längre ner på samma inställningssida anger du färgerna som schemarutnätet använder för att färglägga enskilda celler — till exempel för lördagar, söndagar, helgdagar, officiella helgdagar, förseglade (avslutade) perioder, skiftbyten, tillägg samt ram- och textfärger. Varje färg kan ändras individuellt via en färgväljare; listan visar för respektive rad beteckningen och den aktuellt valda färgen.

## Bra att veta

- Ändringar av namn, ikon och logotyp gäller omedelbart, utan separat klick på Spara — det finns inget formulär med avbryt-alternativ.
- Ett generöst valt färgschema underlättar snabb visuell orientering i schemarutnätet, till exempel för att på en blick skilja helger och helgdagar från vardagar — särskilt användbart för stora team med många rader.
- Dessa färger påverkar enbart visningen av själva schemarutnätet, inte andra delar av gränssnittet.
