# Användarhantering

Under Inställningar > Användarhantering hanterar du **inloggningskontona** (systemanvändare) som personer använder för att logga in i Klacks — det är inte samma sak som personal- eller adresshanteringen, eftersom ett inloggningskonto kan, men inte måste, vara kopplat till ett personregisterkort.

## Så fungerar det

**Lägg till användare:** Dialogen "+ Lägg till användare" genererar automatiskt ett förslag på användarnamn utifrån för- och efternamn (minst 2 tecken vardera). E-postadressen är obligatorisk och måste innehålla ett @-tecken; det går att spara först när alla fält är korrekt ifyllda. Vid skapandet genereras automatiskt ett lösenord — för att personen själv ska kunna ange ett eget använder du därefter "Återställ lösenord" på personens rad.

**Redigera användare:** Ett dubbelklick på namnraden öppnar redigeringsdialogen. Där kan användarnamnet ändras i efterhand — vid nyregistrering är det bara ett förslag.

**Återställ lösenord:** Länken "Återställ lösenord" på en användares rad skickar ett e-postmeddelande till den registrerade adressen, via vilket personen själv anger ett nytt lösenord.

**Ta bort användare:** Papperskorgs-knappen tar bort ett inloggningskonto permanent, efter bekräftelse i borttagningsdialogen.

**Tilldela rättigheter:** Via en rullgardinsmeny på raden tilldelar du **Supervisor** eller **Admin**; ändringen gäller omedelbart, utan separat klick på Spara. Vad dessa två roller får göra i detalj beskrivs under "Roller och rättigheter" i Kom igång — här handlar det bara om hanteringen av själva kontona.

## Bra att veta

- Den aktuellt inloggade användaren visas också i listan, men kan där varken redigeras, få ändrade rättigheter eller tas bort av sig själv.
- Ett inloggningskonto är oberoende av personal-/medarbetarhanteringen — du kan till exempel skapa ett konto för ett externt IT-företag utan att behöva skapa ett medarbetarkort för det.
- Det går inte att direkt sätta ett lösenord som administratör: Nya konton får ett automatiskt genererat lösenord, och varje senare ändring sker via e-postlänken "Återställ lösenord".
