# Funktionsplugins

Funktionsplugins utökar Klacks med valfria funktionsområden som inte tillhör kärnprodukten — till exempel Messaging (skickande och mottagande via Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat och Zalo) eller Floor Plan (visuell ritningsredigerare med arbetsplatsmarkörer och skifttilldelning).

## Så fungerar det

**Hantera installerade plugins:** Under Inställningar > Funktionsplugins ser du en lista över redan installerade plugins. På varje rad slår du på eller av pluginet via en reglagebrytare (aktiverad/inaktiverad) utan att avinstallera det, eller tar bort det helt via avinstallationsåtgärden.

**Hitta nya plugins:** Via lupp-knappen öppnas marketplace-dialogen, där du kan söka efter ytterligare tillgängliga plugins och installera dem direkt.

## Bra att veta

- Ett inaktiverat plugin förblir installerat, men är spärrat för alla användare: tillhörande sidor går inte att nå och tillhörande API-slutpunkter ger ett 404-svar, som om de inte fanns.
- Efter aktivering/inaktivering eller installation/avinstallation laddar Klacks automatiskt om översättningarna, så att nya plugin-texter blir synliga direkt utan att du behöver ladda om sidan.
- Vissa plugins har egna inställningskort (t.ex. konfiguration av Messaging-leverantör) — dessa visas först efter installationen på ett annat ställe i inställningarna.
