# Språkplugins

Klacks stödjer fyra kärnspråk fast inbyggda i frontend — tyska, engelska, franska, italienska — samt 21 ytterligare språk som installerbara språkplugins (Language Packs), bland annat arabiska, tjeckiska, danska, grekiska, spanska, finska, hebreiska, indonesiska, japanska, koreanska, malajiska, norska (bokmål), nederländska, polska, portugisiska, rumänska, svenska, thailändska, vietnamesiska samt förenklad och traditionell kinesiska.

## Så fungerar det

**Översikt:** Under Inställningar > Språkplugins ser du alla kärnspråk (alltid aktiva, kan inte tas bort) samt de redan installerade språkpluginsen.

**Installera:** Via lupp-knappen öppnas marketplace-dialogen, där du kan söka efter och installera fler språk.

**Avinstallera:** Via avinstallationsåtgärden på raden tar du bort ett installerat språkplugin igen; språkkonfigurationen laddas därefter om automatiskt.

## Bra att veta

- Ett språkplugin levererar mer än bara översättningstexter: det innehåller även land- och kantonspecifika data samt översatta kalenderregler och handböcker på respektive språk.
- De fyra kärnspråken finns alltid tillgängliga och kan inte avinstalleras — bara de 21 ytterligare språken är valfria.
- Täckningsgraden för ett språkplugin kan ligga under 100 %, om nya funktioner tillkom innan översättningen hann uppdateras; saknade texter faller då tillbaka till engelska, som är det tekniska reservspråket för alla plugin-språk.
