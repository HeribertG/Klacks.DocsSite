# Den guidade installationsturen

Ny i Klacks? Du behöver inte leta: Klacksy, den inbyggda assistenten, guidar dig steg för steg genom grundinställningen.

## Så fungerar det

Turen startar du direkt i chatten med Klacksy (till exempel med "Visa mig appen" eller via förslaget vid första starten). Därefter går det station för station genom de viktigaste områdena:

- Klacksy **förklarar varje område** med några få meningar — från företagets grundinställningar och adressen via kalender, användarhantering och grupper till medarbetare, arbetspass, tillgänglighet, frånvaro, helgdagar, periodavslut, e-postuppsättning och Klacksy-/plugin-inställningarna — totalt 16 stationer.
- Den **motsvarande navigationsikonen pulserar** samtidigt i sidopanelen, så att du alltid ser var du befinner dig.
- Du bestämmer tempot: fortsätt, hoppa över en station eller avsluta turen när du vill.

Så uppstår steg för steg en fullständig överblick över applikationen, helt utan att läsa en handbok.

## Bra att veta

- Turen kan upprepas hur många gånger som helst — även för nya teammedlemmar.
- Vid varje station kan du ställa följdfrågor till Klacksy; han svarar utifrån kontexten på aktuell sida.
- Snabbast upplever du turen i Playground: logga in, öppna Klacksy, börja skriva.
