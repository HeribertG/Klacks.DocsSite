# التثبيت وبيئة التجربة

يمكنكم تجربة Klacks خلال دقيقة واحدة — وتثبيته خلال دقائق قليلة. دون تسجيل، ودون أي محادثات بيعية.

## الخيار الأول: بيئة التجربة (Playground) بدون أي تثبيت

هناك نسخة عامة من Klacks تحتوي على بيانات تجريبية تعمل على **[klacks-software.ch](https://klacks-software.ch:7643)**:

- **تسجيل الدخول:** `admin@test.com` · **كلمة المرور:** `P@ssw0rt1`
- تحصلون على صلاحيات مدير كاملة — جرّبوا كل شيء بحرية.
- تتم **إعادة ضبط جميع البيانات تلقائيًا كل يوم**. لذا يُرجى عدم إدخال بيانات شخصية حقيقية.

## الخيار الثاني: على خادمكم الخاص (On-Premise)

توفر حزمة On-Premise كل ما تحتاجونه: صور Docker، برنامج التثبيت، قاعدة البيانات، HTTPS، والتحديثات التلقائية.

```bash
curl -fsSLO https://github.com/HeribertG/Klacks.Api/releases/latest/download/klacks-onprem.zip
unzip klacks-onprem.zip && cd onprem && ./install.sh
```

على أنظمة Windows: استخدموا `install.ps1` بدلًا من `install.sh`. يقوم برنامج التثبيت بإنشاء كلمات المرور والشهادات بنفسه، ويحمّل صور Docker الحالية، وينتظر حتى يعمل كل شيء. تسجيل الدخول الأول يتم عبر `admin@test.com` / `P@ssw0rt1` — **يُرجى تغيير كلمة المرور هذه فورًا بعد أول تسجيل دخول.**

## معلومات مفيدة

- توجد خدمة تحديث مرفقة تحافظ على تحديث تثبيتكم تلقائيًا — مع نسخة احتياطية قبل كل تحديث وتراجع تلقائي في حال حدوث مشاكل.
- تبقى بياناتكم بالكامل ضمن بنيتكم التحتية؛ Klacks لا يرسل أي شيء إلى الخارج.
- الشرط الوحيد هو توفر خادم يعمل عليه Docker، مع اتصال إنترنت صادر (لصور Docker والتحديثات)، ومنافذ مفتوحة 80/443.
