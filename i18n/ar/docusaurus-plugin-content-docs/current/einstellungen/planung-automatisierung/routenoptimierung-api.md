في الإعدادات > OpenRoute، تودعون مفتاح API الخاص بالخدمة الخارجية openrouteservice.org، والتي يستخدمها Klacks لتحسين المسارات (Routenoptimierung).

## كيف يعمل هذا

تحسب خدمة OpenRouteService أوقات السفر والمسافات — لركوب السيارة والدراجة والمشي — وتوفر بذلك الأساس لتحسين المسارات في Klacks. تُدخلون في الحقل الوحيد بهذه الصفحة مفتاح API الشخصي الخاص بكم. يمكنكم الحصول على مفتاح مجاني عبر رابط التسجيل الموجود في الصفحة والمؤدي إلى openrouteservice.org.

## معلومات مفيدة

- يُرسَل المفتاح مباشرة من المتصفح إلى openrouteservice.org، ولهذا يُعرض عمداً **بنص واضح** بدلاً من إخفائه (`***`) — على عكس مثلاً كلمات مرور البريد الإلكتروني. هذا مقصود حتى تتمكنوا من ملاحظة خطأ كتابي عرضي أو مفتاح منتهي الصلاحية، وليس خطأً في العرض.
- رغم ذلك، يُخزَّن المفتاح في قاعدة البيانات بشكل مشفّر.
- بدون مفتاح صالح، لا يعمل تحسين المسارات — ولا يمكن حينها حساب أوقات السفر والمسافات.
- لتحويل العناوين إلى إحداثيات فقط (الترميز الجغرافي/Geocoding)، يستخدم Klacks خدمة أخرى منفصلة — هذا المفتاح مخصص حصراً لحساب المسارات.
