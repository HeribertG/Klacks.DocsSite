# مزوّدو ونماذج LLM

يحتاج Klacksy إلى نموذج لغوي لكي يفهم ويتصرف. في الإعدادات > مزوّدو LLM ونماذج LLM وسجل مزامنة LLM، تحددون أي المزوّدين وأي النماذج المحددة يُسمح لـ Klacks باستخدامها لهذا الغرض.

## كيف يعمل الأمر

**إضافة مزوّد:** يجمع المزوّد بين عنوان URL الأساسي وإصدار API ومفتاح API الخاص بجهة معينة. ستة مزوّدين (OpenAI وAnthropic وGoogle وAzure وMistral وDeepSeek) مدمجون مباشرة في الكود؛ أما أي مزوّد آخر فيعمل عبر واجهة عامة متوافقة مع OpenAI — ما يتيح ربط جهات مثل Groq وTogether AI وFireworks وOpenRouter وxAI وCerebras وPerplexity أو نموذج تديرونه بأنفسكم. عبر زر "البحث عن مزوّدين"، يقترح Klacks أيضًا مزوّدين لم تقوموا بإضافتهم بعد: من كتالوج منسَّق لعناوين موثّقة، أو — إذا كان البحث على الإنترنت مُفعَّلًا — عبر بحث فعلي على الويب. يتم التحقق من إمكانية الوصول إلى كل اقتراح حيًّا قبل عرضه؛ وتُضاف المزوّدات المختارة معطَّلة وبدون مفتاح API، لتقوموا أنتم بإضافته لاحقًا.

**مفتاح API والأولوية:** بعد الحفظ، لا يُعرض المفتاح مرة أخرى كنص صريح في الواجهة — بل يُذكر فقط ما إذا كان قد تم تعيينه ("مُهيّأ"). تحدد الأولوية الترتيب الذي يُستدعى فيه المزوّدون المفعَّلون.

**إدارة النماذج:** في نماذج LLM، تُنشئون لكل مزوّد نماذج فردية (اسم العرض، معرّف النموذج التقني لدى الجهة، نافذة السياق، الحد الأقصى للرموز (tokens)، التكلفة لكل 1000 رمز إدخال/إخراج). يمكن وسم نموذج ما كنموذج افتراضي؛ ولا يمكن حذف مزوّد طالما أن أحد نماذجه هو الافتراضي الحالي.

**سجل مزامنة LLM:** تتحقق خدمة تعمل في الخلفية تلقائيًا كل 24 ساعة افتراضيًا من النماذج الجديدة المتاحة أو التي أصبحت غير متاحة لدى المزوّدين المفعَّلين. يتم اختبار النماذج الجديدة مسبقًا بطلب اختبار حقيقي، ولا تُدرَج مفعَّلة إلا في حال النجاح؛ أما النماذج التي لم تعد متوفرة فتُعطَّل تلقائيًا. يعرض سجل المزامنة التاريخ الكامل لكل مزوّد مع التاريخ وعدد النماذج الجديدة/المعطَّلة/الفاشلة ونتائج الاختبار القابلة للتوسيع. يتلقى المسؤولون أيضًا إشعارًا منبثقًا بعد تسجيل الدخول التالي حول أحداث المزامنة غير المقروءة.

## من الجيد معرفته

- ستة مزوّدين فقط (OpenAI وAnthropic وGoogle وAzure وMistral وDeepSeek) لديهم تطبيق مخصص — بقية المزوّدين يعملون عبر واجهة عامة متوافقة مع OpenAI.
- يحتوي كتالوج "البحث عن مزوّدين" حاليًا على اثني عشر مزوّدًا موثّقًا؛ ولا تظهر اقتراحات الويب إلا إذا كان البحث على الإنترنت مُهيَّأً في الإعدادات.
- تعمل خدمة سجل المزامنة في الخلفية بشكل مستقل عن جلستكم — لا حاجة لإبقاء الصفحة مفتوحة لذلك.
