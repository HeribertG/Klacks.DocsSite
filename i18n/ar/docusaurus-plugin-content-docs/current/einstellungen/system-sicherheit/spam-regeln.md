# قواعد البريد المزعج

في القائمة الإعدادات > قواعد البريد المزعج، تحدّدون قواعد تصفية تُستخدم لتصنيف رسائل البريد الإلكتروني الواردة تلقائيًا كبريد مزعج ضمن وحدة صندوق الوارد — وتسري هذه القواعد سواء أثناء الاستلام المستمر للرسائل الجديدة (عبر IMAP Polling) أو عند إعادة تقييم الرسائل التي وصلت سابقًا.

## كيف يعمل الأمر

**إنشاء قاعدة:** اختاروا نوع القاعدة — يحتوي عنوان المرسل على، نطاق المرسل، يحتوي الموضوع على، أو يحتوي المحتوى على — ثم أدخلوا النمط المراد فحصه وانقروا على إضافة.

**إدارة القواعد:** يمكن تفعيل كل قاعدة أو تعطيلها عبر مربع اختيار دون حذفها، أو إزالتها نهائيًا عبر زر سلة المهملات.

## من المفيد معرفته

- تبقى القواعد المعطَّلة محفوظة، لكنها لا تعود تؤثر على الرسائل الجديدة أو المُعاد تقييمها — وهذا مفيد لتعليق قاعدة مؤقتًا للاختبار بدلًا من حذفها.
- يستخدم Klacks لاستيراد طلبات نظام تخطيط موارد المؤسسات (ERP) نفس صندوق الوارد المستخدم لوحدة البريد الإلكتروني العادية (بنية صندوق بريد واحد). لذلك يمكن نظريًا لقاعدة واسعة النطاق بشكل مفرط (مثل نطاق مرسل كامل) أن تصنّف أيضًا رسائل طلبات ERP الواردة كبريد مزعج — لذا افحصوا القواعد الجديدة بعناية قبل تفعيلها.
- استخدموا قواعد نطاق المرسل لحظر مؤسسات مرسِلة غير مرغوب فيها بالكامل، بدلًا من تسجيل كل عنوان على حدة كقاعدة "يحتوي عنوان المرسل على".
