# إضافات الميزات

تُوسّع إضافات الميزات (Feature-Plugins) نطاق Klacks بمجالات وظيفية اختيارية لا تنتمي إلى المنتج الأساسي — مثل المراسلة (الإرسال والاستقبال عبر Telegram وSlack وViber وLINE وMicrosoft Teams والرسائل النصية القصيرة وSignal وThreema وKakaoTalk وWeChat وZalo) أو المخطط الطابقي (Floor Plan)، وهو محرر مرئي لمخطط المكان يتضمن علامات لمواقع العمل وربطها بالدوامات.

## كيف يعمل الأمر

**إدارة الإضافات المثبَّتة:** في القائمة الإعدادات > إضافات الميزات، تشاهدون قائمة بالإضافات المثبَّتة بالفعل. يمكنكم في كل صف تفعيل الإضافة أو إيقافها عبر مفتاح تبديل (مفعّلة/معطّلة) دون إلغاء تثبيتها، أو إزالتها بالكامل عبر إجراء إلغاء التثبيت.

**البحث عن إضافات جديدة:** يفتح زر العدسة المكبِّرة مربع حوار السوق (Marketplace)، حيث يمكنكم البحث عن إضافات أخرى متاحة وتثبيتها مباشرة.

## من المفيد معرفته

- تبقى الإضافة المعطَّلة مثبَّتة، لكنها تصبح محظورة على جميع المستخدمين: الصفحات المرتبطة بها تصبح غير قابلة للوصول، ونقاط نهاية واجهة برمجة التطبيقات (API) المرتبطة بها تُعيد استجابة 404 كأنها غير موجودة أصلًا.
- بعد التفعيل/التعطيل أو التثبيت/إلغاء التثبيت، يُعيد Klacks تحميل الترجمات تلقائيًا حتى تظهر نصوص الإضافة الجديدة فورًا دون الحاجة إلى إعادة تحميل الصفحة.
- تأتي بعض الإضافات ببطاقات إعدادات خاصة بها (مثل إعدادات مزوّدي خدمة المراسلة) — وتظهر هذه البطاقات فقط بعد التثبيت في موضع آخر ضمن الإعدادات.
