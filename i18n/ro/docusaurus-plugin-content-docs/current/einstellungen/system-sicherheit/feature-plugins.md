# Plugin-uri de funcționalități

Plugin-urile de funcționalități extind Klacks cu domenii funcționale opționale care nu fac parte din produsul de bază — de exemplu Messaging (trimitere și primire prin Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat și Zalo) sau Floor Plan (editor vizual de plan al spațiilor cu marcaje pentru posturi de lucru și atribuire de ture).

## Cum funcționează

**Administrarea plugin-urilor instalate:** În Setări > Plugin-uri de funcționalități vedeți o listă a plugin-urilor deja instalate. Pentru fiecare rând activați sau dezactivați plugin-ul printr-un comutator (activat/dezactivat), fără a-l dezinstala, sau îl eliminați complet prin acțiunea de dezinstalare.

**Găsirea de plugin-uri noi:** Butonul cu lupă deschide dialogul Marketplace, în care puteți căuta alte plugin-uri disponibile și le puteți instala direct.

## Bine de știut

- Un plugin dezactivat rămâne instalat, dar este blocat pentru toți utilizatorii: paginile aferente nu sunt accesibile, iar punctele finale API aferente returnează un răspuns 404, ca și cum nu ar exista.
- După activare/dezactivare sau instalare/dezinstalare, Klacks reîncarcă automat traducerile, astfel încât textele noului plugin să fie vizibile imediat, fără a fi nevoie să reîncărcați pagina.
- Unele plugin-uri aduc cu ele propriile carduri de setări (de exemplu configurarea furnizorilor de mesagerie) — acestea apar abia după instalare, în altă parte a setărilor.
