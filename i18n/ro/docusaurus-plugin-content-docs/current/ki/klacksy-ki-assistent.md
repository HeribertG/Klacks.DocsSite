# Klacksy: asistentul dumneavoastră AI pentru planificare

„Completează turele de dimineață pentru săptămâna viitoare" — această propoziție este suficientă. Klacksy este asistentul AI integrat care operează Klacks prin limbaj natural, scris sau vorbit.

## Cum funcționează

Klacksy vă înțelege cerințele și le traduce în acțiuni concrete — sunt disponibile peste **250 de abilități**:

- **Planificare:** completarea turelor, înlocuirea absențelor, mutarea serviciilor
- **Administrare:** crearea angajaților, gestionarea grupurilor, setarea drepturilor
- **Navigare:** „Arată-mi planul de servicii al echipei Berna" deschide vizualizarea corectă
- **Explicații:** Klacksy răspunde la întrebări despre aplicație și ghidează utilizatorii noi prin configurare

Înainte de fiecare modificare, Klacksy rezumă ce se va întâmpla și așteaptă confirmarea dumneavoastră — nimic nu se întâmplă pe la spate.

## Alegere liberă a modelului — inclusiv complet local

Ceea ce este special: **alegeți singur modelul de limbaj.** Sunt acceptați OpenAI, Anthropic, Google, DeepSeek și alți furnizori — sau un model rulat local. Astfel este posibilă și combinația pe care aproape niciun furnizor nu o oferă: un asistent AI la care **niciun set de date personale nu părăsește sediul.** Exact de asta au nevoie spitalele, Spitex și serviciile de securitate.

## Bine de știut

- Klacksy vorbește cele patru limbi principale (germană, franceză, italiană, engleză) și, prin plugin-uri lingvistice, alte 21.
- Printr-un server Model Context Protocol integrat, Klacks poate fi conectat și la instrumente AI externe.
- În Playground, Klacksy este activ — întrebați-l pur și simplu ceva.
