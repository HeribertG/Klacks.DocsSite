# Plugin tính năng

Plugin tính năng mở rộng Klacks với các khu vực chức năng tùy chọn không thuộc sản phẩm cốt lõi — ví dụ như Messaging (gửi và nhận qua Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat và Zalo) hoặc Floor Plan (trình chỉnh sửa sơ đồ mặt bằng trực quan với các điểm đánh dấu chỗ làm việc và gán ca làm việc).

## Cách hoạt động

**Quản lý các plugin đã cài đặt:** Trong mục Cài đặt > Plugin tính năng, bạn thấy danh sách các plugin đã được cài đặt. Với mỗi dòng, bạn có thể bật hoặc tắt plugin qua công tắc chuyển đổi (kích hoạt/vô hiệu hóa) mà không cần gỡ cài đặt, hoặc xóa hoàn toàn qua thao tác Gỡ cài đặt.

**Tìm plugin mới:** Nút kính lúp mở hộp thoại Marketplace, tại đó bạn có thể tìm kiếm các plugin khác đang có sẵn và cài đặt trực tiếp.

## Nên biết

- Một plugin bị vô hiệu hóa vẫn được cài đặt, nhưng bị khóa đối với tất cả người dùng: các trang liên quan không thể truy cập được và các endpoint API liên quan trả về phản hồi 404, như thể chúng không tồn tại.
- Sau khi kích hoạt/vô hiệu hóa hoặc cài đặt/gỡ cài đặt, Klacks tự động tải lại các bản dịch để các văn bản plugin mới hiển thị ngay lập tức mà không cần bạn phải tải lại trang.
- Một số plugin đi kèm với các thẻ cài đặt riêng (ví dụ cấu hình nhà cung cấp Messaging) — những thẻ này chỉ xuất hiện sau khi cài đặt, ở một vị trí khác trong phần cài đặt.
