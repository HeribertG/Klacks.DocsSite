# Plugin funzionali

I plugin funzionali estendono Klacks con aree funzionali opzionali che non fanno parte del prodotto principale — ad esempio Messaggistica (invio e ricezione tramite Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat e Zalo) o Floor Plan (editor visivo della planimetria con indicatori delle postazioni di lavoro e assegnazione dei turni).

## Come funziona

**Gestione dei plugin installati:** In Impostazioni > Plugin funzionali è visibile un elenco dei plugin già installati. Per ogni riga è possibile attivare o disattivare il plugin tramite un interruttore (attivato/disattivato) senza disinstallarlo, oppure rimuoverlo completamente tramite l'azione di disinstallazione.

**Ricerca di nuovi plugin:** Tramite il pulsante con la lente d'ingrandimento si apre la finestra di dialogo del marketplace, in cui è possibile cercare altri plugin disponibili e installarli direttamente.

## Da sapere

- Un plugin disattivato rimane installato, ma è bloccato per tutti gli utenti: le pagine corrispondenti non sono raggiungibili e i relativi endpoint API restituiscono una risposta 404, come se non esistessero.
- Dopo l'attivazione/disattivazione o l'installazione/disinstallazione, Klacks ricarica automaticamente le traduzioni, in modo che i nuovi testi del plugin siano immediatamente visibili senza dover ricaricare la pagina.
- Alcuni plugin includono schede di impostazione proprie (ad es. la configurazione dei provider di messaggistica) — queste compaiono solo dopo l'installazione, in un altro punto delle impostazioni.
