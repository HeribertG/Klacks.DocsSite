# Plugin linguistici

Klacks supporta stabilmente quattro lingue principali nel frontend — tedesco, inglese, francese, italiano — oltre a 21 lingue aggiuntive installabili come plugin linguistici (Language Packs), tra cui arabo, ceco, danese, greco, spagnolo, finlandese, ebraico, indonesiano, giapponese, coreano, malese, norvegese (bokmål), olandese, polacco, portoghese, rumeno, svedese, thailandese, vietnamita, nonché cinese semplificato e tradizionale.

## Come funziona

**Panoramica:** In Impostazioni > Plugin linguistici vedete tutte le lingue principali (sempre attive, non rimovibili) e i plugin linguistici già installati.

**Installazione:** Il pulsante con la lente apre la finestra di dialogo del Marketplace, in cui potete cercare e installare altre lingue.

**Disinstallazione:** Tramite l'azione di disinstallazione nella riga rimuovete un plugin linguistico installato; la configurazione linguistica viene poi ricaricata automaticamente.

## Utile da sapere

- Un plugin linguistico fornisce più delle sole traduzioni: porta con sé anche dati specifici per paese e cantone, oltre a regole del calendario e manuali tradotti nella rispettiva lingua.
- Le quattro lingue principali sono sempre presenti e non possono essere disinstallate — solo le 21 lingue aggiuntive sono opzionali.
- Il grado di copertura di un plugin linguistico può essere inferiore al 100% se sono state aggiunte nuove funzioni prima che la traduzione fosse aggiornata; i testi mancanti tornano quindi all'inglese, la lingua di fallback tecnica per tutte le lingue dei plugin.
