万年历会自动计算任意年份的法定假日 — 您只需定义一次规则，Klacks 便知晓未来所有年份的假日。

## 运作方式

每个法定假日都由一条简短的规则描述。基本类型有两种：

- **固定日期**，格式为 `MM/DD`：`01/01` 是元旦，`08/01` 是瑞士国庆日，`12/25` 是圣诞节。
- **与复活节挂钩的日期**，格式为 `EASTER+XX` 或 `EASTER-XX`：复活节日期依照高斯复活节公式自动计算，假日则表示为与复活节的间隔天数。例如：`EASTER-02` = 耶稣受难日，`EASTER+01` = 复活节星期一，`EASTER+39` = 耶稣升天节，`EASTER+50` = 圣灵降临节星期一，`EASTER+60` = 圣体节。

此外还有**带星期几偏移的固定日期**（`MM/DD+XX+WW` 或 `MM/DD+XX-WW`），用于「十一月的第一个星期四」（`11/01+00+TH`）或「五月的最后一个星期一」（`05/25+00-MO`）这类假日。

借助 **SubRules**（子规则），当假日落在特定星期几时可自动顺延 — 例如 `SA+2;SU+1`：假日落在周六或周日时，将移至周一。多条规则之间用 `;` 分隔。

每条规则还可录入名称与描述（多语言）、国家与州/联邦州，以及两个复选框：**法定假日**和**带薪**。

## 小贴士

- 星期缩写采用英文：`SU`、`MO`、`TU`、`WE`、`TH`、`FR`、`SA`。
- SubRules 主要用于将落在周末的假日顺延到下一个工作日。
- **带薪**复选框与工资计算相关。
- 通过国家与州/联邦州可以处理地区差异 — 例如耶稣受难日在瑞士并非所有州都适用（例外：VS、TI）。
