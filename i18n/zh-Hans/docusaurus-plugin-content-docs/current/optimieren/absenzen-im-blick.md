# 缺勤一目了然

休假、病假、进修、兵役 — 谁在什么时候缺勤，在 Klacks 中以清晰的甘特图呈现，纵览全年。

## 运作方式

- 每条缺勤都会在相应人员的时间轴上显示为一段彩色条 — 时间重叠与集中缺勤一眼即可看出。
- **自定义缺勤类型**由您自行定义，包括颜色与行为方式（带薪/无薪、计入/不计入工时）。
- 一键即可导出 **PDF** — 适合张贴公告或存档。
- 缺勤会直接影响排班：缺勤人员不会被[自动排班](../planen/auto-planung.md)排入班次，冲突也会立即显示在[排班网格](../planen/plan-raster.md)中。

## 小贴士

- 甘特图总览非常适合团队的休假规划：整个夏季的人手紧张时段一览无遗。
- 如果有人临时缺勤，Klacks 会立即显示既有资质又有空的替补人选 — 不必推翻整个排班表。
- 缺勤数值（例如一整天或半天的休假登记）会作为总和计入报表中的工时统计。
