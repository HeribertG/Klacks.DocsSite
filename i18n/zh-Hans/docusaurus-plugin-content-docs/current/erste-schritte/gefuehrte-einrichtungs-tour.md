# 引导式设置导览

初次使用 Klacks？您无需自己摸索：内置助手 Klacksy 会一步一步带您完成基础设置。

## 运作方式

导览直接在与 Klacksy 的聊天中启动（例如输入「带我看看这个应用」，或点击首次启动时出现的建议）。随后 Klacksy 会一站接一站地带您走过最重要的功能区域：

- Klacksy 用寥寥数语**讲解每个区域** — 从员工库、分组、合同，一直到法定假日与设置。
- 与当前内容**对应的导航图标会在侧边栏中脉动闪烁**，让您随时清楚自己身在何处。
- 节奏由您掌握：继续、返回或结束导览 — 随时都可以。

大约一刻钟之后，您就能对整个应用建立起全面的认识，完全不必翻阅手册。

## 小贴士

- 导览可以随时重复进行 — 也适合新加入的团队成员。
- 在每一站您都可以向 Klacksy 追问；他会结合当前页面的上下文作答。
- 体验导览最快的方式是 Playground：登录、打开 Klacksy、开始输入。
