# 排班网格：您的交互式时间矩阵

日常工作的核心：一个强大的时间矩阵，行代表员工或工作地点，列代表日期。

## 运作方式

- **拖放操作：** 直接用鼠标移动班次 — 从一天挪到另一天，从一个人换到另一个人。
- **实时同步：** 改动会立即呈现给所有打开同一排班表的人。两位排班人员可以同时工作，互不覆盖。
- **一目了然：** 资质、冲突与锁定会直接在网格中标出 — 一旦某项分配不合规，您立刻就能看到。
- **性能出色：** 网格专为大型排班表打造；即使数百名员工、整整一个月的排班，操作依然流畅。

## 小贴士

- 超出人员分组归属期或合同有效期的单元格会显示为灰色且无法排班 — 从源头避免误排。
- 网格与自动排班协同工作：先自动填充，再手动微调。
- 排班表所处的审批阶段可能会锁定编辑 — 参见[审批流程](../planen/freigabe-workflow.md)。
