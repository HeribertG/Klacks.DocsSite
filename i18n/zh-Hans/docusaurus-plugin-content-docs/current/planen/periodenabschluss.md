期间结算是您的月末结算工作台：您可以放行单个日期、结束整个期间并将其封存 — 封存后的数据是报表分析与导出的正式依据。

## 运作方式

页面分为三个标签页：**期间**、**导出**和**日志**。

1. **选择期间：** 通过起始与结束日期界定时间范围，并可按需筛选某个分组（或「所有分组」）。支付周期可选每周、每两周、每月或自定义。
2. **封存：** 作为管理员，点击*封存期间*（或*全部封存*）即可结束这些条目。确认对话框会预先显示将有多少条目受到影响。
3. **导出：** 从已封存的期间生成导出文件 — 格式包括 CSV、JSON、XML、DATEV 或 BMD NTCS，语言与货币可自由选择；如有需要还可以针对特定订单导出。*导出*标签页列出所有已生成的文件，包括时间范围、创建日期与创建人。
4. **重新开启：** 作为管理员，您可以在需要时重新开启已封存的期间（*重新开启期间*或*全部解封*）。此时**必须填写理由**。
5. **日志：** 每一次封存与重新开启都会记录在*日志*标签页中 — 包括操作、时间与执行人。

## 小贴士

- 未关闭的班次**不会**被导出。如果只有部分班次已关闭，Klacks 会向您发出警告（「y 个班次中有 x 个已关闭」）。
- 若某个时间范围已生成过导出文件，再次改动时 Klacks 会提出警告：这可能造成与财务账目的不一致。
- 重新开启已封存的期间可能影响工资结算 — 因此 Klacks 会再次确认并要求填写理由。
- 期间结算可通过左侧导航栏中的结算图标进入。
