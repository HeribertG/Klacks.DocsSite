# 自动排班：数秒生成排班表

排班工作中最繁琐的部分 — 把人员分配到班次 — Klacks 一键即可完成。

## 运作方式

一键自动排班的背后是一套**遗传算法**：它生成数千种排班方案，逐一评估，并将最优方案继续组合演化 — 直至得出一个均衡的结果。硬性规则始终优先，之后才轮到其他目标：

- **规则** — 劳动法规、休息时间、资质要求与可用时间是不可协商的：无论其余部分多优秀，方案都不能靠违反硬性规则取胜
- **覆盖率** — 每个班次都按要求人数配备人员
- **公平性** — 工作时长分配均衡，且每人的班次顺序尽量遵循早班 → 晚班 → 夜班的模式
- **连续性** — 尽量让员工留在其惯常的工作地点

您可以直接在排班表中针对所选时间段启动自动排班。结果会以普通排班草稿的形式呈现：一切仍可手动微调，不会有任何内容未经确认就被固定下来。

## 小贴士

- 资质要求是硬性约束：不具备所需资质的人员不会被排入班次。
- 自动排班会尊重已有的排班条目 — 您可以先手动设定关键班次，其余交给算法填充。
- 对结果不满意？重新排一次即可：算法会找出同样优秀的替代方案。
