# 欢迎来到 Klacks 文档中心

**Klacks** 是一款面向轮班与外勤企业的开源人员排班平台：自动编排班次、优化巡回路线、通过 AI 助手操控 — 本地部署，AI 模型任您选择。适用于居家护理、医院、安保、保洁与物流企业。

本 Wiki 将 Klacks 的工作方式拆成一个个易于消化的小主题。每个页面几分钟即可读完 — 所有功能都可以立即在公共 Playground 中亲手试用。
