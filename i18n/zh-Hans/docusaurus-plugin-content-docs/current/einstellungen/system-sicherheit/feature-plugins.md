# 功能插件

功能插件为 Klacks 扩展了不属于核心产品的可选功能模块 — 例如消息（通过 Telegram、Slack、Viber、LINE、Microsoft Teams、短信、Signal、Threema、KakaoTalk、微信和 Zalo 收发消息）或平面图（带工位标记和班次分配的可视化空间平面图编辑器）。

## 运作方式

**管理已安装的插件：** 在「设置 > 功能插件」中，您可以看到已安装插件的列表。每一行都可以通过开关切换该插件的启用/禁用状态，而无需卸载它，也可以通过卸载操作将其彻底移除。

**查找新插件：** 点击放大镜按钮会打开市场对话框，您可以在其中搜索其他可用插件并直接安装。

## 小贴士

- 被禁用的插件仍保持已安装状态，但对所有用户都被锁定：相关页面无法访问，相关 API 端点会返回 404 响应，如同它们根本不存在一样。
- 启用/禁用或安装/卸载后，Klacks 会自动重新加载翻译，让新的插件文本立即可见，无需您手动刷新页面。
- 有些插件自带独立的设置卡片（例如消息服务提供商配置）— 这些卡片只有在安装后才会出现在设置的其他位置。
