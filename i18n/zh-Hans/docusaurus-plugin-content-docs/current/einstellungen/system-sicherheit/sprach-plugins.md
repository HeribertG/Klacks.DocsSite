# 语言插件

Klacks 在前端固定支持四种核心语言 — 德语、英语、法语、意大利语 — 此外还有另外 21 种语言可作为可安装的语言插件（Language Packs）使用，包括阿拉伯语、捷克语、丹麦语、希腊语、西班牙语、芬兰语、希伯来语、印度尼西亚语、日语、韩语、马来语、挪威语（书面语）、荷兰语、波兰语、葡萄牙语、罗马尼亚语、瑞典语、泰语、越南语，以及简体中文和繁体中文等。

## 运作方式

**概览：** 在「设置 > 语言插件」中，您可以看到所有核心语言（始终启用，不可移除）以及已安装的语言插件。

**安装：** 点击放大镜按钮会打开市场对话框，您可以在其中搜索并安装更多语言。

**卸载：** 通过该行中的卸载操作，可以移除已安装的语言插件；之后语言配置会自动重新加载。

## 小贴士

- 一个语言插件带来的不仅仅是翻译文本：它还包含特定国家和州/省的相关数据，以及该语言版本的日历规则和操作手册。
- 四种核心语言始终存在，无法卸载 — 只有另外 21 种语言是可选的。
- 如果在翻译跟进之前就新增了功能，语言插件的覆盖率可能会低于 100%；此时缺失的文本会回退显示为英语，英语是所有插件语言的技术性后备语言。
