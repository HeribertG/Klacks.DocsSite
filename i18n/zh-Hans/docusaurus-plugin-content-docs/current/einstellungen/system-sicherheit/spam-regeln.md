# 垃圾邮件规则

在「设置 > 垃圾邮件规则」中，您可以定义过滤规则，让收件箱模块自动将符合条件的邮件标记为垃圾邮件 — 这些规则既作用于持续拉取新邮件（IMAP 轮询）的过程，也作用于对已收到邮件的事后重新评估。

## 运作方式

**创建规则：** 选择一种规则类型 — 发件人包含、发件人域名、主题包含或内容包含 — 输入要匹配的模式，然后点击添加。

**管理规则：** 每条规则都可以通过复选框启用或禁用，而无需删除；也可以通过垃圾桶按钮永久删除。

## 小贴士

- 被禁用的规则仍会保留，但不再作用于新邮件或重新评估的邮件 — 适合用于临时暂停某条规则，而不必将其删除。
- Klacks 的 ERP 订单导入功能使用的是与普通电子邮件收件箱模块相同的邮箱（单邮箱架构）。因此，一条设置过于宽泛的规则（例如整个发件人域名）理论上也可能把 ERP 订单邮件误判为垃圾邮件 — 请在启用新规则前仔细核查。
- 使用发件人域名规则可以整体屏蔽不需要的发件组织，而不必为每个地址单独创建发件人包含规则。
