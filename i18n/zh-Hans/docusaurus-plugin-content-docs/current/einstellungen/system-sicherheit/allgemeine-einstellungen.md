# 常规设置

在「设置 > 常规」中，您可以设置 Klacks 安装实例的基本身份标识：应用名称、图标和徽标 — 同一设置页面上还有一个独立的板块，用于配置排班表格的配色方案。

## 运作方式

**应用名称：** 在此填写的名称会立即成为应用的浏览器标题，并出现在界面的其他位置，例如自动发送的电子邮件中。

**图标：** 您可以通过拖放或选择文件的方式上传自定义 Favicon（`.ico` 或 `.png`），它会替换浏览器标签页中的默认图标。点击「移除」可将其重置为 `favicon.ico`。

**徽标：** 同样可以通过拖放或选择文件的方式上传自定义徽标（`.jpeg`、`.png` 或 `.gif`），Klacks 会按比例缩放以供显示。点击「移除」可将其删除。

### 排班表格的配色方案

在同一设置页面往下，您可以设定排班表格用于为各个单元格上色的颜色 — 例如周六、周日、假日、法定节假日、已封存（已结算）的周期、换班、加班津贴，以及边框和字体颜色。每种颜色都可以通过颜色选择器单独修改；列表中会分别显示名称和当前选定的颜色。

## 小贴士

- 对名称、图标和徽标的更改会立即生效，无需单独点击保存 — 这里没有带取消选项的表单。
- 精心设计的配色方案有助于在排班表格中快速获得视觉定位，例如一眼区分周末、假日与工作日 — 对行数较多的大团队尤为有用。
- 这些颜色只影响排班表格本身的显示，不影响界面的其他区域。
