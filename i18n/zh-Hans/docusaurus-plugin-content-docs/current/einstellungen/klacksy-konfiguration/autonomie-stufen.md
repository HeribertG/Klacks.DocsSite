# 自主等级

Klacksy 在需要征求您的同意之前，可以自主行动到什么程度？您可以在「设置 > Klacksy 自主性」中为自己单独设定 — 该设置按用户账户生效，而非对所有用户统一生效。

## 运作方式

**四个等级：**

- **建议** — Klacksy 仅提出建议，每一项更改都需要您确认。
- **辅助** — 可撤销的操作直接执行，其余操作需要确认。
- **自主**（默认）— 除特别敏感的操作外，所有操作均直接执行。
- **完全自主** — 即使是多步骤计划，也无需中间确认即可执行。

**分级如何生效：** Klacksy 可能执行的每一项操作在内部都按风险分类：纯只读操作始终立即执行；可撤销的操作或与场景绑定的操作（例如在测试排班中）至少需要「辅助」等级；不可撤销的操作至少需要「自主」等级。敏感操作 — 用户管理、权限设置以及对自主性设置本身的更改 — **无论选择哪个等级**，始终需要明确确认。

**确认流程：** 如果某项操作需要您的批准，Klacksy 不会立即执行，而是将其搁置 5 分钟等待期，并总结将会发生的情况。只有当您用自己的话明确表示同意后，Klacksy 才会触发这个被搁置的操作 — 它绝不会代替您自行确认。

## 小贴士

- 自主等级是按用户账户设置的个人选项 — 每位用户都可以选择自己感到舒适的等级。
- 更高的等级并不意味着完全不再征询意见：敏感操作始终需要确认。
- 系统默认启用「自主」等级 — 这与 Klacksy 在引入自主等级功能之前的行为一致。
