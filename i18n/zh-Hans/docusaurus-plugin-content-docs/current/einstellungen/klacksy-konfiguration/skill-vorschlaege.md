# Klacksy 技能建议

Klacksy 会从自己的错误中学习：如果您在聊天中纠正了 Klacksy 选错的技能，系统会记住这次纠正。管理员可以在「设置 > Klacksy 技能建议」中查看由此产生的技能描述改进建议，并决定是否采纳。

## 运作方式

**建议的来源：** 如果 Klacksy 在聊天中为某个请求选择了错误的技能，而用户对此进行了纠正，这段对话就会被标记为「已纠正」。一个内部优化器会分析这些经过纠正的对话，并由此推导出对技能描述的具体文本修改建议 — 目的是让 Klacksy 下次能正确匹配到相应的技能。

**生成建议：** 点击「生成建议」按钮会触发对最近 30 段经过纠正的对话的分析，如果能找到规律，就会据此生成新的建议。

**每张建议卡片显示：** 涉及的技能和被修改的字段、修改前后的对比、说明该修改为何合理的理由，以及 — 可展开查看的 — 触发该建议的具体用户请求。

**决定：** 每一条待处理的建议都需要单独**采纳**（技能描述会立即被修改）或**驳回**（建议被舍弃，不做任何更改）。系统不会自动采纳 — 始终需要人工参与决定。

## 小贴士

- 没有聊天中的纠正就不会产生任何建议 — 系统需要真实的纠错案例作为训练基础，仅靠使用是不够的。
- 该功能只修改技能的描述文本，以便 Klacksy 能正确选中它 — 并不会改变某项技能实际执行的内容。
- 该功能需要管理员权限。
