# Klacksy 人格设定

Klacksy 没有一成不变、硬编码的性格 — 您可以在「设置 > Klacksy 人格」中，用自己的话定义 Klacksy 的行为方式、说话方式以及应对各种情况的方式。

## 运作方式

**六个文本字段：** 人格设定分为六个部分，每个部分都有各自的自由文本字段：

- **身份** — Klacksy 是谁？角色、目的与背景。
- **性格与动机** — 性格特点、基本态度，以及是什么驱动着 Klacksy。
- **语气** — Klacksy 听起来是什么样？正式还是随意，简洁还是详尽。
- **界限** — Klacksy 不可以做什么？数据保护、禁忌话题、敏感领域。
- **沟通风格** — 回答的结构、如何处理不确定性、如何追问。
- **价值观与连续性** — 核心价值观，以及 Klacksy 如何随时间发展变化。

**保存：** 每个字段在失去焦点（离开该字段）时会单独自动保存 — 无需专门的保存按钮。更改会立即在与 Klacksy 的下一次对话中生效。

## 小贴士

- 人格设定属于共用的 Klacksy 默认智能体 — 更改会影响**所有**用户，而不仅仅是您自己的账户。
- 这些字段都是纯自由文本 — 没有格式要求；经验表明，简短清晰的句子比冗长的修饰性描述效果更可靠。
- 留空字段没有问题：Klacksy 会使用其内置的默认基调。还有一些内部行为细节由系统自身维护，无法在此处直接设置。
