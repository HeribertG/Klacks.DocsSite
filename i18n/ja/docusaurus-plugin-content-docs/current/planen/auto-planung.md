# 自動計画: 数秒でできるプラン

勤務計画の中でもっとも手間のかかる作業 — 人をシフトに割り当てること — をKlacksはボタン一つでこなします。

## 仕組み

ワンクリックの自動計画の裏側では、**遺伝的アルゴリズム**が動いています。何千ものプランのバリエーションを生成し、それらを評価し、優れたものどうしをさらに組み合わせていく — バランスの取れた解が得られるまで、これを繰り返します。その際、厳格なルールが常に優先され、それを満たした上で他の目標が考慮されます。

- **ルール** — 労働法、休息時間、資格、可用性は交渉の余地がありません。どれほど良い結果であっても、厳格なルールを破ることは許されません
- **カバー率** — 各シフトに必要な人数が確実に配置されます
- **公平性** — 労働時間は均等に配分され、各人のシフトの並びは可能な限り早番→遅番→夜勤のパターンに従います
- **継続性** — 可能な限り、従業員はいつもの勤務地に留まります

自動計画は、選択した期間の勤務表から直接起動します。結果は通常のプラン下書きとして表示されます。すべて手動で調整可能で、確認なしに確定されることはありません。

## 知っておくと便利なこと

- 資格は実際の制約として働きます。必要な資格を持たない人は割り当てられません。
- 自動計画は既存のエントリーを尊重します — つまり、先に固定点を設定してから、残りを埋めさせることができます。
- 結果が気に入らない場合は、もう一度計画し直すだけで構いません。アルゴリズムは同等の別案を見つけます。
