# プラン・グリッド: あなたの対話型タイム・マトリックス

日々の業務の中心となるのが、この高機能なタイム・マトリックスです。行には従業員や顧客が、列には日付が並びます。

## 仕組み

- **ドラッグ&ドロップ:** シフトはマウス操作でそのまま人から人へ移動できます。塗りつぶしハンドルを使えば、同じ行の他の日にもエントリーをコピーできます。
- **リアルタイム:** 変更は、同じプランを開いている全員に即座に反映されます。2人の計画担当者が同時に作業しても、競合は検知され、変更が気づかれないまま上書きされることはありません。
- **すべてが一目瞭然:** 資格やロックはグリッド上に直接表示され、競合はさらに専用の一覧でKlacksが通知します — 割り当てが許可されない場合、すぐに気づくことができます。
- **高速:** このグリッドは大規模なプランのために作られており、数百人の従業員がひと月分でも、快適に操作できます。

## 知っておくと便利なこと

- あるグループに属していない、または契約期間外にあるセルは、灰色表示となり計画できません。こうすることで、うっかりした割り当てが発生しません。
- グリッドと自動計画は連携して働きます。まず自動で埋めさせ、その後に手動で微調整するという流れです。
- それぞれのエントリーの承認段階に応じて、編集がロックされることがあります — 詳しくは[承認ワークフロー: 下書きから確定プランへ](../planen/freigabe-workflow.md)を参照してください。
