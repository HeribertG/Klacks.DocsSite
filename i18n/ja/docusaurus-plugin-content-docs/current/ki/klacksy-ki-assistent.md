# Klacksy: 計画のためのAIアシスタント

「来週の早番を埋めて」— この一言で十分です。Klacksyは、自然な言葉で、テキストでも音声でもKlacksを操作できる内蔵AIアシスタントです。

## 仕組み

Klacksyはあなたの要望を理解し、それを具体的なアクションへと変換します — **250を超える機能**が用意されています。

- **計画する:** シフトを埋める、欠員を補充する、勤務を振り替える
- **管理する:** 従業員を登録する、グループを整備する、権限を設定する
- **移動する:** 「ベルンチームの勤務表を見せて」と言えば、正しい画面が開きます
- **説明する:** Klacksyはアプリに関する質問に答え、新しいユーザーをセットアップの間案内します

変更を行う前に、Klacksyは何が起こるかを要約し、あなたの確認を待ちます — 何も知らないうちに実行されることはありません。

## 自由なモデル選択 — 完全にローカルでも

特徴的なのは、**言語モデルを自分で選べる**という点です。OpenAI、Anthropic、Google、DeepSeekなど各社のモデルに対応しているほか、ローカルで動作するモデルも利用できます。これにより、他社ではほとんど実現できない組み合わせが可能になります — **一件の個人データも外部に出さないAIアシスタント**です。まさにこれこそ、病院、訪問看護(Spitex)、警備サービスが必要としているものです。

## 知っておくと便利なこと

- Klacksyは4つの主要言語(ドイツ語、フランス語、イタリア語、英語)を話し、言語プラグインを通じて他の21言語にも対応します。
- 内蔵のModel-Context-Protocolサーバーを通じて、Klacksを外部のAIツールと接続することもできます。
- Playgroundでは、Klacksyが実際に動作しています — 気軽に何か質問してみてください。
