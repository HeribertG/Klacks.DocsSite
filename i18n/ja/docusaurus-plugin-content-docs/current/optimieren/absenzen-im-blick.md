# 欠勤を一目で把握

休暇、病気、研修、兵役——誰がいつ不在なのかを、Klacksでは1年間を通した見やすいガントチャートで確認できます。

## 仕組み

- それぞれの欠勤は、該当する人のタイムライン上に色付きのバーとして表示されます。重複や偏りが一目でわかります。
- **独自の欠勤タイプ**を自分で定義できます。色や挙動(有給/無給、時間への影響あり/なし)も設定可能です。
- **PDF**へのエクスポートはワンクリックで完了します——掲示や保管に便利です。
- 欠勤は計画に直接影響します。不在の人は[自動計画:数秒でできるプラン](../planen/auto-planung.md)では割り当てられず、[プラン・グリッド:対話型の時間マトリクス](../planen/plan-raster.md)でもコンフリクトが即座に表示されます。

## 知っておくと便利なこと

- ガントチャートの概観はチーム内の休暇計画に適しています。夏季全体のボトルネックを一目で確認できます。
- 誰かが急に欠勤しても、Klacksは資格を持つ利用可能な代替要員を即座に表示します。プラン全体を崩す必要はありません。
- 欠勤の値(例えば休暇の全日または半日の登録)は、レポートの時間集計に合計値として反映されます。
