# ガイド付きセットアップツアー

Klacksをはじめて使いますか?何かを探し回る必要はありません。組み込みアシスタントのKlacksyが、基本的なセットアップを一歩ずつ案内してくれます。

## 仕組み

ツアーはKlacksyとのチャットで直接開始します(例えば「アプリを見せて」と入力するか、初回起動時の提案から)。その後は、主要な領域をステーションごとに順番に巡っていきます。

- Klacksyが**各領域を数文で説明**します ― 会社の基本設定や住所、カレンダー、ユーザー管理、グループから、従業員、サービス、可用性、休暇、祝日、期間締め、メール設定、Klacksy/プラグイン設定まで、合計16のステーションです。
- その間、**対応するナビゲーションのアイコンがサイドバーで点滅**するので、今どこにいるのかが常にわかります。
- ペースはあなたが決められます ― 先に進む、あるステーションをスキップする、いつでもツアーを終了する、といった操作が可能です。

こうして、マニュアルを読むことなく、アプリケーションの全体像を一歩ずつ完全に把握できます。

## 知っておくと良いこと

- ツアーは何度でも繰り返すことができます ― 新しいチームメンバーのためにも使えます。
- 各ステーションで、Klacksyに質問を投げかけることができます。現在のページの文脈に沿って回答してくれます。
- 一番手っ取り早くツアーを体験するにはPlaygroundを使うことです。ログインしてKlacksyを開き、入力を始めるだけです。
