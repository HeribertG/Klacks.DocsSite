# 一般設定

「設定 > 一般」では、Klacksインストールの基本アイデンティティ ― アプリケーション名、アイコン、ロゴ ― を設定します。さらに、同じ階層にある別のセクションとして、プランマトリクスの配色も設定できます。

## 仕組み

**アプリケーション名:** ここに入力した名前は、即座にアプリケーションのブラウザタイトルとして反映され、自動送信されるメールなど、画面上の他の箇所にも表示されます。

**アイコン:** 独自のファビコン(`.ico`または`.png`)をドラッグ&ドロップまたはファイル選択でアップロードすると、ブラウザタブの標準アイコンが置き換えられます。「削除」を押すと、`favicon.ico`に戻ります。

**ロゴ:** 独自のロゴ(`.jpeg`、`.png`または`.gif`)も同様にドラッグ&ドロップまたはファイル選択でアップロードでき、Klacksが表示用に比例縮小します。「削除」で再び消去されます。

### プランマトリクスの配色

同じ設定ページの下部では、プランマトリクスが各セルに使用する色 ― 土曜日、日曜日、祝日、公式祝日、封印済み(確定済み)期間、シフト交代、割増などの色、および枠線色・文字色 ― を設定します。各色はカラーピッカーで個別に変更可能で、リストにはそれぞれの識別子と現在選択されている色が表示されます。

## 知っておくと良いこと

- 名前・アイコン・ロゴの変更は、別途「保存」をクリックすることなく即座に反映されます ― キャンセルオプション付きのフォームはありません。
- 配色をゆとりを持って設定しておくと、プランマトリクス上での視覚的な把握が素早くなります。例えば週末や祝日を平日と一目で区別できるようになり、行数の多い大規模チームで特に役立ちます。
- これらの色は、プランマトリクス自体の表示にのみ作用し、画面の他の領域には影響しません。
