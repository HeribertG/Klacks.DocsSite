# 言語プラグイン

Klacksはフロントエンドに4つのコア言語 ― ドイツ語、英語、フランス語、イタリア語 ― を固定でサポートしており、さらにインストール可能な言語プラグイン(Language Packs)として、アラビア語、チェコ語、デンマーク語、ギリシャ語、スペイン語、フィンランド語、ヘブライ語、インドネシア語、日本語、韓国語、マレー語、ノルウェー語(ブークモール)、オランダ語、ポーランド語、ポルトガル語、ルーマニア語、スウェーデン語、タイ語、ベトナム語、および簡体字・繁体字中国語を含む21の追加言語をサポートしています。

## 仕組み

**概要:** 「設定 > 言語プラグイン」では、すべてのコア言語(常に有効で削除不可)と、すでにインストール済みの言語プラグインが表示されます。

**インストール:** 虫眼鏡ボタンでマーケットプレイスダイアログが開き、他の言語を検索してインストールできます。

**アンインストール:** 行内のアンインストール操作で、インストール済みの言語プラグインを再び削除できます。その後、言語設定は自動的に再読み込みされます。

## 知っておくと良いこと

- 言語プラグインは翻訳テキストだけでなく、国・州(カントン)固有のデータや、それぞれの言語に翻訳されたカレンダールール、マニュアルも提供します。
- 4つのコア言語は常に存在し、アンインストールすることはできません ― オプションなのは追加の21言語のみです。
- 翻訳が追いつく前に新機能が追加された場合、言語プラグインのカバー率が100%を下回ることがあります。その場合、不足しているテキストは、全プラグイン言語共通の技術的フォールバック言語である英語で表示されます。
