設定 > DeepL では、Klacks がテキストを自動翻訳するために使用する DeepL API キーを設定します — 現在は主に受信トレイの受信メールが対象です。

## 仕組み

**API キー:** DeepL API キー（無料の Free API の場合は `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx` 形式、有料の Pro API の場合は `:fx` の付かない形式）を入力する欄が一つだけあります。Klacks は `:fx` の末尾から自動的に2つの DeepL API アドレスのどちらを使用するかを判別するため、Free と Pro を切り替える別個の設定は不要です。アカウントとキー自体は deepl.com/pro-api で取得できます。

**使用方法:** 「受信トレイ」では、各メールを翻訳アイコンで現在設定されている UI 言語に翻訳できます。原文と翻訳文はスイッチで切り替えて比較できます。キーが設定されていない場合、この機能は利用できません。

## 知っておくとよいこと

- この設定はインストール全体に対してグローバルです — ユーザーごとのキーはありません。
- このページには専用の「テスト」ボタンはありません。キーが機能するかどうかは、受信トレイで最初に翻訳を試みた際に確認できます。
- キーが無効または期限切れの場合、Klacks は DeepL からの応答（認証エラー）によってそれを検知し、空欄や誤った翻訳を表示する代わりにその旨を通知します。
- キーは暗号化されて保存され、フォーム上ではマスクされて表示されます。
