「設定 > OpenRoute」では、Klacksが経路最適化に使用する外部サービスopenrouteservice.orgのAPIキーを登録します。

## 仕組み

OpenRouteServiceは車、自転車、徒歩の移動時間と距離を計算し、Klacksにおける経路最適化の基盤となります。このページの唯一の入力欄に、あなた個人のAPIキーを入力します。無料のキーは、ページ内にリンクされているopenrouteservice.orgの登録リンクから取得できます。

## 知っておくと良いこと

- このキーはブラウザから直接openrouteservice.orgへ送信されるため、意図的に（`***`のようにマスクされず）**平文で**表示されます——例えばメールパスワードとは異なります。これは、入力ミスや期限切れのキーに気づけるようにするための仕様であり、表示上の不具合ではありません。
- とはいえ、データベース内ではこのキーは暗号化されて保存されます。
- 有効なキーがなければ経路最適化は機能しません——移動時間や距離を計算できなくなります。
- 住所を座標に変換する処理（ジオコーディング）自体には、Klacksは別の独立したサービスを使用します——このキーは経路計算専用です。
