# 自律レベル

Klacksyがどこまで自律的に行動し、いつ確認を求めるべきか。設定 > Klacksy自律レベルで、ご自身用にこれを設定します — この設定はユーザーアカウントごとに適用され、全ユーザー共通ではありません。

## 仕組み

**4つのレベル:**

- **提案のみ** — Klacksyは提案するだけで、すべての変更にあなたの確認が必要です。
- **アシスト** — 取り消し可能な操作は直接実行され、それ以外はすべて確認が必要です。
- **自律**(標準) — 特に慎重を要するもの以外は、すべての操作が直接実行されます。
- **完全自律** — 複数ステップにわたるプランも、途中の承認なしで実行されます。

**レベルの適用方法:** Klacksyが実行しうる各操作は、内部でリスクに応じて分類されています。読み取り専用の操作は常に即座に実行されます。取り消し可能な操作、またはシナリオに紐づく操作(例: テスト計画内での操作)は、少なくとも「アシスト」レベルが必要です。取り消し不能な操作は、少なくとも「自律」レベルが必要です。慎重を要する操作 — ユーザー管理、権限、および自律設定自体の変更 — は、**選択されているレベルに関係なく**常に明示的な確認が必要です。

**確認ワークフロー:** 操作に承認が必要な場合、Klacksyはそれを即座に実行せず、5分間の待機時間を設けて保留し、何が実行されるかを要約します。あなた自身の言葉で明示的に同意した場合にのみ、Klacksyは保留していた操作を実行します — Klacksyがあなたに代わって自ら確認することは決してありません。

## 知っておくと良いこと

- 自律レベルはユーザーアカウントごとの個人設定です — 各ユーザーが自分の快適なレベルを選択できます。
- レベルを上げても、まったく確認されなくなるわけではありません: 慎重を要する操作は常に確認が必須です。
- 標準では「自律」レベルが有効になっています — これは自律レベル導入以前からのKlacksyの挙動に相当します。
