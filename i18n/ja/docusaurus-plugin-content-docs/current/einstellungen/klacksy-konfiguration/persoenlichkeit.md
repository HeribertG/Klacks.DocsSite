# Klacksyのパーソナリティ

Klacksyには固定的でハードコーディングされた性格はありません — 設定 > Klacksy パーソナリティで、Klacksyがどのように振る舞い、話し、状況にどう反応するかを、ご自身の言葉で設定できます。

## 仕組み

**6つのテキストフィールド:** パーソナリティは6つのセクションに分かれており、それぞれに独自の自由記述フィールドがあります。

- **アイデンティティ** — Klacksyとは何者か。役割、目的、コンテキスト。
- **パーソナリティと動機** — 性格、基本姿勢、Klacksyを動かすもの。
- **トーン** — Klacksyはどのように聞こえるか。フォーマルかカジュアルか、簡潔か詳細か。
- **境界線** — Klacksyがしてはいけないことは何か。データ保護、タブーとされるテーマ、デリケートな領域。
- **コミュニケーションスタイル** — 回答の構成、不確実性への対応、質問の仕方。
- **価値観と継続性** — 指針となる価値観と、Klacksyが時間とともにどのように発展するか。

**保存:** 各フィールドは、フォーカスが外れる（入力欄から離れる）と個別に自動保存されます。専用の保存ボタンは必要ありません。変更は次回のKlacksyとのチャットに即座に反映されます。

## 知っておくと良いこと

- パーソナリティは共有のKlacksy標準エージェントに属しています — 変更は自分のアカウントだけでなく、**すべて**のユーザーに影響します。
- フィールドは完全な自由記述であり、書式に関する規定はありません。経験上、短く明確な文章の方が長い装飾的な文章よりも信頼性が高い傾向があります。
- フィールドが空でも問題ありません。その場合Klacksyは内蔵の基本姿勢を使用します。その他一部の内部的な挙動はシステム自体によって管理されており、ここで直接設定することはできません。
