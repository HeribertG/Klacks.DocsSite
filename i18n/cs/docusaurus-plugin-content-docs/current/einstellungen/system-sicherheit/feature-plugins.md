# Funkční pluginy

Funkční pluginy rozšiřují Klacks o volitelné funkční oblasti, které nejsou součástí základního produktu — například Messaging (odesílání a přijímání přes Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat a Zalo) nebo Floor Plan (vizuální editor plánu místností se značkami pracovišť a přiřazením směn).

## Jak to funguje

**Správa nainstalovaných pluginů:** V části Nastavení > Funkční pluginy vidíte seznam již nainstalovaných pluginů. V každém řádku plugin přepínačem zapnete nebo vypnete (aktivováno/deaktivováno), aniž byste jej museli odinstalovat, nebo jej pomocí akce Odinstalovat zcela odstraníte.

**Vyhledání nových pluginů:** Tlačítko s lupou otevře dialog Marketplace, ve kterém můžete vyhledávat další dostupné pluginy a přímo je instalovat.

## Dobré vědět

- Deaktivovaný plugin zůstává nainstalovaný, ale je pro všechny uživatele zablokovaný: příslušné stránky nejsou dostupné a odpovídající API endpointy vracejí odpověď 404, jako by neexistovaly.
- Po aktivaci/deaktivaci nebo instalaci/odinstalaci Klacks automaticky znovu načte překlady, aby byly nové texty pluginu ihned viditelné, aniž byste museli obnovovat stránku.
- Některé pluginy přinášejí vlastní karty nastavení (např. konfiguraci poskytovatele Messaging) — ty se objeví až po instalaci na jiném místě v nastavení.
