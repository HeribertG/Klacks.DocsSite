# Plugin Bahasa

Klacks mendukung empat bahasa inti secara tetap di frontend — Jerman, Inggris, Prancis, Italia — serta 21 bahasa tambahan sebagai plugin bahasa (Language Packs) yang dapat dipasang, di antaranya Arab, Ceko, Denmark, Yunani, Spanyol, Finlandia, Ibrani, Indonesia, Jepang, Korea, Melayu, Norwegia (Bokmål), Belanda, Polandia, Portugis, Rumania, Swedia, Thailand, Vietnam, serta bahasa Tionghoa sederhana dan tradisional.

## Cara Kerjanya

**Ringkasan:** Di Pengaturan > Plugin Bahasa, Anda melihat semua bahasa inti (selalu aktif, tidak dapat dihapus) serta plugin bahasa yang sudah terpasang.

**Memasang:** Melalui tombol kaca pembesar, dialog Marketplace terbuka, tempat Anda dapat mencari dan memasang bahasa lainnya.

**Mencopot Pemasangan:** Melalui aksi Copot Pemasangan pada baris tersebut, Anda menghapus kembali plugin bahasa yang terpasang; konfigurasi bahasa kemudian dimuat ulang secara otomatis.

## Yang Perlu Diketahui

- Plugin bahasa menyediakan lebih dari sekadar teks terjemahan: plugin ini juga membawa data spesifik negara dan kanton, serta aturan kalender dan buku panduan yang telah diterjemahkan ke bahasa yang bersangkutan.
- Keempat bahasa inti selalu tersedia dan tidak dapat dicopot — hanya 21 bahasa tambahan yang bersifat opsional.
- Tingkat cakupan sebuah plugin bahasa dapat berada di bawah 100% jika fitur baru ditambahkan sebelum terjemahannya menyusul; teks yang hilang kemudian akan kembali ke bahasa Inggris, yang menjadi bahasa fallback teknis untuk semua bahasa plugin.
