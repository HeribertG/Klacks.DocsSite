# Καλώς ήρθατε στην τεκμηρίωση του Klacks

Το **Klacks** είναι η πλατφόρμα ανοιχτού κώδικα για τον προγραμματισμό προσωπικού σε επιχειρήσεις βαρδιών και αποστολών: αυτόματος προγραμματισμός υπηρεσιών, βελτιστοποίηση διαδρομών, έλεγχος μέσω βοηθού AI — On-Premise, με το μοντέλο AI της επιλογής σας. Για φορείς κατ' οίκον φροντίδας, νοσοκομεία, επιχειρήσεις ασφαλείας, καθαρισμού και εφοδιαστικής αλυσίδας.

Αυτό το wiki εξηγεί σε μικρές δόσεις πώς λειτουργεί το Klacks. Κάθε σελίδα διαβάζεται σε λίγα λεπτά — και όλα μπορούν να δοκιμαστούν αμέσως στο δημόσιο Playground.
