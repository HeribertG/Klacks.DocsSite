# Εγκατάσταση και Playground

Το Klacks το δοκιμάζετε σε ένα λεπτό — και το εγκαθιστάτε σε λίγα λεπτά. Χωρίς εγγραφή, χωρίς συζητήσεις πωλήσεων.

## Επιλογή 1: Playground (χωρίς εγκατάσταση)

Μια δημόσια εγκατάσταση Klacks με δείγματα δεδομένων τρέχει στο **[klacks-software.ch](https://klacks-software.ch:7643)**:

- **Σύνδεση:** `admin@test.com` · **Κωδικός:** `P@ssw0rt1`
- Έχετε πλήρη δικαιώματα διαχειριστή — δοκιμάστε τα πάντα.
- Όλα τα δεδομένα **επαναφέρονται αυτόματα κάθε μέρα**. Μην εισάγετε πραγματικά προσωπικά δεδομένα.

## Επιλογή 2: Στον δικό σας διακομιστή (On-Premise)

Το πακέτο On-Premise περιλαμβάνει τα πάντα: εικόνες Docker, πρόγραμμα εγκατάστασης, βάση δεδομένων, HTTPS και αυτόματες ενημερώσεις.

```bash
curl -fsSLO https://github.com/HeribertG/Klacks.Api/releases/latest/download/klacks-onprem.zip
unzip klacks-onprem.zip && cd onprem && ./install.sh
```

Σε Windows: `install.ps1` αντί για `install.sh`. Το πρόγραμμα εγκατάστασης δημιουργεί μόνο του κωδικούς και πιστοποιητικά, κατεβάζει τις τρέχουσες εικόνες Docker και περιμένει μέχρι όλα να λειτουργήσουν. Πρώτη σύνδεση με `admin@test.com` / `P@ssw0rt1` — **αλλάξτε αυτόν τον κωδικό αμέσως μετά την πρώτη σύνδεση.**

## Καλό να το ξέρετε

- Μια ενσωματωμένη υπηρεσία ενημερώσεων διατηρεί την εγκατάστασή σας αυτόματα ενημερωμένη — με αντίγραφο ασφαλείας πριν από κάθε ενημέρωση και αυτόματη επαναφορά σε περίπτωση προβλήματος.
- Τα δεδομένα σας παραμένουν εξ ολοκλήρου στη δική σας υποδομή· το Klacks δεν επικοινωνεί προς τα έξω.
- Προϋπόθεση είναι ένας διακομιστής με Docker, εξερχόμενη πρόσβαση στο διαδίκτυο (για εικόνες Docker και ενημερώσεις) και ανοιχτές θύρες 80/443.
