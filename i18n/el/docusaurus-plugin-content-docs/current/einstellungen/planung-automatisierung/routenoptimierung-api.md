Στην ενότητα Ρυθμίσεις > OpenRoute καταχωρείτε το κλειδί API για την εξωτερική υπηρεσία openrouteservice.org, την οποία χρησιμοποιεί το Klacks για τη βελτιστοποίηση διαδρομών.

## Πώς λειτουργεί

Η OpenRouteService υπολογίζει χρόνους διαδρομής και αποστάσεις — για μετακίνηση με αυτοκίνητο, ποδήλατο και με τα πόδια — και παρέχει έτσι τη βάση για τη βελτιστοποίηση διαδρομών στο Klacks. Στο μοναδικό πεδίο αυτής της σελίδας καταχωρείτε το προσωπικό σας κλειδί API. Ένα δωρεάν κλειδί μπορείτε να αποκτήσετε μέσω του συνδέσμου εγγραφής προς το openrouteservice.org, ο οποίος βρίσκεται στη σελίδα.

## Καλό να το γνωρίζετε

- Το κλειδί αποστέλλεται απευθείας από το πρόγραμμα περιήγησης στο openrouteservice.org και για αυτόν τον λόγο εμφανίζεται σκόπιμα **σε απλό κείμενο** αντί να είναι κρυμμένο (`***`) — σε αντίθεση π.χ. με τους κωδικούς πρόσβασης email. Αυτό γίνεται εσκεμμένα, ώστε να μπορείτε να εντοπίσετε ένα τυχαίο τυπογραφικό λάθος ή ένα ληγμένο κλειδί· δεν πρόκειται για σφάλμα εμφάνισης.
- Στη βάση δεδομένων το κλειδί αποθηκεύεται ωστόσο κρυπτογραφημένο.
- Χωρίς έγκυρο κλειδί η βελτιστοποίηση διαδρομών δεν λειτουργεί — οι χρόνοι διαδρομής και οι αποστάσεις δεν μπορούν τότε να υπολογιστούν.
- Για την απλή μετατροπή διευθύνσεων σε συντεταγμένες (geocoding) το Klacks χρησιμοποιεί μια άλλη, ξεχωριστή υπηρεσία — αυτό το κλειδί είναι υπεύθυνο αποκλειστικά για τον υπολογισμό διαδρομών.
