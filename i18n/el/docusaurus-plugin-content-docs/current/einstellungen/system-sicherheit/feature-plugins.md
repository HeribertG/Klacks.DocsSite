# Πρόσθετα Λειτουργιών

Τα πρόσθετα λειτουργιών επεκτείνουν το Klacks με προαιρετικές λειτουργικές περιοχές που δεν ανήκουν στο βασικό προϊόν — όπως το Messaging (αποστολή και λήψη μέσω Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat και Zalo) ή το Floor Plan (οπτικός επεξεργαστής κάτοψης χώρου με δείκτες θέσεων εργασίας και ανάθεση βαρδιών).

## Πώς λειτουργεί

**Διαχείριση εγκατεστημένων πρόσθετων:** Στο Ρυθμίσεις > Πρόσθετα Λειτουργιών βλέπετε μια λίστα με τα ήδη εγκατεστημένα πρόσθετα. Σε κάθε γραμμή ενεργοποιείτε ή απενεργοποιείτε το πρόσθετο μέσω διακόπτη (ενεργό/ανενεργό), χωρίς να το απεγκαταστήσετε, ή το αφαιρείτε πλήρως μέσω της ενέργειας Απεγκατάσταση.

**Εύρεση νέων πρόσθετων:** Μέσω του κουμπιού μεγεθυντικού φακού ανοίγει ο διάλογος Marketplace, όπου μπορείτε να αναζητήσετε περισσότερα διαθέσιμα πρόσθετα και να τα εγκαταστήσετε απευθείας.

## Καλό είναι να γνωρίζετε

- Ένα απενεργοποιημένο πρόσθετο παραμένει εγκατεστημένο, αλλά είναι κλειδωμένο για όλους τους χρήστες: οι σχετικές σελίδες δεν είναι προσβάσιμες και τα σχετικά endpoints του API επιστρέφουν απάντηση 404, σαν να μην υπάρχουν.
- Μετά από ενεργοποίηση/απενεργοποίηση ή εγκατάσταση/απεγκατάσταση, το Klacks επαναφορτώνει αυτόματα τις μεταφράσεις, ώστε τα νέα κείμενα του πρόσθετου να είναι αμέσως ορατά, χωρίς να χρειάζεται να ανανεώσετε τη σελίδα.
- Ορισμένα πρόσθετα φέρνουν δικές τους κάρτες ρυθμίσεων (π.χ. διαμόρφωση παρόχου Messaging) — αυτές εμφανίζονται μόνο μετά την εγκατάσταση, σε άλλο σημείο των ρυθμίσεων.
