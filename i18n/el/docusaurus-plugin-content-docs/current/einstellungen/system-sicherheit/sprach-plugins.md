# Πρόσθετα Γλωσσών

Το Klacks υποστηρίζει μόνιμα τέσσερις βασικές γλώσσες στο frontend — Γερμανικά, Αγγλικά, Γαλλικά, Ιταλικά — καθώς και 21 επιπλέον γλώσσες ως εγκαταστάσιμα πρόσθετα γλωσσών (Language Packs), μεταξύ άλλων Αραβικά, Τσεχικά, Δανικά, Ελληνικά, Ισπανικά, Φινλανδικά, Εβραϊκά, Ινδονησιακά, Ιαπωνικά, Κορεατικά, Μαλαισιανά, Νορβηγικά (Bokmål), Ολλανδικά, Πολωνικά, Πορτογαλικά, Ρουμανικά, Σουηδικά, Ταϊλανδέζικα, Βιετναμέζικα καθώς και απλοποιημένα και παραδοσιακά Κινέζικα.

## Πώς λειτουργεί

**Επισκόπηση:** Στο Ρυθμίσεις > Πρόσθετα Γλωσσών βλέπετε όλες τις βασικές γλώσσες (πάντα ενεργές, δεν αφαιρούνται) καθώς και τα ήδη εγκατεστημένα πρόσθετα γλωσσών.

**Εγκατάσταση:** Μέσω του κουμπιού μεγεθυντικού φακού ανοίγει ο διάλογος Marketplace, όπου μπορείτε να αναζητήσετε και να εγκαταστήσετε περισσότερες γλώσσες.

**Απεγκατάσταση:** Μέσω της ενέργειας Απεγκατάσταση στη γραμμή αφαιρείτε ξανά ένα εγκατεστημένο πρόσθετο γλώσσας· η διαμόρφωση γλωσσών επαναφορτώνεται αυτόματα στη συνέχεια.

## Καλό είναι να γνωρίζετε

- Ένα πρόσθετο γλώσσας προσφέρει περισσότερα από απλά μεταφρασμένα κείμενα: φέρνει επίσης δεδομένα ειδικά ανά χώρα και καντόνι, καθώς και μεταφρασμένους κανόνες ημερολογίου και εγχειρίδια στην αντίστοιχη γλώσσα.
- Οι τέσσερις βασικές γλώσσες υπάρχουν πάντα και δεν μπορούν να απεγκατασταθούν — μόνο οι 21 επιπλέον γλώσσες είναι προαιρετικές.
- Ο βαθμός κάλυψης ενός πρόσθετου γλώσσας μπορεί να είναι κάτω από 100%, αν προστέθηκαν νέες λειτουργίες πριν προλάβει να ακολουθήσει η μετάφραση· τα κείμενα που λείπουν επιστρέφουν τότε στα Αγγλικά, την τεχνική εφεδρική γλώσσα για όλες τις γλώσσες πρόσθετων.
