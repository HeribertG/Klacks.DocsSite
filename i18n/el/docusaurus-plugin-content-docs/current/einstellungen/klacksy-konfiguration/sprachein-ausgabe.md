# Είσοδος και έξοδος ομιλίας

Ο Klacksy μπορεί να χρησιμοποιηθεί και μέσω ομιλίας — στις Ρυθμίσεις > Γλώσσα Klacksy ρυθμίζετε με ποια υπηρεσία μετατρέπεται η φωνή σας σε κείμενο (Speech-to-Text) και με ποια φωνή απαντά ο Klacksy (Text-to-Speech).

## Πώς λειτουργεί

**Αναγνώριση ομιλίας (STT):** Διατίθενται τέσσερις μηχανές — ο δωρεάν browser (Web Speech API), το Deepgram (ροή σε πραγματικό χρόνο), το Groq Whisper (γρήγορο και οικονομικό) και το AssemblyAI (ροή). Για όλες εκτός από τον browser καταχωρείτε δικό σας κλειδί API και μπορείτε να δοκιμάσετε απευθείας τη σύνδεση.

**Σύνθεση ομιλίας (TTS):** Επίσης τέσσερις πάροχοι — Edge TTS (δωρεάν), OpenAI TTS, ElevenLabs και Google Cloud TTS. Ανάλογα με τον επιλεγμένο πάροχο, ο Klacks φορτώνει αυτόματα τις διαθέσιμες φωνές· και εδώ μπορεί να δοκιμαστεί η σύνδεση.

**Λειτουργία εξόδου:** Καθορίζει αν οι απαντήσεις εμφανίζονται μόνο ως κείμενο, μόνο ως ήχος, ως κείμενο με χειροκίνητα ενεργοποιήσιμη εκφώνηση ή ως κείμενο με αυτόματη εκφώνηση. Η ανίχνευση σιωπής (500–3000 ms, προεπιλογή 1500 ms) καθορίζει πόσο χρόνο περιμένει μετά την ομιλία προτού η εγγραφή σταλεί αυτόματα.

**Καθαρισμός μεταγραφής:** Όταν η επιλογή είναι ενεργή, το ακατέργαστο κείμενο STT περνά επιπλέον από ένα LLM που αφαιρεί λέξεις-γέμισμα, επιλύει αυτοδιορθώσεις και εξομαλύνει γραμματική/αριθμούς. Το μοντέλο που χρησιμοποιείται γι' αυτό το επιλέγετε εσείς· μέσω του «Εύρεση καλύτερου μοντέλου» ο Klacks δοκιμάζει όλα τα ενεργοποιημένα μοντέλα με παράθυρο πλαισίου από 16.000 tokens και προτείνει τους πιο οικονομικούς, ταχύτερους υποψηφίους (χρονικό όριο 30 δευτ. ανά μοντέλο). Το prompt καθαρισμού είναι επεξεργάσιμο και μπορεί να επαναφερθεί στην εργοστασιακή ρύθμιση.

**Λεξικό:** Για όρους που η αναγνώριση ομιλίας παρερμηνεύει τακτικά (π.χ. κύρια ονόματα ή τεχνικούς όρους), δημιουργείτε καταχωρίσεις με τον σωστό όρο, κατηγορία, παραλλαγές προφοράς και προαιρετική αντιστοίχιση γλώσσας — ο Klacksy διορθώνει έπειτα αυτούς τους όρους στοχευμένα.

## Καλό να το ξέρετε

- Η λειτουργία browser δεν απαιτεί κλειδί API, αλλά εξαρτάται από την αναγνώριση ομιλίας του εκάστοτε browser και δεν είναι συγκρίσιμη σε ποιότητα με τις επί πληρωμή υπηρεσίες.
- Τα κλειδιά API εμφανίζονται μετά την αποθήκευση μόνο ως «Διαμορφωμένο», ποτέ πλέον σε απλό κείμενο.
- Ο έλεγχος μοντέλου εδώ εξετάζει αποκλειστικά την καταλληλότητα για τον καθαρισμό μεταγραφής (παράθυρο πλαισίου ≥ 16.000 tokens) — για τη γενική καταλληλότητα ως κύριο μοντέλο του Klacksy υπάρχει ξεχωριστή, αυστηρότερη δοκιμή (βλ. Δοκιμή σύνδεσης μοντέλου).
