# Paramètres généraux

Sous Paramètres > Général, vous définissez l'identité de base de votre installation Klacks : nom de l'application, icône et logo — ainsi que, dans une section distincte au même niveau des paramètres, le jeu de couleurs de la grille de planification.

## Comment ça fonctionne

**Nom de l'application :** le nom saisi ici est immédiatement repris comme titre du navigateur pour l'application et apparaît à d'autres endroits de l'interface, par exemple dans les e-mails envoyés automatiquement.

**Icône :** vous téléchargez votre propre favicon (`.ico` ou `.png`) par glisser-déposer ou par sélection de fichier ; il remplace l'icône par défaut dans l'onglet du navigateur. Le bouton « Supprimer » le réinitialise à `favicon.ico`.

**Logo :** vous téléchargez de la même manière votre propre logo (`.jpeg`, `.png` ou `.gif`) par glisser-déposer ou par sélection de fichier ; Klacks le redimensionne proportionnellement pour l'affichage. Le bouton « Supprimer » le supprime à nouveau.

### Jeu de couleurs de la grille de planification

Plus bas sur la même page de paramètres, vous définissez les couleurs avec lesquelles la grille de planification colore certaines cellules — par exemple pour les samedis, les dimanches, les jours fériés, les jours fériés officiels, les périodes verrouillées (clôturées), les changements d'équipe, les suppléments, ainsi que les couleurs de bordure et de police. Chaque couleur peut être modifiée individuellement via un sélecteur de couleur ; la liste affiche à chaque fois le libellé et la couleur actuellement choisie.

## Bon à savoir

- Les modifications du nom, de l'icône et du logo prennent effet immédiatement, sans clic sur un bouton d'enregistrement séparé — il n'y a pas de formulaire avec option d'annulation.
- Un jeu de couleurs bien choisi facilite le repérage visuel rapide dans la grille de planification, par exemple pour distinguer les week-ends et les jours fériés des jours ouvrables en un coup d'œil — particulièrement utile pour les grandes équipes avec de nombreuses lignes.
- Ces couleurs n'influencent que l'affichage de la grille de planification elle-même, pas les autres zones de l'interface.
