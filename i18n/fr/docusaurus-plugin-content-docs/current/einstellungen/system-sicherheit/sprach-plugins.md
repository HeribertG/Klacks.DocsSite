# Plugins linguistiques

Klacks prend en charge en permanence quatre langues principales dans le frontend — allemand, anglais, français, italien — ainsi que 21 autres langues sous forme de plugins linguistiques installables (packs de langue), parmi lesquelles l'arabe, le tchèque, le danois, le grec, l'espagnol, le finnois, l'hébreu, l'indonésien, le japonais, le coréen, le malais, le norvégien (bokmål), le néerlandais, le polonais, le portugais, le roumain, le suédois, le thaï, le vietnamien ainsi que le chinois simplifié et traditionnel.

## Comment ça fonctionne

**Vue d'ensemble :** sous Paramètres > Plugins linguistiques, vous voyez toutes les langues principales (toujours actives, non supprimables) ainsi que les plugins linguistiques déjà installés.

**Installer :** le bouton loupe ouvre la boîte de dialogue du Marketplace, dans laquelle vous pouvez rechercher et installer d'autres langues.

**Désinstaller :** l'action Désinstaller sur la ligne retire un plugin linguistique installé ; la configuration linguistique est ensuite automatiquement rechargée.

## Bon à savoir

- Un plugin linguistique apporte plus que de simples textes de traduction : il inclut aussi des données spécifiques au pays et au canton, ainsi que des règles de calendrier et des manuels traduits dans la langue correspondante.
- Les quatre langues principales sont toujours présentes et ne peuvent pas être désinstallées — seules les 21 langues supplémentaires sont optionnelles.
- Le taux de couverture d'un plugin linguistique peut être inférieur à 100 % lorsque de nouvelles fonctionnalités ont été ajoutées avant que la traduction ne les rattrape ; les textes manquants basculent alors sur l'anglais, la langue de repli technique pour toutes les langues des plugins.
