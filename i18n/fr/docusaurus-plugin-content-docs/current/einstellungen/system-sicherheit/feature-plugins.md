# Plugins de fonctionnalités

Les plugins de fonctionnalités étendent Klacks avec des domaines fonctionnels optionnels qui ne font pas partie du produit principal — par exemple la messagerie (envoi et réception via Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat et Zalo) ou le plan de salle (éditeur visuel de plan avec marqueurs de postes de travail et affectation des équipes).

## Comment ça fonctionne

**Gérer les plugins installés :** sous Paramètres > Plugins de fonctionnalités, vous voyez la liste des plugins déjà installés. Pour chaque ligne, vous activez ou désactivez le plugin via un interrupteur, sans le désinstaller, ou le supprimez complètement via l'action Désinstaller.

**Trouver de nouveaux plugins :** le bouton loupe ouvre la boîte de dialogue du Marketplace, dans laquelle vous pouvez rechercher d'autres plugins disponibles et les installer directement.

## Bon à savoir

- Un plugin désactivé reste installé, mais est verrouillé pour tous les utilisateurs : les pages associées ne sont pas accessibles et les points d'API correspondants renvoient une réponse 404, comme s'ils n'existaient pas.
- Après une activation/désactivation ou une installation/désinstallation, Klacks recharge automatiquement les traductions, afin que les nouveaux textes du plugin soient immédiatement visibles sans que vous ayez besoin de recharger la page.
- Certains plugins apportent leurs propres cartes de paramètres (par exemple la configuration des fournisseurs de messagerie) — celles-ci n'apparaissent qu'après l'installation, à un autre endroit des paramètres.
