Dans Paramètres > Organisation > Filiales, vous gérez les adresses de vos sites d'exploitation — une simple liste de vos propres emplacements, indépendante des collaborateurs ou des plannings.

## Comment ça fonctionne

**Créer une filiale :** Le bouton « + Ajouter une filiale » ouvre une fenêtre modale avec quatre champs : nom, adresse, téléphone et e-mail. Le nom et l'adresse sont obligatoires — sans ces deux champs, la fenêtre modale ne peut pas être enregistrée.

**Modifier :** Un clic sur le nom d'une filiale dans la liste ouvre la même fenêtre modale pour la modification.

**Supprimer :** L'icône de corbeille en fin de ligne supprime la filiale après une demande de confirmation.

**Enregistrement :** Chaque modification est appliquée immédiatement dès que vous cliquez sur « Enregistrer » dans la fenêtre modale. Cela distingue les filiales de la carte voisine Pays, dans la même section Organisation, dont les modifications ne sont sauvegardées que via la barre d'enregistrement commune de la page Paramètres.

## Bon à savoir

- Les filiales sont de simples fiches d'adresse : il n'existe actuellement aucun lien avec les collaborateurs, les groupes, les contrats ou les plannings — la liste sert uniquement à donner une vue d'ensemble de vos sites.
- L'assistant Klacksy peut également créer ou supprimer des filiales par commande vocale (« crée une filiale ... »).
