# La visite guidée de configuration

Nouveau sur Klacks ? Pas besoin de chercher : Klacksy, l'assistant intégré, vous guide pas à pas à travers la configuration de base.

## Comment ça marche

Vous démarrez la visite directement dans le chat avec Klacksy (par exemple en tapant « Montre-moi l'application » ou via la suggestion proposée au premier démarrage). Ensuite, vous parcourez étape par étape les domaines les plus importants :

- Klacksy **explique chaque domaine** en quelques phrases — des paramètres de base de l'entreprise et de l'adresse, en passant par le calendrier, la gestion des utilisateurs et des groupes, jusqu'aux collaborateurs, services, disponibilités, absences, jours fériés, clôture de période, configuration des e-mails et paramètres Klacksy/plugins — soit 16 étapes au total.
- L'**icône de navigation correspondante clignote** dans la barre latérale, pour que vous sachiez toujours où vous en êtes.
- Vous décidez du rythme : continuer, passer une étape ou arrêter la visite à tout moment.

Vous obtenez ainsi, étape par étape, une vue d'ensemble complète de l'application, sans avoir à lire de manuel.

## Bon à savoir

- La visite peut être répétée autant de fois que vous le souhaitez — aussi pour les nouveaux membres de l'équipe.
- À chaque étape, vous pouvez poser des questions à Klacksy ; il répond dans le contexte de la page actuelle.
- Le plus rapide pour découvrir la visite est de le faire dans le Playground : connectez-vous, ouvrez Klacksy, et lancez-vous.
