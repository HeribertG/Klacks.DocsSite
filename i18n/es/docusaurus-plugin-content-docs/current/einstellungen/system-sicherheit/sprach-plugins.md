# Plugins de idioma

Klacks admite de forma fija cuatro idiomas principales en el frontend — alemán, inglés, francés, italiano — así como otros 21 idiomas instalables como plugins de idioma (Language Packs), entre ellos árabe, checo, danés, griego, español, finlandés, hebreo, indonesio, japonés, coreano, malayo, noruego (bokmål), neerlandés, polaco, portugués, rumano, sueco, tailandés, vietnamita, así como chino simplificado y tradicional.

## Cómo funciona

**Vista general:** En Configuración > Plugins de idioma se ven todos los idiomas principales (siempre activos, no se pueden eliminar) así como los plugins de idioma ya instalados.

**Instalar:** El botón de la lupa abre el diálogo del Marketplace, donde puede buscar e instalar otros idiomas.

**Desinstalar:** Mediante la acción de desinstalar en la fila se elimina un plugin de idioma instalado; la configuración de idioma se vuelve a cargar automáticamente después.

## Bueno saberlo

- Un plugin de idioma aporta más que simples textos de traducción: también incluye datos específicos de país y cantón, así como reglas de calendario y manuales traducidos al idioma correspondiente.
- Los cuatro idiomas principales están siempre presentes y no se pueden desinstalar — solo los 21 idiomas adicionales son opcionales.
- El grado de cobertura de un plugin de idioma puede estar por debajo del 100 % si se añadieron nuevas funciones antes de que se completara la traducción; los textos faltantes recurren entonces al inglés, el idioma de reserva técnico para todos los idiomas de los plugins.
