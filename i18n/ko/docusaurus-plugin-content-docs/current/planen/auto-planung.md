# 자동 계획: 몇 초 만에 완성되는 계획

근무 시프트 계획에서 가장 번거로운 부분 — 사람을 시프트에 배정하는 작업 — 을 Klacks는 버튼 클릭 한 번으로 처리합니다.

## 작동 방식

1클릭 자동 계획의 배경에는 **유전 알고리즘**이 있습니다. 이 알고리즘은 수천 개의 계획 변형을 생성하고, 이를 평가한 뒤 가장 우수한 것들을 계속 조합합니다 — 균형 잡힌 해결책이 나올 때까지 이 과정을 반복합니다. 이때 엄격한 규칙이 항상 우선하며, 그다음에야 다른 목표들이 고려됩니다.

- **규칙** — 노동법, 휴식 시간, 자격 요건, 가용성은 협상 대상이 아닙니다. 아무리 좋은 결과라도 엄격한 규칙을 위반해서는 안 됩니다.
- **인력 배치** — 모든 시프트에 요구되는 인원수만큼 사람이 배치됩니다.
- **공정성** — 근무 시간이 고르게 분배되며, 개인별 시프트 순서는 가능한 한 이른 근무 → 늦은 근무 → 야간 근무 패턴을 따릅니다.
- **연속성** — 가능하다면 직원들은 익숙한 근무지에 계속 배치됩니다.

자동 계획은 원하는 기간에 대해 근무 계획표에서 직접 시작할 수 있습니다. 결과는 일반적인 계획 초안 형태로 나타납니다. 모든 내용은 수동으로 다시 조정할 수 있으며, 묻지 않고 확정되는 것은 없습니다.

## 알아두면 좋은 점

- 자격 요건은 실제 제약으로 작동합니다: 요구되는 자격이 없는 사람은 배치되지 않습니다.
- 자동 계획은 이미 존재하는 항목을 존중합니다 — 따라서 먼저 고정 지점을 설정한 뒤 나머지를 채우게 할 수 있습니다.
- 결과가 마음에 들지 않으면 다시 계획을 실행하면 됩니다. 알고리즘은 동등한 수준의 다른 대안을 찾아냅니다.
