# 안내형 초기 설정 투어

Klacks가 처음이신가요? 아무것도 찾아볼 필요 없습니다. 내장 어시스턴트인 Klacksy가 기본 설정 과정을 단계별로 안내해 드립니다.

## 작동 방식

투어는 Klacksy와의 채팅에서 바로 시작할 수 있습니다(예: "앱을 보여줘"라고 입력하거나, 첫 시작 시 나타나는 제안을 통해). 그 다음부터는 주요 영역을 하나씩 차례로 안내받습니다.

- Klacksy가 **각 영역을 몇 문장으로 설명**합니다 — 회사 기본 설정과 주소부터 캘린더, 사용자 관리, 그룹, 직원, 근무, 가용성, 결근, 공휴일, 기간 마감, 이메일 설정, Klacksy/플러그인 설정까지 — 총 16개 단계로 구성됩니다.
- 그 동안 사이드바에서 **해당하는 내비게이션 아이콘이 깜빡여** 지금 어느 위치에 있는지 항상 알 수 있습니다.
- 속도는 사용자가 정합니다. 다음으로 넘어가거나, 한 단계를 건너뛰거나, 언제든 투어를 종료할 수 있습니다.

이렇게 매뉴얼을 읽지 않고도 단계별로 애플리케이션 전체에 대한 완전한 개요를 얻을 수 있습니다.

## 알아두면 좋은 점

- 투어는 몇 번이든 반복할 수 있습니다 — 새로운 팀원을 위해서도 마찬가지입니다.
- 각 단계에서 Klacksy에게 추가 질문을 할 수 있으며, 현재 페이지의 맥락에 맞게 답변합니다.
- 투어를 가장 빠르게 체험하는 방법은 Playground입니다. 로그인하고, Klacksy를 열고, 바로 입력을 시작하세요.
