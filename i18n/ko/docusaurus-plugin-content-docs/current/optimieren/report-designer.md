리포트 디자이너를 사용하면 근무표부터 주소 목록까지, 외부 도구 없이 Klacks 안에서 직접 자신만의 인쇄 템플릿을 만들 수 있습니다.

## 작동 방식

리포트는 *디자이너* 탭에서 편집하는 세 영역으로 구성됩니다.

- **헤더** — 모든 페이지 상단에 인쇄됩니다. 왼쪽, 가운데, 오른쪽 세 영역으로 나뉘며, 각 영역에는 데이터베이스 필드, 자유 텍스트, 이미지(예: 이미지 관리자를 통해 업로드한 회사 로고)를 넣을 수 있습니다.
- **본문** — 하나 이상의 데이터 테이블을 포함합니다. 필드 목록에서 드래그 앤 드롭으로 열을 추가하며, 순서도 드래그 앤 드롭으로 조정하고, 너비는 열마다 퍼센트 입력란으로 지정합니다.
- **푸터** — 모든 페이지 하단에 인쇄됩니다.

각 테이블은 *데이터 소스* 탭에서 선택하는 **데이터 소스**와 연결됩니다: 근무 일정(Schedule), 목록 또는 상세 형태의 주소, 결근, 근무, 그룹, 또는 컨테이너 템플릿입니다. 데이터 소스에 따라 사용 가능한 필드가 결정됩니다.

디자인을 위해 다음 기능을 사용할 수 있습니다.
- **서식 지정** — 셀 및 자유 텍스트별로 글꼴, 크기, 굵게/기울임/밑줄, 텍스트 색상, 정렬을 지정합니다.
- **셀 테두리** — 각 변(위/오른쪽/아래/왼쪽)마다 개별적으로 선 스타일과 색상을 지정합니다.
- **테이블 바닥글** — SUM 필드로, 숫자 열은 자동으로 합산됩니다.
- **자유 텍스트 행** — 데이터 테이블 앞이나 뒤에 배치하며, 안내 사항, 조건, 서명란 등에 활용합니다.
- **페이지 설정** — *일반* 탭에서 세로/가로 방향, 용지 크기(A4, A3, B4/B5 JIS, Letter), mm 단위 페이지 여백을 지정합니다.

## 알아두면 좋은 점

- **병합 필드:** 선택한 필드의 툴바를 통해 여러 필드를 하나의 열로 결합할 수 있습니다. 구분자(쉼표, 공백, 줄바꿈, 하이픈)는 설정 가능합니다.
- **근무 일정** 데이터 소스에는 두 가지 추가 옵션이 있습니다: *일별 항목 합치기*(같은 날의 여러 항목을 한 행으로 표시)와 *전체 기간 표시*(항목이 없는 날도 포함해 모든 날짜를 표시).
- 테이블 바닥글은 해당 테이블 바로 아래의 체크박스로 활성화하며, 바닥글 필드의 SUM 표시는 별도로 켜거나 끌 수 있습니다.
