# 일반 설정

설정 > 일반에서는 Klacks 설치의 기본 아이덴티티인 애플리케이션 이름, 아이콘, 로고를 등록합니다 — 그리고 같은 설정 레벨의 별도 섹션으로, 계획표 그리드(Plan-Raster)의 색상 구성표도 함께 설정합니다.

## 작동 방식

**애플리케이션 이름:** 여기에 입력한 이름은 즉시 애플리케이션의 브라우저 제목으로 적용되며, 자동으로 발송되는 이메일 등 UI의 다른 여러 곳에도 표시됩니다.

**아이콘:** 사용자 지정 파비콘(`.ico` 또는 `.png`)을 드래그 앤 드롭 또는 파일 선택으로 업로드하면 브라우저 탭의 기본 아이콘을 대체합니다. "제거"를 클릭하면 다시 `favicon.ico`로 되돌립니다.

**로고:** 사용자 지정 로고(`.jpeg`, `.png` 또는 `.gif`)도 마찬가지로 드래그 앤 드롭 또는 파일 선택으로 업로드할 수 있으며, Klacks가 표시를 위해 비율에 맞게 크기를 조정합니다. "제거"를 클릭하면 다시 삭제됩니다.

### 계획표 그리드의 색상 구성표

같은 설정 페이지의 아래쪽에서 계획표 그리드가 개별 셀을 채색하는 색상을 지정합니다 — 예를 들어 토요일, 일요일, 공휴일, 공식 휴일, 봉인된(마감된) 기간, 교대 전환, 가산금, 그리고 테두리 색상과 글자 색상 등입니다. 각 색상은 색상 선택기를 통해 개별적으로 변경할 수 있으며, 목록에는 각각의 명칭과 현재 선택된 색상이 표시됩니다.

## 알아두면 좋은 점

- 이름, 아이콘, 로고 변경은 별도의 저장 클릭 없이 즉시 적용됩니다 — 취소 옵션이 있는 별도의 폼은 없습니다.
- 넉넉하게 구성한 색상 구성표는 계획표 그리드에서 빠르게 시각적으로 방향을 잡는 데 도움이 됩니다. 예를 들어 주말과 공휴일을 평일과 한눈에 구분할 수 있게 해주며, 행이 많은 대규모 팀에서 특히 유용합니다.
- 이 색상들은 오직 계획표 그리드 자체의 표시에만 적용되며, UI의 다른 영역에는 영향을 주지 않습니다.
