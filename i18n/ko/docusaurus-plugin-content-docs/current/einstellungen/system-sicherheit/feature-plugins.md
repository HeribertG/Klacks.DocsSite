# 기능 플러그인

기능 플러그인(Feature-Plugins)은 핵심 제품에 속하지 않는 선택적 기능 영역으로 Klacks를 확장합니다 — 예를 들어 메시징(Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat, Zalo를 통한 송수신) 또는 Floor Plan(작업 공간 마커와 근무 배정이 포함된 시각적 공간 배치 편집기) 등입니다.

## 작동 방식

**설치된 플러그인 관리:** 설정 > 기능 플러그인에서 이미 설치된 플러그인의 목록을 확인할 수 있습니다. 각 행에서 토글을 통해 플러그인을 제거하지 않고 켜거나 끌 수 있으며(활성화/비활성화), 제거 작업을 통해 완전히 삭제할 수도 있습니다.

**새 플러그인 찾기:** 돋보기 버튼을 클릭하면 마켓플레이스 대화상자가 열리며, 여기서 추가로 사용 가능한 플러그인을 검색하고 바로 설치할 수 있습니다.

## 알아두면 좋은 점

- 비활성화된 플러그인은 설치된 상태로 남아 있지만 모든 사용자에게 잠깁니다: 해당 페이지에 접근할 수 없고, 관련 API 엔드포인트는 마치 존재하지 않는 것처럼 404 응답을 반환합니다.
- 활성화/비활성화 또는 설치/제거 후에는 Klacks가 자동으로 번역을 다시 불러와서, 페이지를 새로고침하지 않아도 새로운 플러그인 텍스트가 즉시 표시됩니다.
- 일부 플러그인은 자체 설정 카드를 함께 제공합니다(예: 메시징 제공자 구성) — 이러한 항목은 설치 후에야 설정의 다른 위치에 나타납니다.
