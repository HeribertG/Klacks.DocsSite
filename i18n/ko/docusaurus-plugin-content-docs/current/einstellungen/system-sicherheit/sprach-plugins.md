# 언어 플러그인

Klacks는 프런트엔드에서 4개의 핵심 언어 — 독일어, 영어, 프랑스어, 이탈리아어 — 를 기본으로 지원하며, 설치 가능한 언어 플러그인(Language Pack) 형태로 21개의 추가 언어를 지원합니다. 여기에는 아랍어, 체코어, 덴마크어, 그리스어, 스페인어, 핀란드어, 히브리어, 인도네시아어, 일본어, 한국어, 말레이어, 노르웨이어(보크말), 네덜란드어, 폴란드어, 포르투갈어, 루마니아어, 스웨덴어, 태국어, 베트남어, 그리고 중국어 간체 및 번체가 포함됩니다.

## 작동 방식

**개요:** 설정 > 언어 플러그인에서 모든 핵심 언어(항상 활성화되어 있으며 제거할 수 없음)와 이미 설치된 언어 플러그인을 확인할 수 있습니다.

**설치:** 돋보기 버튼을 클릭하면 마켓플레이스 대화상자가 열리며, 여기서 추가 언어를 검색하고 설치할 수 있습니다.

**제거:** 행에 있는 제거 작업을 통해 설치된 언어 플러그인을 다시 삭제할 수 있으며, 이후 언어 설정이 자동으로 다시 로드됩니다.

## 알아두면 좋은 점

- 언어 플러그인은 단순한 번역 텍스트 이상을 제공합니다: 국가별·칸톤별 데이터는 물론, 해당 언어로 번역된 캘린더 규칙과 매뉴얼도 함께 제공합니다.
- 4개의 핵심 언어는 항상 존재하며 제거할 수 없습니다 — 추가되는 21개 언어만 선택적으로 설치할 수 있습니다.
- 새 기능이 추가된 뒤 번역이 아직 반영되지 않은 경우, 언어 플러그인의 번역 완성도(커버리지)가 100%보다 낮을 수 있습니다. 이 경우 누락된 텍스트는 모든 플러그인 언어의 기술적 대체 언어인 영어로 대체되어 표시됩니다.
