설정 > DeepL 메뉴에서 DeepL API 키를 등록하면 Klacks가 텍스트를 자동으로 번역할 수 있습니다 — 현재는 주로 받은 편지함의 수신 이메일에 사용됩니다.

## 작동 방식

**API 키:** 하나의 입력란에 DeepL API 키를 입력합니다(무료 Free API의 경우 `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx` 형식, 유료 Pro API의 경우 `:fx` 접미사 없이). Klacks는 `:fx` 접미사를 통해 두 DeepL API 주소 중 어느 쪽을 사용할지 자동으로 인식하므로 Free와 Pro를 별도로 전환할 필요가 없습니다. 계정과 키 자체는 deepl.com/pro-api에서 받을 수 있습니다.

**사용법:** "받은 편지함"에서 모든 이메일을 번역 아이콘을 통해 현재 설정된 화면 언어로 번역할 수 있으며, 전환 버튼으로 원문과 번역문을 비교할 수 있습니다. 키가 등록되어 있지 않으면 이 기능은 사용할 수 없습니다.

## 알아두면 좋은 점

- 이 설정은 전체 설치본에 전역으로 적용됩니다 — 사용자별 키는 존재하지 않습니다.
- 이 페이지에는 별도의 "테스트" 버튼이 없습니다. 키가 정상 작동하는지는 받은 편지함에서 첫 번역을 시도할 때 확인됩니다.
- 키가 유효하지 않거나 만료된 경우, Klacks는 DeepL의 응답(인증 오류)을 통해 이를 인식하고 빈 번역이나 잘못된 번역을 보여주는 대신 이를 알립니다.
- 키는 암호화되어 저장되며 입력 양식에서는 마스킹 처리되어 표시됩니다.
