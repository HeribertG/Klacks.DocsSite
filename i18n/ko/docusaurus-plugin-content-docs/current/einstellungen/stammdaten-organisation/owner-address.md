설정 > 일반 > 사무국 주소 메뉴에서 회사의 기본 주소와 국가, 주(州), 시간대, 공휴일 캘린더에 대한 전역 기본 설정을 등록합니다.

## 작동 방식

**주소 필드:** 이름, 전화번호, 주소 추가 정보, 이메일, 도로명, 우편번호, 지역 — 필수 필드 검증이 없는 순수 자유 입력 필드입니다.

**국가와 주(州):** 두 개의 선택 목록이 있으며, 주(州) 목록은 국가가 선택되어야만 활성화되고 선택된 국가에 해당하는 항목만 표시됩니다.

**시간대:** 브라우저가 지원하는 모든 시간대 중에서 선택합니다. 필드를 비워 두면 Klacks가 선택된 국가로부터 시간대를 자동으로 유추합니다(그렇지 않으면 UTC가 적용됩니다).

**전역 캘린더:** 선택된 국가에 해당하는 캘린더 선택지 중에서 회사 전체의 기본 공휴일 캘린더를 선택합니다. 개별 계약 템플릿에는 이와 별도로 다른 캘린더를 지정할 수 있습니다(설정 > 계약 참고).

**저장:** 이 카드는 설정 페이지의 일반 섹션에 있습니다. 변경 사항은 모아서 설정 페이지의 공통 저장 바를 통해 저장하거나 취소합니다 — 이 카드에는 별도의 저장 버튼이 없습니다.

## 알아두면 좋은 점

- 여기에 등록된 회사명은 생성된 주문 내보내기(ERP 내보내기)에 발신자 정보로 표시됩니다.
- 국가를 변경하면 주(州)가 자동으로 초기화됩니다. 이미 선택된 캘린더는 새 국가에도 해당하는 경우에만 유지되며, 그렇지 않으면 마찬가지로 비워집니다.
