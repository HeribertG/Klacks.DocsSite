설정 > 조직 > 국가 메뉴에서 Klacks에서 선택할 수 있는 국가 목록을 관리합니다 — 예를 들어 사무국 주소에서 사용됩니다.

## 작동 방식

**국가 목록:** 각 행에는 국가의 약어, 이름(현재 선택된 화면 언어로 표시), 국제전화 코드가 표시됩니다. 세 필드 모두 행에서 직접 편집할 수 있으며, 별도의 편집 모달은 없습니다.

**국가 추가:** 목록 끝의 "+ 국가 추가"를 클릭하면 새로운 빈 행이 추가되며, 약어, 이름, 국제전화 코드를 직접 입력합니다.

**검색:** 목록 위의 검색란은 국가 이름으로 필터링합니다.

**삭제:** 휴지통 아이콘은 국가 항목을 삭제 대상으로 표시합니다. 새로 생성했지만 아직 저장되지 않은 항목은 즉시 목록에서 제거됩니다.

**저장:** 조직 섹션의 다른 카드(지점, 계약, 자격)와 달리 국가 목록의 변경 사항은 즉시 반영되지 않고, 모아서 설정 페이지의 공통 저장 바를 통해 저장하거나 취소합니다.

## 알아두면 좋은 점

- 국가 이름은 다국어로 저장되어 있습니다 — 화면 언어를 변경하면 행에서 해당 언어의 이름 필드가 자동으로 표시됩니다.
- 이 국가 목록은 사무국 주소의 국가 선택 등에 사용됩니다 — 여기서 선택하려는 국가가 목록에 없으면 먼저 이곳에서 추가해야 합니다.
