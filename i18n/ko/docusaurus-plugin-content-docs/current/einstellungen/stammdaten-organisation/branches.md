설정 > 조직 > 지점 메뉴에서 사업장 주소를 관리합니다 — 직원이나 근무표와는 무관한 자체 지점의 단순한 목록입니다.

## 작동 방식

**지점 생성:** "+ 지점 추가"를 클릭하면 이름, 주소, 전화번호, 이메일 네 개의 필드가 있는 모달이 열립니다. 필수 필드는 이름과 주소이며, 이 둘이 없으면 모달을 저장할 수 없습니다.

**편집:** 목록에서 지점 이름을 클릭하면 동일한 모달이 편집용으로 열립니다.

**삭제:** 행 끝에 있는 휴지통 아이콘을 클릭하면 확인 절차를 거쳐 지점이 삭제됩니다.

**저장:** 모달에서 "저장"을 클릭하면 각 변경 사항이 즉시 반영됩니다. 이는 같은 조직 섹션에 있는 인접한 국가 카드와 다른 점으로, 국가 카드의 변경 사항은 설정 페이지의 공통 저장 바를 통해서만 저장됩니다.

## 알아두면 좋은 점

- 지점은 순수한 주소 레코드입니다. 현재 직원, 그룹, 계약, 근무표와의 연결 기능은 없으며, 이 목록은 오로지 지점 현황 파악을 위한 것입니다.
- 어시스턴트 Klacksy도 음성 명령으로 지점을 생성하거나 삭제할 수 있습니다("지점을 생성해 줘...").
