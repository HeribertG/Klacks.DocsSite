# Klacksy 성격

Klacksy는 고정되어 미리 프로그래밍된 성격을 가지고 있지 않습니다 — 설정 > Klacksy 성격에서 Klacksy가 어떻게 행동하고, 말하고, 상황에 반응할지를 직접 자신의 언어로 정할 수 있습니다.

## 작동 방식

**여섯 개의 텍스트 필드:** 성격은 여섯 개의 섹션으로 나뉘며, 각각 자체 자유 입력 필드를 가집니다:

- **정체성** — Klacksy는 누구인가? 역할, 목적, 맥락.
- **성격 & 동기** — 성향, 기본 태도, Klacksy를 움직이는 것.
- **말투** — Klacksy는 어떻게 들리는가? 격식체인지 편안한지, 간결한지 장황한지.
- **한계** — Klacksy가 해서는 안 되는 것은 무엇인가? 개인정보 보호, 금기 주제, 민감한 영역.
- **커뮤니케이션 스타일** — 답변의 구성 방식, 불확실성에 대한 대처, 되묻기.
- **가치관 & 연속성** — 핵심 가치와 Klacksy가 시간이 지남에 따라 어떻게 발전하는지.

**저장:** 각 필드는 포커스를 벗어날 때(초점을 잃을 때) 개별적으로 자동 저장됩니다 — 별도의 저장 버튼은 필요하지 않습니다. 변경 사항은 Klacksy와의 다음 대화에 즉시 반영됩니다.

## 알아두면 좋은 점

- 성격은 공유되는 Klacksy 기본 에이전트에 속합니다 — 변경 사항은 자신의 계정뿐 아니라 **모든** 사용자에게 영향을 미칩니다.
- 필드는 순수한 자유 텍스트입니다 — 정해진 형식은 없으며, 경험상 짧고 명확한 문장이 길게 꾸민 표현보다 더 신뢰할 만하게 작동합니다.
- 빈 필드는 문제가 되지 않습니다: 이 경우 Klacksy는 내장된 기본 태도를 사용합니다. 몇몇 다른 내부 행동 측면은 시스템 자체가 관리하며 여기서 직접 설정할 수 없습니다.
