설정 > 마법사 설정(Wizard-Einstellungen)에서는 Klacks의 계획 도우미 중 하나인 "Holistic Harmonizer"가 평가에 사용할 AI 언어 모델을 선택합니다.

## 작동 방식

드롭다운에서는 Klacks에서 활성화된 LLM 모델 중 Holistic Harmonizer가 전역적으로 사용할 모델을 선택합니다(또는 사용하지 않으려면 "(없음)"을 선택). "모든 모델 확인" 버튼을 클릭하면 Klacks가 작은 테스트 이미지를 활성화된 각 모델에 전송하여 응답 여부와 이미지 판독 가능 여부를 확인합니다 — 모델당 최대 90초의 타임아웃이 적용됩니다. 결과는 모델, 제공자, 상태(정상/오류), 응답 시간, 그리고 성공적으로 테스트된 모델을 바로 적용할 수 있는 버튼이 포함된 표로 표시됩니다.

이미지 처리는 부수적인 기능이 아니라 필수 조건입니다: 페이지의 안내 문구에 따르면 이 도우미("Wizard 3")는 작동하기 위해 이미지 처리가 가능한 모델이 필요합니다.

## 알아두면 좋은 점

- 이 페이지는 근무를 직원에게 자동으로 배정하는 원클릭 자동 계획 기능과는 **관련이 없습니다** — 해당 기능은 LLM 없이 유전 알고리즘으로 작동하며 여기에 설정 항목이 없습니다.
- 모델로 "(없음)"이 설정되어 있으면 Holistic Harmonizer는 사실상 비활성화된 상태입니다.
- 어떤 LLM 모델이 애초에 선택 가능한지(활성화/비활성화 여부)는 AI 설정의 다른 곳에서 지정합니다 — 여기서는 이미 활성화된 모델 중에서만 선택합니다.
- 확인 버튼은 순수한 자체 테스트(연결 가능 여부 + 이미지 판독 능력 + 응답 시간)일 뿐이며, 그 자체로는 계획 데이터를 변경하지 않습니다.
