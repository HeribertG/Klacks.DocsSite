캘린더 선택(Kalenderauswahl)은 하나 이상의 공휴일 캘린더(국가 + 주/州)를 이름이 지정된 패키지로 묶어, 이후 계약이나 그룹에 배정할 수 있도록 합니다.

## 작동 방식

목록에는 각 캘린더 선택의 이름과 배정된 캘린더 항목 수가 표시됩니다. 행을 클릭하면 편집 폼이 열립니다. 폼에서 국가를 선택한 다음 원하는 캘린더를 체크합니다 — 해당 국가의 전국 공휴일 캘린더 및 원하는 만큼의 주/州. 선택된 항목은 제거 가능한 칩(chip) 형태로 표시됩니다.

Klacks에는 미리 설치된 캘린더 선택이 제공됩니다: 스위스의 26개 칸톤(Kanton) 전체를 각각 개별적으로, 그리고 오스트리아, 독일, 이탈리아, 미국, 리히텐슈타인, 프랑스용 캘린더가 각각 하나씩 포함되어 있습니다. 이렇게 미리 설치된 선택 항목에는 "System" 배지가 표시됩니다.

직접 만든 캘린더 선택은 "+ 캘린더 선택 추가"를 통해 생성합니다 — 예를 들어 여러 칸톤을 회사별 조합으로 묶고 싶을 때 사용합니다.

## 알아두면 좋은 점

- 미리 설치된("System") 캘린더 선택은 **삭제할 수 없습니다** — 이 경우 휴지통 버튼 자체가 표시되지 않습니다. 다만 편집(이름, 구성)은 계속 가능합니다.
- 직접 만든 캘린더 선택은 현재 계약이나 그룹에서 사용되고 있지 **않은** 경우에만 삭제할 수 있습니다 — 사용 중인 경우 삭제 버튼이 비활성화되며 툴팁으로 사유가 표시됩니다.
- 배정은 계약에서 이루어집니다: 계약에서 원하는 캘린더 선택을 공휴일 캘린더로 지정하면, 예를 들어 칸톤별로 다른 공휴일을 반영할 수 있습니다.
- 이 페이지는 캘린더 규칙 세트를 묶고 이름을 붙이는 역할만 합니다. 공휴일 규칙 자체(계산 공식, 요일 이동 등)는 별도로 관리됩니다 — "공휴일 & 캘린더" 항목을 참고하세요.
