설정 > OpenRoute에서는 Klacks가 경로 최적화에 사용하는 외부 서비스 openrouteservice.org의 API 키를 등록합니다.

## 작동 방식

OpenRouteService는 자동차 이동, 자전거 이동, 도보에 대한 이동 시간과 거리를 계산하여 Klacks의 경로 최적화를 위한 기반을 제공합니다. 이 페이지의 유일한 입력란에 개인 API 키를 입력합니다. 무료 키는 페이지에 연결된 openrouteservice.org 등록 링크를 통해 받을 수 있습니다.

## 알아두면 좋은 점

- 이 키는 브라우저에서 openrouteservice.org로 직접 전송되기 때문에, 이메일 비밀번호 등과 달리 마스킹(`***`) 처리 없이 의도적으로 **평문으로** 표시됩니다. 이는 실수로 잘못 입력했거나 만료된 키를 알아볼 수 있도록 하기 위한 설계이며, 표시 오류가 아닙니다.
- 그럼에도 불구하고 데이터베이스에는 이 키가 암호화되어 저장됩니다.
- 유효한 키가 없으면 경로 최적화가 작동하지 않습니다 — 이동 시간과 거리를 계산할 수 없게 됩니다.
- 주소를 좌표로 변환하는 지오코딩(Geocoding)에는 Klacks가 별도의 다른 서비스를 사용합니다 — 이 키는 오직 경로 계산에만 사용됩니다.
