설정 > 사용자별 그룹 표시 권한에서 개별 사용자가 어떤 그룹(및 그에 속한 데이터)을 볼 수 있는지 지정합니다.

## 작동 방식

목록에는 모든 사용자의 이름과 그들에게 할당된(표시 가능한) 루트 그룹의 개수가 표시됩니다. 숫자를 클릭하면 모든 루트 그룹의 체크박스 목록이 있는 폼이 열리며, 여기서 해당 사용자가 볼 수 있는 그룹을 표시합니다. 루트 그룹을 허용하면 그 하위 그룹 전체에도 자동으로 적용됩니다.

관리자는 이 제한에서 제외됩니다. 관리자의 경우 버튼이 비활성화되어 있으며, 표시되는 숫자는 항상 전체 루트 그룹 수와 일치합니다 — 관리자는 원칙적으로 모든 것을 볼 수 있습니다.

## 알아두면 좋은 점

- 할당이 전혀 없는 일반(관리자가 아닌) 사용자의 경우 "할당된 그룹 0개"는 다음을 의미합니다: 이 사용자는 그룹 범위가 지정된 영역(예: 대시보드, 고객 목록)에서 현재 **아무것도** 볼 수 없습니다 — 이는 오작동이 아니라 할당이 없기 때문에 발생하는 직접적인 결과입니다.
- 예외: 시스템에 현재 그룹이 **전혀** 존재하지 않으면 모든 사용자가 모든 것을 볼 수 있습니다 — 그룹 구조가 없는 설치 환경에서는 이 설정을 관리할 필요가 없습니다.
- 여기서 할당을 하기 전에 먼저 기준 데이터(Stammdaten)에서 그룹 구조를 만드세요 — 기존 그룹이 없으면 여기서 체크할 항목이 없습니다.
- 이 설정은 데이터의 **표시 여부(가시성)**를 제어하는 것이며, 역할의 근본적인 **권한**을 제어하는 것이 아닙니다 — 역할/권한은 이와 별개로 관리됩니다.
