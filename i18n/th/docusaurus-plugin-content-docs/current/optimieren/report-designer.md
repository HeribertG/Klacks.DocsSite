ด้วย Report Designer คุณสามารถออกแบบเทมเพลตสำหรับพิมพ์ของคุณเอง — ตั้งแต่ตารางเวรไปจนถึงรายชื่อที่อยู่ — ได้โดยตรงใน Klacks โดยไม่ต้องใช้เครื่องมือภายนอก

## วิธีการทำงาน

รายงานหนึ่งฉบับประกอบด้วยสามส่วน ซึ่งคุณแก้ไขได้ในแท็บ *ออกแบบ*:

- **Header** — พิมพ์อยู่ด้านบนของทุกหน้า มีสามโซน (ซ้าย กลาง ขวา) ซึ่งสามารถใส่ฟิลด์จากฐานข้อมูล ข้อความอิสระ หรือรูปภาพ (เช่น โลโก้บริษัทของคุณที่อัปโหลดผ่านระบบจัดการรูปภาพ)
- **Body** — ประกอบด้วยตารางข้อมูลหนึ่งตารางขึ้นไป คุณเพิ่มคอลัมน์ได้ด้วยการลากวาง (Drag & Drop) จากรายการฟิลด์ และปรับลำดับคอลัมน์ด้วยการลากวางเช่นกัน ส่วนความกว้างปรับผ่านช่องกรอกเปอร์เซ็นต์ของแต่ละคอลัมน์
- **Footer** — พิมพ์อยู่ด้านล่างของทุกหน้า

แต่ละตารางเชื่อมโยงกับ **แหล่งข้อมูล** ซึ่งคุณเลือกได้ในแท็บ *แหล่งข้อมูล*: ตารางเวร (Schedule) ที่อยู่แบบรายการหรือรายละเอียด การลา กะงาน กลุ่ม หรือ Container Template แหล่งข้อมูลเป็นตัวกำหนดว่าฟิลด์ใดจะใช้ได้บ้าง

สำหรับการออกแบบมีตัวเลือกดังนี้:
- **การจัดรูปแบบ** สำหรับแต่ละเซลล์และข้อความอิสระ: แบบอักษร ขนาด ตัวหนา/ตัวเอียง/ขีดเส้นใต้ สีข้อความ การจัดตำแหน่ง
- **เส้นขอบเซลล์** กำหนดแยกแต่ละด้าน (บน/ขวา/ล่าง/ซ้าย) พร้อมรูปแบบและสีของเส้น
- **แถวสรุปท้ายตาราง** พร้อมฟิลด์ SUM: คอลัมน์ตัวเลขจะถูกรวมยอดโดยอัตโนมัติ
- **แถวข้อความอิสระ** ก่อนหรือหลังตารางข้อมูล — เช่น สำหรับหมายเหตุ เงื่อนไข หรือช่องลายเซ็น
- **การตั้งค่าหน้ากระดาษ** ในแท็บ *ทั่วไป*: แนวตั้งหรือแนวนอน ขนาดกระดาษ (A4, A3, B4/B5 JIS, Letter) และระยะขอบหน้ากระดาษเป็นมิลลิเมตร

## ข้อมูลที่ควรทราบ

- **Merged Fields:** ผ่านแถบเครื่องมือของฟิลด์ที่เลือก คุณสามารถรวมหลายฟิลด์ไว้ในคอลัมน์เดียวกันได้ โดยตัวคั่น (จุลภาค เว้นวรรค ขึ้นบรรทัดใหม่ ขีดกลาง) สามารถกำหนดเองได้
- สำหรับแหล่งข้อมูล **ตารางเวร** มีตัวเลือกเพิ่มเติมสองอย่าง: *รวมรายการต่อวัน* (แสดงหลายรายการในวันเดียวกันไว้ในแถวเดียว) และ *แสดงช่วงเวลาเต็ม* (แสดงทุกวัน แม้ไม่มีรายการ)
- เปิดใช้งานแถวสรุปท้ายตารางได้ผ่านช่องกาเครื่องหมายที่อยู่ใต้ตารางนั้นโดยตรง ส่วนป้ายกำกับ SUM ของฟิลด์แถวสรุปสามารถแสดงหรือซ่อนแยกกันได้
