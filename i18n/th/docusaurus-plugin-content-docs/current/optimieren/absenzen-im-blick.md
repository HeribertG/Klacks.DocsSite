# ติดตามการขาดงาน

วันหยุดพักผ่อน การลาป่วย การอบรม การเกณฑ์ทหาร — ใครขาดงานเมื่อไหร่ คุณจะเห็นได้ใน Klacks เป็นแผนภูมิแกนต์ (Gantt) ที่ชัดเจนตลอดทั้งปี

## วิธีการทำงาน

- การขาดงานแต่ละครั้งจะแสดงเป็นแท่งสีบนเส้นเวลาของแต่ละบุคคล — การซ้อนทับและการกระจุกตัวจะเห็นได้ชัดเจนทันที
- คุณสามารถกำหนด **ประเภทการขาดงานของตัวเอง** ได้เอง พร้อมสีและพฤติกรรม (มีเงินเดือน/ไม่มีเงินเดือน มีผลต่อชั่วโมงทำงาน/ไม่มีผล)
- การส่งออกเป็น **PDF** ทำได้ในคลิกเดียว — สะดวกสำหรับการติดประกาศหรือการเก็บเอกสาร
- การขาดงานส่งผลโดยตรงต่อการวางแผน: ผู้ที่ขาดงานจะไม่ถูกจัดเข้าแผนโดย [วางแผนอัตโนมัติ: แผนงานเสร็จในไม่กี่วินาที](../planen/auto-planung.md) และใน [กริดแผนงาน: เมทริกซ์เวลาแบบโต้ตอบของคุณ](../planen/plan-raster.md) ความขัดแย้งจะปรากฏให้เห็นทันที

## ข้อมูลที่ควรทราบ

- ภาพรวมแบบแกนต์เหมาะสำหรับการวางแผนวันหยุดของทีม: เห็นช่วงที่คนขาดพร้อมกันตลอดทั้งฤดูร้อนได้ในภาพเดียว
- หากมีใครขาดงานกะทันหัน Klacks จะแสดงตัวแทนที่มีคุณสมบัติและพร้อมทำงานให้ทันที โดยไม่ต้องล้มแผนงานทั้งหมด
- ค่าการขาดงาน (เช่น การลงบันทึกวันหยุดเต็มวันหรือครึ่งวัน) จะถูกรวมเป็นผลรวมในรายงานสรุปชั่วโมงทำงาน
