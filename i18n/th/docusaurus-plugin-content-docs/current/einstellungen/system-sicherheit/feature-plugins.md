# Feature-Plugins

Feature-Plugins ช่วยขยายขีดความสามารถของ Klacks ด้วยฟังก์ชันเสริมที่ไม่ได้เป็นส่วนหนึ่งของผลิตภัณฑ์หลัก — เช่น Messaging (การส่งและรับข้อความผ่าน Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat และ Zalo) หรือ Floor Plan (เครื่องมือแก้ไขผังพื้นที่แบบเห็นภาพ พร้อมจุดทำเครื่องหมายพื้นที่ทำงานและการกำหนดกะ)

## วิธีการทำงาน

**จัดการปลั๊กอินที่ติดตั้งแล้ว:** ในเมนู การตั้งค่า > Feature-Plugins คุณจะเห็นรายการปลั๊กอินที่ติดตั้งไว้แล้ว แต่ละแถวสามารถเปิดหรือปิดปลั๊กอินได้ด้วยสวิตช์ (เปิดใช้งาน/ปิดใช้งาน) โดยไม่ต้องถอนการติดตั้ง หรือจะลบออกทั้งหมดผ่านการดำเนินการ "ถอนการติดตั้ง" ก็ได้

**ค้นหาปลั๊กอินใหม่:** คลิกปุ่มแว่นขยายเพื่อเปิดกล่องโต้ตอบ Marketplace ซึ่งคุณสามารถค้นหาปลั๊กอินอื่น ๆ ที่มีให้ใช้งานและติดตั้งได้ทันที

## ข้อมูลที่ควรทราบ

- ปลั๊กอินที่ถูกปิดใช้งานจะยังคงติดตั้งอยู่ แต่ถูกล็อกไว้สำหรับผู้ใช้งานทุกคน หน้าที่เกี่ยวข้องจะเข้าถึงไม่ได้ และ API endpoint ที่เกี่ยวข้องจะตอบกลับด้วยสถานะ 404 ราวกับว่าไม่มีอยู่จริง
- หลังจากเปิด/ปิดใช้งาน หรือติดตั้ง/ถอนการติดตั้ง Klacks จะโหลดคำแปลใหม่โดยอัตโนมัติ เพื่อให้ข้อความของปลั๊กอินใหม่ปรากฏขึ้นทันทีโดยไม่ต้องโหลดหน้าใหม่
- ปลั๊กอินบางตัวมาพร้อมกับการ์ดการตั้งค่าของตัวเอง (เช่น การกำหนดค่าผู้ให้บริการ Messaging) — การ์ดเหล่านี้จะปรากฏในจุดอื่นของการตั้งค่าหลังจากติดตั้งเท่านั้น
