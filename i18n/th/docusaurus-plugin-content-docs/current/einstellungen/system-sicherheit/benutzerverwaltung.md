# การจัดการผู้ใช้งาน

ในเมนู การตั้งค่า > การจัดการผู้ใช้งาน คุณสามารถจัดการ **บัญชีล็อกอิน** (ผู้ใช้งานระบบ) ที่บุคคลต่าง ๆ ใช้เข้าสู่ระบบ Klacks ได้ — ซึ่งไม่เหมือนกับการจัดการพนักงานหรือที่อยู่ เพราะบัญชีล็อกอินหนึ่งบัญชีอาจเชื่อมโยงหรือไม่เชื่อมโยงกับการ์ดข้อมูลหลักของบุคคลก็ได้

## วิธีการทำงาน

**เพิ่มผู้ใช้งาน:** กล่องโต้ตอบ "+ เพิ่มผู้ใช้งาน" จะสร้างข้อเสนอชื่อผู้ใช้โดยอัตโนมัติจากชื่อและนามสกุล (แต่ละส่วนต้องมีอย่างน้อย 2 ตัวอักษร) ที่อยู่อีเมลเป็นฟิลด์บังคับและต้องมีเครื่องหมาย @ อยู่ด้วย จะสามารถบันทึกได้ก็ต่อเมื่อกรอกทุกฟิลด์ถูกต้องครบถ้วนแล้ว เมื่อสร้างบัญชีระบบจะสร้างรหัสผ่านโดยอัตโนมัติ — เพื่อให้บุคคลนั้นตั้งรหัสผ่านของตัวเองได้ ให้ใช้ "รีเซ็ตรหัสผ่าน" ในแถวของบัญชีนั้นหลังจากสร้างแล้ว

**แก้ไขผู้ใช้งาน:** ดับเบิลคลิกที่แถวชื่อเพื่อเปิดกล่องโต้ตอบแก้ไข ในนั้นสามารถเปลี่ยนชื่อผู้ใช้ในภายหลังได้ — ตอนสร้างใหม่ชื่อผู้ใช้เป็นเพียงข้อเสนอเท่านั้น

**รีเซ็ตรหัสผ่าน:** ลิงก์ "รีเซ็ตรหัสผ่าน" ในแถวของผู้ใช้จะส่งอีเมลไปยังที่อยู่ที่บันทึกไว้ ซึ่งบุคคลนั้นสามารถใช้ตั้งรหัสผ่านใหม่ด้วยตนเองได้

**ลบผู้ใช้งาน:** ปุ่มถังขยะใช้ลบบัญชีล็อกอินอย่างถาวร หลังจากยืนยันในกล่องโต้ตอบลบ

**กำหนดสิทธิ์:** ใช้เมนูดรอปดาวน์ในแถวเพื่อกำหนดสิทธิ์ **Supervisor** หรือ **Admin** การเปลี่ยนแปลงมีผลทันที โดยไม่ต้องคลิกบันทึกแยก รายละเอียดว่าทั้งสองบทบาทนี้ทำอะไรได้บ้างอธิบายไว้ที่หัวข้อ "บทบาทและสิทธิ์" ในส่วนเริ่มต้นใช้งาน — ที่นี่เกี่ยวข้องเฉพาะการจัดการบัญชีเท่านั้น

## ข้อมูลที่ควรทราบ

- ผู้ใช้งานที่ล็อกอินอยู่ในปัจจุบันจะแสดงอยู่ในรายการด้วย แต่ไม่สามารถแก้ไข เปลี่ยนสิทธิ์ หรือลบตัวเองได้จากที่นั่น
- บัญชีล็อกอินเป็นอิสระจากการจัดการบุคคล/พนักงาน — คุณสามารถสร้างบัญชีสำหรับบริษัทไอทีภายนอกได้ เป็นต้น โดยไม่ต้องสร้างการ์ดพนักงานสำหรับบัญชีนั้น
- ไม่มีการตั้งรหัสผ่านโดยตรงจากผู้ดูแลระบบ บัญชีใหม่จะได้รับรหัสผ่านที่สร้างขึ้นโดยอัตโนมัติ การเปลี่ยนแปลงในภายหลังทุกครั้งต้องทำผ่านลิงก์อีเมล "รีเซ็ตรหัสผ่าน"
