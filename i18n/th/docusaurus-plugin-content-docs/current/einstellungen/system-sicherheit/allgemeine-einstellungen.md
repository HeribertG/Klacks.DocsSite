# การตั้งค่าทั่วไป

ในเมนู การตั้งค่า > ทั่วไป คุณสามารถกำหนดอัตลักษณ์พื้นฐานของการติดตั้ง Klacks ของคุณได้ ได้แก่ ชื่อแอปพลิเคชัน ไอคอน และโลโก้ — รวมถึงส่วนแยกต่างหากในระดับเดียวกันของการตั้งค่า ซึ่งก็คือชุดสีของตารางแผนงาน

## วิธีการทำงาน

**ชื่อแอปพลิเคชัน:** ชื่อที่กรอกในที่นี้จะถูกนำไปใช้เป็นชื่อแท็บเบราว์เซอร์ของแอปพลิเคชันทันที และปรากฏในหลายจุดของหน้าจอ เช่น ในอีเมลที่ส่งอัตโนมัติ

**ไอคอน:** อัปโหลดไฟล์ Favicon ของคุณเอง (`.ico` หรือ `.png`) ได้ด้วยการลากวางหรือเลือกไฟล์ ไอคอนนี้จะแทนที่ไอคอนมาตรฐานในแท็บเบราว์เซอร์ คลิก "ลบ" เพื่อรีเซ็ตกลับเป็น `favicon.ico`

**โลโก้:** อัปโหลดโลโก้ของคุณเอง (`.jpeg`, `.png` หรือ `.gif`) ได้ด้วยวิธีเดียวกัน คือลากวางหรือเลือกไฟล์ Klacks จะปรับขนาดตามสัดส่วนโดยอัตโนมัติสำหรับการแสดงผล คลิก "ลบ" เพื่อลบโลโก้ออก

### ชุดสีของตารางแผนงาน

ในส่วนถัดไปของหน้าการตั้งค่าเดียวกันนี้ คุณสามารถกำหนดสีที่ใช้ระบายเซลล์ต่าง ๆ ในตารางแผนงานได้ เช่น สำหรับวันเสาร์ วันอาทิตย์ วันหยุดนักขัตฤกษ์ วันหยุดราชการ ช่วงเวลาที่ปิดผนึก (เสร็จสมบูรณ์แล้ว) การเปลี่ยนกะ ค่าตอบแทนพิเศษ รวมถึงสีกรอบและสีตัวอักษร แต่ละสีสามารถเปลี่ยนได้แยกกันผ่านตัวเลือกสี โดยรายการจะแสดงชื่อของแต่ละรายการพร้อมสีที่เลือกอยู่ในปัจจุบัน

## ข้อมูลที่ควรทราบ

- การเปลี่ยนแปลงชื่อ ไอคอน และโลโก้จะมีผลทันที โดยไม่ต้องคลิกบันทึกแยก — ไม่มีฟอร์มที่มีตัวเลือกยกเลิก
- ชุดสีที่เลือกอย่างเหมาะสมช่วยให้มองเห็นได้ง่ายและรวดเร็วในตารางแผนงาน เช่น การแยกวันหยุดสุดสัปดาห์และวันหยุดนักขัตฤกษ์ออกจากวันทำงานได้ในทันทีที่มองเห็น — มีประโยชน์อย่างยิ่งสำหรับทีมงานขนาดใหญ่ที่มีจำนวนแถวมาก
- สีเหล่านี้มีผลเฉพาะต่อการแสดงผลของตารางแผนงานเท่านั้น ไม่มีผลต่อส่วนอื่น ๆ ของหน้าจอ
