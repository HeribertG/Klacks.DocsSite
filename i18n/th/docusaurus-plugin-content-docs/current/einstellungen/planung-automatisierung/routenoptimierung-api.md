ที่การตั้งค่า > OpenRoute คุณสามารถกรอก API-Key สำหรับบริการภายนอก openrouteservice.org ซึ่ง Klacks ใช้ในการปรับเส้นทางให้เหมาะสม (Routenoptimierung)

## วิธีการทำงาน

OpenRouteService คำนวณเวลาเดินทางและระยะทาง — สำหรับการขับรถ ปั่นจักรยาน และเดินเท้า — และเป็นพื้นฐานสำหรับการปรับเส้นทางให้เหมาะสมใน Klacks คุณกรอก API-Key ส่วนตัวของคุณในช่องเดียวของหน้านี้ คุณสามารถขอรับ Key ฟรีได้ผ่านลิงก์ลงทะเบียนที่แสดงอยู่บนหน้านี้ ซึ่งเชื่อมไปยัง openrouteservice.org

## ข้อมูลที่ควรทราบ

- Key จะถูกส่งจากเบราว์เซอร์ไปยัง openrouteservice.org โดยตรง ดังนั้นจึงตั้งใจแสดงเป็น **ข้อความธรรมดา** แทนที่จะปิดบัง (`***`) — ต่างจากรหัสผ่านอีเมลเป็นต้น การออกแบบเช่นนี้เพื่อให้คุณสามารถสังเกตเห็นการพิมพ์ผิดหรือ Key ที่หมดอายุได้ ไม่ใช่ข้อผิดพลาดในการแสดงผล
- ถึงอย่างนั้น Key ยังคงถูกจัดเก็บแบบเข้ารหัสในฐานข้อมูล
- หากไม่มี Key ที่ใช้งานได้ ระบบปรับเส้นทางให้เหมาะสมจะไม่ทำงาน — จะไม่สามารถคำนวณเวลาเดินทางและระยะทางได้
- สำหรับการแปลงที่อยู่เป็นพิกัด (Geocoding) เพียงอย่างเดียว Klacks ใช้บริการอื่นแยกต่างหาก — Key นี้ใช้สำหรับการคำนวณเส้นทางเท่านั้น
