# บุคลิกภาพของ Klacksy

Klacksy ไม่มีตัวตนที่ถูกกำหนดไว้ตายตัวในโค้ด ที่ การตั้งค่า > บุคลิกภาพของ Klacksy คุณสามารถกำหนดด้วยคำพูดของคุณเองว่า Klacksy ควรมีพฤติกรรม พูดจา และตอบสนองต่อสถานการณ์ต่าง ๆ อย่างไร

## วิธีการทำงาน

**ช่องข้อความหกช่อง:** บุคลิกภาพแบ่งออกเป็นหกส่วน แต่ละส่วนมีช่องข้อความอิสระของตัวเอง

- **ตัวตน** — Klacksy คือใคร บทบาท จุดประสงค์ และบริบท
- **บุคลิกภาพและแรงขับเคลื่อน** — ลักษณะนิสัย ทัศนคติพื้นฐาน สิ่งที่ขับเคลื่อน Klacksy
- **น้ำเสียง** — Klacksy ฟังดูเป็นอย่างไร เป็นทางการหรือเป็นกันเอง กระชับหรือละเอียด
- **ขอบเขต** — สิ่งที่ Klacksy ห้ามทำ การคุ้มครองข้อมูล หัวข้อต้องห้าม ประเด็นที่ละเอียดอ่อน
- **รูปแบบการสื่อสาร** — โครงสร้างของคำตอบ วิธีจัดการกับความไม่แน่นอน การถามกลับ
- **คุณค่าและความต่อเนื่อง** — คุณค่าหลักและวิธีที่ Klacksy พัฒนาไปตามกาลเวลา

**การบันทึก:** แต่ละช่องจะถูกบันทึกโดยอัตโนมัติเมื่อคุณออกจากช่องนั้น (เมื่อเสียโฟกัส) — ไม่จำเป็นต้องมีปุ่มบันทึกแยกต่างหาก การเปลี่ยนแปลงจะมีผลทันทีตั้งแต่การสนทนาครั้งถัดไปกับ Klacksy

## ข้อมูลที่ควรทราบ

- บุคลิกภาพนี้เป็นของเอเจนต์มาตรฐาน Klacksy ที่ใช้ร่วมกัน — การเปลี่ยนแปลงจะส่งผลต่อผู้ใช้ **ทุกคน** ไม่ใช่แค่บัญชีของคุณเอง
- ช่องข้อความเป็นข้อความอิสระล้วน ๆ ไม่มีข้อกำหนดด้านรูปแบบ — จากประสบการณ์ ประโยคที่สั้นและชัดเจนมักได้ผลดีกว่าคำบรรยายที่ยืดยาว
- ช่องที่ว่างเปล่าไม่ใช่ปัญหา Klacksy จะใช้ทัศนคติพื้นฐานที่มีมาในตัว ส่วนพฤติกรรมภายในบางอย่างจะถูกดูแลโดยระบบเองและไม่สามารถปรับตั้งได้โดยตรงที่นี่
