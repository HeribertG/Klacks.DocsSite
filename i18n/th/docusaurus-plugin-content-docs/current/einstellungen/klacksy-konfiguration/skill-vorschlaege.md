# ข้อเสนอแนะทักษะของ Klacksy

Klacksy เรียนรู้จากข้อผิดพลาดของตัวเอง หากคุณแก้ไขในแชทว่า Klacksy เลือกทักษะผิด ระบบจะจดจำการแก้ไขนั้นไว้ ที่ การตั้งค่า > ข้อเสนอแนะทักษะของ Klacksy ผู้ดูแลระบบจะเห็นว่ามีข้อเสนอแนะการปรับปรุงคำอธิบายทักษะใดบ้างที่เกิดขึ้นจากสิ่งนี้ และเป็นผู้ตัดสินใจว่าจะนำมาใช้หรือไม่

## วิธีการทำงาน

**ที่มาของข้อเสนอแนะ:** เมื่อ Klacksy เลือกทักษะ (Skill) ผิดสำหรับคำขอหนึ่งในแชท และผู้ใช้แก้ไขให้ถูกต้อง ลำดับการสนทนานั้นจะถูกทำเครื่องหมายว่า "ได้รับการแก้ไข" ตัวปรับปรุงภายในระบบจะวิเคราะห์ลำดับการสนทนาที่ถูกแก้ไขเหล่านี้ และสรุปออกมาเป็นการเปลี่ยนแปลงข้อความที่ชัดเจนสำหรับคำอธิบายทักษะ โดยมีเป้าหมายให้ Klacksy เลือกทักษะที่เกี่ยวข้องได้ถูกต้องในครั้งถัดไป

**การสร้างข้อเสนอแนะ:** ปุ่ม "สร้างข้อเสนอแนะ" จะเริ่มการวิเคราะห์ลำดับการสนทนาที่ถูกแก้ไข 30 รายการล่าสุด และสร้างข้อเสนอแนะใหม่ขึ้นมาหากพบรูปแบบที่ชัดเจน

**การ์ดข้อเสนอแนะแต่ละใบแสดง:** ทักษะที่เกี่ยวข้องและฟิลด์ที่มีการเปลี่ยนแปลง การเปรียบเทียบก่อน/หลังของการเปลี่ยนแปลง เหตุผลว่าทำไมการเปลี่ยนแปลงนี้จึงสมเหตุสมผล รวมถึง — ที่สามารถขยายดูได้ — คำขอจริงของผู้ใช้ที่เป็นต้นเหตุของข้อเสนอแนะนี้

**การตัดสินใจ:** ข้อเสนอแนะที่ยังค้างอยู่แต่ละรายการจะถูก **นำมาใช้** (คำอธิบายทักษะจะถูกเปลี่ยนทันที) หรือ **ปฏิเสธ** (ข้อเสนอแนะจะถูกยกเลิกโดยไม่มีการเปลี่ยนแปลงใด ๆ) จะไม่มีการนำมาใช้โดยอัตโนมัติ — ต้องมีมนุษย์เป็นผู้ตัดสินใจร่วมด้วยเสมอ

## ข้อมูลที่ควรทราบ

- หากไม่มีการแก้ไขในแชท ก็จะไม่มีข้อเสนอแนะเกิดขึ้น — ระบบต้องการการแก้ไขข้อผิดพลาดจริงเป็นฐานข้อมูลสำหรับการฝึกฝน ไม่ใช่แค่การใช้งานทั่วไป
- ฟังก์ชันนี้เปลี่ยนแปลงเฉพาะข้อความคำอธิบายของทักษะเพื่อให้ Klacksy เลือกได้ถูกต้องเท่านั้น ไม่ได้เปลี่ยนสิ่งที่ทักษะนั้นทำจริง ๆ
- ฟังก์ชันนี้ต้องใช้สิทธิ์ผู้ดูแลระบบ
