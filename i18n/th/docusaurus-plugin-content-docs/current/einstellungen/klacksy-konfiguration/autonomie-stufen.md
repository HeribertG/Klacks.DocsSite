# ระดับความเป็นอิสระ (Autonomie-Stufen)

Klacksy ควรตัดสินใจทำสิ่งต่าง ๆ ด้วยตัวเองได้มากแค่ไหน ก่อนที่จะต้องถามคุณก่อน คุณกำหนดเรื่องนี้ได้เป็นการส่วนตัวที่ การตั้งค่า > ระดับความเป็นอิสระของ Klacksy — การตั้งค่านี้ผูกกับบัญชีผู้ใช้ของแต่ละคน ไม่ใช่การตั้งค่าร่วมกันของผู้ใช้ทุกคน

## วิธีการทำงาน

**สี่ระดับ:**

- **เสนอแนะ** — Klacksy จะเสนอแนะเท่านั้น การเปลี่ยนแปลงทุกอย่างต้องได้รับการยืนยันจากคุณ
- **ผู้ช่วย** — การกระทำที่ย้อนกลับได้จะดำเนินการทันที ส่วนที่เหลือทั้งหมดต้องได้รับการยืนยัน
- **อัตโนมัติ** (ค่าเริ่มต้น) — การกระทำทั้งหมดดำเนินการทันที ยกเว้นกรณีที่มีความเสี่ยงสูงเป็นพิเศษ
- **อัตโนมัติเต็มรูปแบบ** — แม้แต่แผนที่มีหลายขั้นตอนก็ดำเนินไปได้โดยไม่ต้องขออนุมัติระหว่างทาง

**ระดับความเป็นอิสระมีผลอย่างไร:** การกระทำทุกอย่างที่ Klacksy อาจดำเนินการจะถูกจัดประเภทตามความเสี่ยงไว้ภายในระบบ กล่าวคือ การกระทำที่เป็นการอ่านข้อมูลอย่างเดียวจะทำงานได้ทันทีเสมอ ส่วนการกระทำที่ย้อนกลับได้หรือผูกกับสถานการณ์จำลอง (เช่น ในแผนทดสอบ) ต้องมีระดับอย่างน้อยผู้ช่วย และการกระทำที่ย้อนกลับไม่ได้ต้องมีระดับอย่างน้อยอัตโนมัติ ส่วนการกระทำที่มีความละเอียดอ่อน เช่น การจัดการผู้ใช้ สิทธิ์การเข้าถึง และการเปลี่ยนแปลงการตั้งค่าความเป็นอิสระนี้เอง จะต้องได้รับการยืนยันอย่างชัดเจนเสมอ **ไม่ว่าจะเลือกระดับใดก็ตาม**

**ขั้นตอนการยืนยัน:** เมื่อการกระทำใดต้องได้รับการอนุมัติจากคุณ Klacksy จะไม่ดำเนินการทันที แต่จะพักไว้เป็นเวลา 5 นาที พร้อมสรุปให้ทราบว่าจะเกิดอะไรขึ้น ก็ต่อเมื่อคุณยืนยันด้วยคำพูดของคุณเองอย่างชัดเจนเท่านั้น Klacksy จึงจะดำเนินการที่พักไว้นั้น — ระบบจะไม่ยืนยันแทนคุณเองโดยเด็ดขาด

## ข้อมูลที่ควรทราบ

- ระดับความเป็นอิสระเป็นการตั้งค่าส่วนบุคคลต่อบัญชีผู้ใช้แต่ละคน — ผู้ใช้แต่ละคนสามารถเลือกระดับที่ตนสบายใจได้เอง
- ระดับที่สูงขึ้นไม่ได้หมายความว่าจะไม่มีการถามใด ๆ อีกเลย การกระทำที่มีความละเอียดอ่อนยังคงต้องได้รับการยืนยันเสมอ
- ตามค่าเริ่มต้น ระดับอัตโนมัติจะถูกเปิดใช้งาน ซึ่งตรงกับพฤติกรรมที่ Klacksy เคยมีอยู่แล้วก่อนที่จะมีการนำระดับความเป็นอิสระมาใช้
