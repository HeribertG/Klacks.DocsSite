ที่ การตั้งค่า > จุดนำเข้าจาก ERP คุณกำหนดค่าวิธีการนำเข้าคำสั่งซื้อจากระบบ ERP ภายนอกในรูปแบบไฟล์ XML เข้าสู่ Klacks โดยอัตโนมัติ

## วิธีการทำงาน

**สามช่องทางในการส่งข้อมูล:** ระบบ ERP สามารถวางไฟล์ไว้ในพื้นที่จัดเก็บของจุดรับข้อมูล (Klacks จะดึงไฟล์เหล่านั้นไปโดยอัตโนมัติตามตารางเวลา) หรือส่งไฟล์แบบ push ผ่าน HTTP-POST ไปยัง endpoint สำหรับนำเข้าโดยตรง (ยืนยันตัวตนด้วยโทเคนการเข้าถึง) หรือผู้ดูแลระบบอัปโหลดไฟล์ XML ด้วยตนเองในหน้านี้ผ่านการลากและวาง

**จุดรับข้อมูล (Drop-Point)** เป็นเสมือนตู้จดหมายกลางที่ Klacks จัดการเองสำหรับไฟล์นำเข้าทั้งหมด โดยแสดงสามส่วน คือ ขาเข้า (รอการประมวลผลรอบถัดไป) ประมวลผลแล้ว (คลังเก็บ) และข้อผิดพลาด (ไฟล์ที่ไม่ถูกต้องพร้อมเหตุผล และสามารถนำเข้าซ้ำได้) ตารางเวลา (นิพจน์ Cron ค่าเริ่มต้นคือทุกชั่วโมง) และสวิตช์เปิด/ปิดใช้งานควบคุมการดึงข้อมูลอัตโนมัติ

**โทเคนการเข้าถึง** จำเป็นเฉพาะเมื่อระบบ ERP ส่งไฟล์แบบ push เท่านั้น โทเคนแต่ละตัวเริ่มต้นด้วย `klacks_erp_` มีอายุการใช้งาน 1–730 วัน (ค่าเริ่มต้น 365 วัน) และจะแสดงเป็นข้อความธรรมดา **เพียงครั้งเดียวตอนสร้าง** เท่านั้น โทเคนนี้ใช้ได้เฉพาะกับ endpoint สำหรับนำเข้าเท่านั้น ไม่ใช่สำหรับการเข้าถึง API ส่วนอื่น และสามารถเพิกถอนได้ตลอดเวลา

**การประมวลผล:** คำสั่งซื้อจะถูกจับคู่กับลูกค้าที่มีอยู่แล้วหรือลูกค้าที่สร้างใหม่ โดยอ้างอิงจากรหัสระบบต้นทางและหมายเลขอ้างอิงลูกค้าภายนอก (หรือใช้ชื่อบริษัท/รหัสไปรษณีย์/ถนนแทนหากไม่มี) และจะถูกสร้างขึ้นเป็นฉบับร่างก่อน ซึ่งผู้วางแผนต้องตรวจสอบและปิดผนึก การส่งซ้ำที่มีการเปลี่ยนแปลงของคำสั่งซื้อที่ปิดผนึกไปแล้ว จะปิดคำสั่งซื้อเดิมโดยอัตโนมัติและสร้างเวอร์ชันใหม่ที่เชื่อมโยงกัน ส่วนการส่งซ้ำที่ไม่มีการเปลี่ยนแปลงจะไม่ก่อให้เกิดผลใด ๆ

## ข้อมูลที่ควรทราบ

- คำสั่งซื้อรายการที่มีข้อผิดพลาด (เช่น ไม่มีวันในสัปดาห์ หรือมีหมายเลขอ้างอิงซ้ำ) จะถูกปฏิเสธ โดยไม่ปิดกั้นคำสั่งซื้อที่ถูกต้องรายการอื่นในไฟล์เดียวกัน แต่ไฟล์ทั้งหมดจะยังคงถูกจัดไว้ในส่วนข้อผิดพลาด เพื่อไม่ให้มีสิ่งใดถูกมองข้าม
- ไฟล์ที่ประมวลผลแล้วจะถูกเก็บเป็นคลังจดหมายเหตุ ไม่ถูกลบทิ้ง
- รูปแบบ XML ที่ชัดเจน (ฟิลด์ที่จำเป็น รูปแบบวันที่ ตรรกะวันในสัปดาห์) มีเอกสารอธิบายไว้ในคู่มือบนหน้าการตั้งค่าโดยตรง รวมถึงมีไฟล์ตัวอย่างให้ดาวน์โหลดได้
- การกำหนดค่านี้อยู่ในส่วนการตั้งค่า จึงสงวนไว้สำหรับบทบาทที่มีสิทธิ์ผู้ดูแลระบบเท่านั้น
