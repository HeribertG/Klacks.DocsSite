# ทัวร์แนะนำการตั้งค่าเบื้องต้น

เพิ่งเริ่มใช้ Klacks ใช่ไหม? ไม่ต้องเสียเวลาค้นหาเอง เพราะ Klacksy ผู้ช่วยที่ติดตั้งมาในตัวจะพาคุณไปทีละขั้นตอนตลอดการตั้งค่าเบื้องต้น

## วิธีการทำงาน

คุณสามารถเริ่มทัวร์นี้ได้โดยตรงในหน้าแชทกับ Klacksy (เช่น พิมพ์ "แสดงแอปให้ดูหน่อย" หรือกดจากคำแนะนำที่ปรากฏตอนเริ่มใช้งานครั้งแรก) จากนั้นระบบจะพาคุณไล่ผ่านจุดสำคัญต่าง ๆ ทีละจุด:

- Klacksy **อธิบายแต่ละส่วน** ด้วยประโยคสั้น ๆ ไม่กี่ประโยค — ตั้งแต่การตั้งค่าพื้นฐานของบริษัทและที่อยู่ ปฏิทิน การจัดการผู้ใช้งานและกลุ่ม ไปจนถึงพนักงาน เวร ความพร้อมให้บริการ การขาดงาน วันหยุด การปิดรอบระยะเวลา การตั้งค่าอีเมล และการตั้งค่า Klacksy/ปลั๊กอิน — รวมทั้งหมด 16 จุด
- **ไอคอนนำทางที่เกี่ยวข้องจะกะพริบ** อยู่ในแถบด้านข้าง เพื่อให้คุณเห็นเสมอว่าตอนนี้อยู่ตรงไหน
- คุณเป็นผู้กำหนดจังหวะเอง: ไปต่อ ข้ามจุดใดจุดหนึ่ง หรือจบทัวร์เมื่อไหร่ก็ได้

ด้วยวิธีนี้ คุณจะได้ภาพรวมที่สมบูรณ์ของแอปพลิเคชันทีละขั้นตอน โดยไม่ต้องอ่านคู่มือเลย

## ข้อมูลที่ควรทราบ

- ทัวร์นี้สามารถทำซ้ำได้กี่ครั้งก็ได้ตามต้องการ — รวมถึงสำหรับสมาชิกทีมใหม่ด้วย
- ในแต่ละจุด คุณสามารถถาม Klacksy กลับได้ตลอด และเขาจะตอบตามบริบทของหน้าที่กำลังแสดงอยู่
- วิธีที่เร็วที่สุดในการลองทัวร์นี้คือใน Playground: เข้าสู่ระบบ เปิด Klacksy แล้วเริ่มพิมพ์ได้เลย
