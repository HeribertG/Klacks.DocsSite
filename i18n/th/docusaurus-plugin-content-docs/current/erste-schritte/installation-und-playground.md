# การติดตั้งและ Playground

คุณสามารถลองใช้ Klacks ได้ภายในหนึ่งนาที และติดตั้งได้ภายในไม่กี่นาที ไม่ต้องลงทะเบียน ไม่ต้องคุยกับฝ่ายขาย

## แบบที่ 1: Playground (ไม่ต้องติดตั้งอะไรเลย)

มีอินสแตนซ์ Klacks สาธารณะพร้อมข้อมูลตัวอย่างให้ทดลองใช้ที่ **[klacks-software.ch](https://klacks-software.ch:7643)**:

- **เข้าสู่ระบบ:** `admin@test.com` · **รหัสผ่าน:** `P@ssw0rt1`
- คุณจะได้รับสิทธิ์ผู้ดูแลระบบเต็มรูปแบบ — ลองใช้ทุกฟีเจอร์ได้เลย
- ข้อมูลทั้งหมดจะ **ถูกรีเซ็ตอัตโนมัติทุกวัน** กรุณาอย่ากรอกข้อมูลส่วนบุคคลจริง

## แบบที่ 2: บนเซิร์ฟเวอร์ของคุณเอง (On-Premise)

แพ็กเกจ On-Premise มาพร้อมทุกอย่างที่จำเป็น: Docker images, ตัวติดตั้ง, ฐานข้อมูล, HTTPS และการอัปเดตอัตโนมัติ

```bash
curl -fsSLO https://github.com/HeribertG/Klacks.Api/releases/latest/download/klacks-onprem.zip
unzip klacks-onprem.zip && cd onprem && ./install.sh
```

สำหรับ Windows ใช้ `install.ps1` แทน `install.sh` ตัวติดตั้งจะสร้างรหัสผ่านและใบรับรองให้เอง ดาวน์โหลด Docker images เวอร์ชันล่าสุด และรอจนกว่าทุกอย่างจะทำงานได้ การเข้าสู่ระบบครั้งแรกใช้ `admin@test.com` / `P@ssw0rt1` — **กรุณาเปลี่ยนรหัสผ่านนี้ทันทีหลังจากเข้าสู่ระบบครั้งแรก**

## ข้อมูลที่ควรทราบ

- บริการอัปเดตที่มาพร้อมระบบจะทำให้การติดตั้งของคุณทันสมัยอยู่เสมอโดยอัตโนมัติ — พร้อมสำรองข้อมูลก่อนการอัปเดตทุกครั้งและย้อนกลับอัตโนมัติหากเกิดปัญหา
- ข้อมูลของคุณจะอยู่บนโครงสร้างพื้นฐานของคุณเองทั้งหมด Klacks จะไม่ส่งข้อมูลกลับไปที่อื่น
- ข้อกำหนดคือต้องมีเซิร์ฟเวอร์ที่ติดตั้ง Docker มีการเชื่อมต่ออินเทอร์เน็ตขาออก (สำหรับ Docker images และการอัปเดต) และเปิดพอร์ต 80/443
