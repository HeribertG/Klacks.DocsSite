# การวางแผนแบบโมดูลาร์: แบ่งแยกแล้วปกครอง

แทนที่จะต้องจัดการแผนงานขนาดใหญ่ที่ยุ่งเหยิง คุณสามารถแบ่งมันออกเป็นแผนย่อยขนาดเล็กที่ชัดเจนได้ — และ Klacks จะเชื่อมโยงทุกอย่างเข้าด้วยกันโดยอัตโนมัติ

## วิธีการทำงาน

1. **แบ่งแยก** — จัดโครงสร้างตารางเวรงานขนาดใหญ่ให้เป็นส่วนย่อยที่จัดการได้ง่าย: แยกตามทีม แผนก หรือสถานที่
2. **วางแผนเพียงครั้งเดียว** — ผู้ที่ทำงานในหลายส่วนจะถูกจัดเข้าแผนเพียงครั้งเดียว และจะปรากฏอย่างถูกต้องในทุกที่โดยอัตโนมัติ
3. **ซิงค์อัตโนมัติ** — การเปลี่ยนแปลงหนึ่งครั้งจะอัปเดตทันทีในทุกที่

เคล็ดลับเบื้องหลังคือ: แผ่นแผนงานจะเก็บ **การอ้างอิงแทนที่จะเป็นสำเนา** กะงานหนึ่งกะจะมีอยู่เพียงชุดเดียวเท่านั้น — ไม่ว่าจะปรากฏในแผนย่อยกี่แผนก็ตาม ด้วยเหตุนี้การจองซ้ำจึงไม่มีทางเล็ดลอดไปได้โดยไม่มีใครสังเกตเห็น: ระบบตรวจสอบการชนกันที่ทำงานอยู่ตลอดเวลาจะตรวจพบการซ้อนทับทันที ก่อนที่จะถูกบันทึกอย่างถาวร

## ข้อมูลที่ควรทราบ

- งานของบุคคลที่ใช้ร่วมกันซึ่งอยู่นอกส่วนงานนั้น จะปรากฏในแผนย่อยเป็นรายการที่ถูกล็อก — ทำให้ผู้วางแผนแต่ละคนเห็นได้ว่า "คนของตน" ถูกใช้งานที่อื่นเมื่อใด โดยไม่สามารถแก้ไขแผนของผู้อื่นได้
- คุณควบคุมการมองเห็นแผนย่อยได้ผ่านสิทธิ์การเข้าถึงตามกลุ่ม
- การวางแผนแบบโมดูลาร์และ [การวางแผนอัตโนมัติ: แผนงานเสร็จในไม่กี่วินาที](../planen/auto-planung.md) เสริมกันได้ดี: แต่ละส่วนสามารถถูกวางแผนโดยอัตโนมัติได้ด้วยตนเอง
