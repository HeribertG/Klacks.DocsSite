# การวางแผนอัตโนมัติ: แผนงานเสร็จในไม่กี่วินาที

ส่วนที่ยุ่งยากที่สุดของการวางแผนเวรงาน — การจับคู่คนเข้ากับกะงาน — Klacks จัดการให้เพียงแค่กดปุ่มเดียว

## วิธีการทำงาน

เบื้องหลังการวางแผนอัตโนมัติแบบคลิกเดียวคือ **อัลกอริทึมพันธุกรรม (Genetic Algorithm)**: ระบบจะสร้างแผนหลายพันรูปแบบ ประเมินผล และนำแบบที่ดีที่สุดมาผสมผสานต่อไปเรื่อย ๆ จนกว่าจะได้ผลลัพธ์ที่สมดุล กฎเกณฑ์เข้มงวดจะมีความสำคัญเป็นอันดับแรกเสมอ จากนั้นจึงพิจารณาเป้าหมายอื่น ๆ:

- **กฎเกณฑ์** — กฎหมายแรงงาน เวลาพัก คุณสมบัติ และความพร้อมทำงาน เป็นสิ่งที่ต่อรองไม่ได้: ไม่ว่าผลลัพธ์จะดีเพียงใด ก็ห้ามละเมิดกฎเกณฑ์เข้มงวด
- **ความครอบคลุม** — ทุกกะงานจะถูกจัดให้มีจำนวนคนตามที่กำหนด
- **ความเป็นธรรม** — ชั่วโมงทำงานถูกกระจายอย่างสม่ำเสมอ และลำดับกะของแต่ละคนจะเป็นไปตามรูปแบบ เช้า → บ่าย → กลางคืน เท่าที่จะทำได้
- **ความต่อเนื่อง** — พนักงานจะยังคงอยู่ ณ สถานที่ปฏิบัติงานที่คุ้นเคยเท่าที่จะเป็นไปได้

คุณสามารถเริ่มการวางแผนอัตโนมัติได้โดยตรงจากตารางเวรสำหรับช่วงเวลาที่เลือก ผลลัพธ์จะปรากฏเป็นฉบับร่างแผนงานตามปกติ: ทุกอย่างยังสามารถปรับแก้ด้วยมือได้ ไม่มีสิ่งใดถูกยืนยันโดยไม่ได้รับอนุญาต

## ข้อมูลที่ควรทราบ

- คุณสมบัติทำหน้าที่เป็นข้อจำกัดที่แท้จริง: ผู้ที่ไม่มีคุณสมบัติตามที่กำหนดจะไม่ถูกจัดเข้าแผน
- การวางแผนอัตโนมัติเคารพรายการที่มีอยู่แล้ว — คุณจึงสามารถกำหนดจุดตายตัวก่อน แล้วปล่อยให้ระบบเติมส่วนที่เหลือได้
- หากไม่พอใจผลลัพธ์ ก็เพียงแค่สั่งวางแผนใหม่อีกครั้ง: อัลกอริทึมจะหาทางเลือกอื่นที่มีคุณภาพเทียบเท่ากัน
