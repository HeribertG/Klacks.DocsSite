# ยินดีต้อนรับสู่เอกสาร Klacks

**Klacks** คือแพลตฟอร์มโอเพนซอร์สสำหรับการวางแผนกำลังคนในธุรกิจที่ทำงานเป็นกะและมีการปฏิบัติงานภาคสนาม: วางแผนเวรอัตโนมัติ ปรับปรุงเส้นทางเดินรถให้เหมาะสม ควบคุมผ่านผู้ช่วย AI — ใช้งานแบบ On-Premise พร้อมเลือกโมเดล AI ได้ตามต้องการ เหมาะสำหรับหน่วยงานดูแลผู้ป่วยที่บ้าน โรงพยาบาล บริษัทรักษาความปลอดภัย บริษัททำความสะอาด และธุรกิจโลจิสติกส์

วิกินี้อธิบายการทำงานของ Klacks เป็นเนื้อหาสั้น ๆ ทีละส่วน แต่ละหน้าใช้เวลาอ่านเพียงไม่กี่นาที และทุกอย่างสามารถลองใช้งานได้ทันทีใน Playground สาธารณะ
