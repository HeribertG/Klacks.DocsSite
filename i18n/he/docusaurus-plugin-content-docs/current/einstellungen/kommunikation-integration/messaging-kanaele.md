בהגדרות > ספקי Messaging אתם מחברים ערוצי צ'אט ו-SMS חיצוניים, שדרכם Klacks (וגם Klacksy) יכולה לפנות ישירות לעובדים וללקוחות — במקום דוא"ל או בנוסף אליו.

## כך זה עובד

**עיקרון משותף:** כל הערוצים מוגדרים דרך אותה רשימה: שם פנימי, שם תצוגה, סוג ספק ופרטי גישה שונים בהתאם לערוץ (טוקן, מזהה חשבון, תעודה וכדומה), וכן מתג הפעלה/כיבוי וכפתור בדיקה לכל רשומה, כדי לבדוק את החיבור באופן מיידי. מי שההודעה מגיעה אליו נקבע לפי המזהה הספציפי לערוץ (מספר טלפון, שם משתמש, User-ID וכדומה) שהוזן אצל העובד או הלקוח תחת כתובת > פרטי קשר — אינכם צריכים להקליד בעצמכם מזהים גולמיים.

**לשם מה נעשה שימוש ב-Messaging ב-Klacks:** כערוץ התראה נוסף לצד דוא"ל, למשל עבור הודעות מלוח המשמרות או הודעות מ-Klacksy — בהתאם לערוצים המופעלים ולנתונים שהוגדרו לכל אדם.

**11 הערוצים במבט כללי:**

| ערוץ | שליחה/קבלה | ייחוד |
|---|---|---|
| Slack | שליחה + קבלה | יש להזמין את הבוט (Bot) במפורש לכל ערוץ באמצעות `/invite`. |
| Telegram | שליחה + קבלה | על הנמען להפעיל את הבוט בעצמו מראש באמצעות `/start`. |
| Signal | שליחה בלבד | דורש גשר (bridge) signal-cli מופעל עצמאית (Docker) עם מספר טלפון משלו המצומד באמצעות קוד QR — אין API רשמי מבוסס ענן. |
| Microsoft Teams | שליחה בלבד | Webhook של Power-Automate לכל ערוץ; אין הודעות 1:1, עבור מספר ערוצי Teams נדרשות מספר רשומות. |
| SMS | שליחה בלבד | מספר טלפון בפורמט E.164; בחשבונות ניסיון (Trial, למשל Twilio) רק למספרים שאומתו מראש, ומדינת היעד חייבת להיות מאושרת. |
| WeChat | שליחה בלבד | אימות כ-Official Account מחוץ לסין מורכב; מגיע רק לעוקבים (Followers) עם אינטראקציה ב-48 השעות האחרונות. |
| KakaoTalk | שליחה בלבד | Access Token תקף לכ-6 שעות בלבד ויש לחדש אותו ידנית; על הנמען להיות "חבר" של האפליקציה. |
| Line | שליחה + קבלה | מכסת שליחה חודשית מוגבלת בהתאם לתוכנית; מזהה הנמען נודע רק לאחר הודעה נכנסת. |
| Threema | שליחה בלבד | חשבון Threema-Gateway בתשלום (מודל יתרה/קרדיט); אין קבלה, מכיוון שלא נעשה שימוש במצב קצה-לקצה. |
| Viber | שליחה + קבלה | הבוט רשאי לפנות רק לאנשים שנרשמו אליו מראש; ה-Webhook דורש HTTPS נגיש באופן ציבורי. |
| Zalo | שליחה בלבד | Access Token פג תוקף לאחר כ-25 שעות, שליחה אפשרית רק בתוך חלון אינטראקציה. |

## טוב לדעת

- לכל ערוץ קיים מדריך מפורט משלו עם הנחיות שלב-אחר-שלב להגדרה אצל הספק הרלוונטי — ניתן לגישה ישירות בדף ההגדרות של רשומת הספק המתאימה.
- ניתן להפעיל מספר ערוצים במקביל; אילו מהם ייעשה בהם שימוש בפועל תלוי בפרטי הקשר שהוגדרו אצל האדם הרלוונטי.
- בכל הערוצים בעלי Access Token מוגבל בזמן (למשל KakaoTalk, Zalo) יש לחדש אותו מחוץ ל-Klacks אצל הספק ולעדכן אותו ב-Klacks, אחרת השליחה דרך ערוץ זה תיפסק.
