# תוספי תכונות

תוספי תכונות מרחיבים את Klacks בתחומי פונקציונליות אופציונליים שאינם חלק ממוצר הליבה — למשל Messaging (שליחה וקבלה דרך Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat ו-Zalo) או Floor Plan (עורך תוכנית קומה חזותי עם סמני עמדות עבודה ושיוך משמרות).

## איך זה עובד

**ניהול תוספים מותקנים:** תחת הגדרות > תוספי תכונות רואים רשימה של התוספים המותקנים כבר. בכל שורה מפעילים או מכבים את התוסף באמצעות מתג (מופעל/מושבת) מבלי להסירו, או מסירים אותו לחלוטין דרך פעולת ההסרה.

**מציאת תוספים חדשים:** דרך כפתור זכוכית המגדלת נפתח דיאלוג ה-Marketplace, שבו מחפשים תוספים נוספים זמינים ומתקינים אותם ישירות.

## טוב לדעת

- תוסף מושבת נשאר מותקן, אך חסום עבור כל המשתמשים: העמודים המשויכים אליו אינם נגישים ונקודות הקצה (API) המשויכות מחזירות תגובת 404, כאילו אינן קיימות.
- לאחר הפעלה/השבתה או התקנה/הסרה, Klacks טוען מחדש אוטומטית את התרגומים, כך שטקסטים חדשים של התוסף גלויים מיד, מבלי שתצטרכו לרענן את העמוד.
- חלק מהתוספים מביאים עמם כרטיסי הגדרות משלהם (למשל תצורת ספקי Messaging) — אלה מופיעים רק לאחר ההתקנה במקום אחר בהגדרות.
