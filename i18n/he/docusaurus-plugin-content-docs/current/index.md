# ברוכים הבאים לתיעוד של Klacks

**Klacks** היא הפלטפורמה הקוד-פתוח לתכנון כוח אדם בעסקי משמרות ותפעול: תכנון משמרות אוטומטי, אופטימיזציית מסלולים, שליטה באמצעות עוזר בינה מלאכותית - On-Premise, עם מודל ה-AI לבחירתכם. עבור ספיטקס (Spitex), בתי חולים, עסקי אבטחה, ניקיון ולוגיסטיקה.

ויקי זה מסביר בפרקים קטנים כיצד Klacks פועלת. כל עמוד נקרא תוך דקות ספורות - וכל דבר ניתן לנסות מיד ב-Playground הציבורי.
