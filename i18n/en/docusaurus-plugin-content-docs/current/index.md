# Welcome to the Klacks Documentation

**Klacks** is the open-source platform for workforce scheduling in shift-based and field-service operations: build rosters automatically, optimize routes, and run it all through an AI assistant — On-Premise, with the AI model of your choice. Built for home care, hospitals, security, cleaning, and logistics companies.

This wiki explains how Klacks works, in bite-sized pieces. Each page takes just a few minutes to read — and everything can be tried out right away in the public Playground.
