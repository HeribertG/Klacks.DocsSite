Under Settings > Organization > Qualifications you maintain the catalog of all qualifications that can be assigned to employees and clients in your business — from language skills to industry-specific professional qualifications.

## How it works

**Creating a qualification:** "+ Add qualification" opens a modal with a name (required, stored in multiple languages) and an optional description, each entered in the currently selected interface language.

**Icon:** Every qualification can be given an emoji as a visual symbol — handy for quick recognition in lists.

**Time-limited:** A toggle marks qualifications that have an expiry date (e.g. certificates that need to be renewed).

**Type:** Every qualification is either of type Language or Work.

**Country:** Optionally, a qualification can be assigned to one or more countries (e.g. country-specific driving licenses or professional certifications).

**Category:** Only selectable for qualifications of type Work — assigns the qualification to an industry (Home Care, Security, Logistics, Healthcare, Hospitality, Construction, Cleaning, Transport, or General). This category later serves as a filter, for example when searching for suitable employees.

**Searching, editing, and deleting:** A search field filters the list by name; clicking an entry opens the edit modal, and the trash icon deletes the qualification (with a confirmation prompt).

**Saving:** Every change is persisted immediately when saved in the modal — there is no shared save bar.

## Good to know

- This page only manages the **catalog** of qualifications themselves (name, icon, type, country, category). Assigning individual qualifications to employees or clients happens elsewhere (person or shift management).
- The category field only appears for type "Work" — language qualifications have no category.
