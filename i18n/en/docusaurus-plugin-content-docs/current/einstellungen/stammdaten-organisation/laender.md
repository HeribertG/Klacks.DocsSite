Under Settings > Organization > Countries you manage the list of countries available for selection in Klacks — for example, in the Head Office Address.

## How it works

**Countries list:** Each row shows a country's code, name (in the currently selected interface language), and phone prefix. All three fields are editable directly in the row — there is no separate edit modal.

**Adding a country:** "+ Add country" at the end of the list appends a new, empty row; you enter the code, name, and prefix directly.

**Searching:** A search field above the list filters by country name.

**Deleting:** The trash icon marks a country entry for deletion; a freshly added, not-yet-saved entry is removed from the list immediately.

**Saving:** Unlike the other cards in the Organization section (Branches, Contracts, Qualifications), changes to the countries list are not applied immediately, but are collected and saved or discarded via the settings page's shared save bar.

## Good to know

- The country name is stored in multiple languages — when you switch the interface language, the row automatically shows the name field for that language.
- The countries list feeds, among other things, the country selection in the Head Office Address — create a missing country here first before you want to select it there.
