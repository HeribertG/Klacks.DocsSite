A calendar selection bundles one or more holiday calendars (country + canton/state) into a named package, which you then assign to a contract or a group.

## How it works

The list shows each calendar selection with its name and the number of assigned calendar entries; clicking a row opens the edit form. In the form you choose a country and then check off the desired calendars — the country's national holiday calendar as well as any number of cantons/states. Selected entries appear as removable chips.

Klacks ships with pre-installed calendar selections: all 26 Swiss cantons individually, plus one calendar each for Austria, Germany, Italy, the USA, Liechtenstein, and France. These pre-installed selections are marked with a "System" badge.

You create your own calendar selections via "+ Add calendar selection" — for example, to bundle several cantons into a company-specific combination.

## Good to know

- Pre-installed ("System") calendar selections **cannot be deleted** — the trash icon isn't even shown for them. Editing (name, composition) remains possible for them.
- Custom calendar selections can be deleted as long as they are **not** currently used by a contract or a group — in that case the delete button is disabled and shows a reason as a tooltip.
- The assignment happens at the contract: there you set the desired calendar selection as the holiday calendar, which lets you, for example, map cantonally different public holidays.
- This page only bundles and names calendar rule sets. The holiday rules themselves (calculation formula, weekday shifting, etc.) are maintained separately — see "Holidays & Calendars."
