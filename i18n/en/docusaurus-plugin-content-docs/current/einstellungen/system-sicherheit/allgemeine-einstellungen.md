# General Settings

Under Settings > General you set the base identity of your Klacks installation: application name, icon, and logo — plus, as its own section at the same level of Settings, the color scheme of the schedule grid.

## How it works

**Application name:** The name entered here is immediately adopted as the application's browser title and appears in further places in the UI, for example in automatically sent emails.

**Icon:** You upload a custom favicon (`.ico` or `.png`) via drag-and-drop or file selection; it replaces the default icon in the browser tab. "Remove" resets it back to `favicon.ico`.

**Logo:** You likewise upload a custom logo (`.jpeg`, `.png`, or `.gif`) via drag-and-drop or file selection; Klacks scales it proportionally for display. "Remove" deletes it again.

### Color scheme of the schedule grid

Further down on the same settings page, you set the colors the schedule grid uses to shade individual cells — for example for Saturdays, Sundays, holidays, official public holidays, sealed (closed) periods, shift changes, surcharges, as well as border and font colors. Each color can be changed individually via a color picker; the list shows the label and the currently chosen color for each.

## Good to know

- Changes to name, icon, and logo take effect immediately, with no separate save click — there is no form with a cancel option.
- A generously chosen color scheme makes quick visual orientation in the schedule grid easier, for example to distinguish weekends and holidays from working days at a glance — especially helpful for large teams with many rows.
- These colors only affect the display of the schedule grid itself, not other areas of the UI.
