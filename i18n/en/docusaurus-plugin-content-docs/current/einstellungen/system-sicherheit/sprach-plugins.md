# Language Plugins

Klacks supports four core languages built into the frontend — German, English, French, Italian — plus 21 further languages as installable language plugins (language packs), including among others Arabic, Czech, Danish, Greek, Spanish, Finnish, Hebrew, Indonesian, Japanese, Korean, Malay, Norwegian (Bokmål), Dutch, Polish, Portuguese, Romanian, Swedish, Thai, Vietnamese, as well as Simplified and Traditional Chinese.

## How it works

**Overview:** Under Settings > Language Plugins you see all core languages (always active, not removable) as well as the language plugins already installed.

**Installing:** The magnifying-glass button opens the marketplace dialog, where you can search for and install further languages.

**Uninstalling:** The Uninstall action in the row removes an installed language plugin again; the language configuration is then automatically reloaded.

## Good to know

- A language plugin delivers more than just translated texts: it also brings country- and canton-specific data, as well as translated calendar rules and manuals, in the respective language.
- The four core languages are always present and cannot be uninstalled — only the 21 additional languages are optional.
- The coverage level of a language plugin can be below 100% if new features were added before the translation caught up; missing texts then fall back to English, the technical fallback language for all plugin languages.
