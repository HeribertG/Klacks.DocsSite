# Språkplugins

Klacks støtter fire kjernespråk fast i frontend — tysk, engelsk, fransk, italiensk — samt 21 andre språk som installerbare språkplugins (Language Packs), blant annet arabisk, tsjekkisk, dansk, gresk, spansk, finsk, hebraisk, indonesisk, japansk, koreansk, malayisk, norsk (bokmål), nederlandsk, polsk, portugisisk, rumensk, svensk, thai, vietnamesisk samt forenklet og tradisjonell kinesisk.

## Slik fungerer det

**Oversikt:** Under Innstillinger > Språkplugins ser du alle kjernespråkene (alltid aktive, kan ikke fjernes) samt de allerede installerte språkpluginene.

**Installere:** Lupe-knappen åpner markedsplass-dialogen, der du kan søke etter og installere flere språk.

**Avinstallere:** Via avinstaller-handlingen på raden fjerner du en installert språkplugin igjen; språkkonfigurasjonen lastes deretter automatisk på nytt.

## Greit å vite

- En språkplugin leverer mer enn bare oversettelsestekster: den bringer også med seg land- og kantonsspesifikke data samt oversatte kalenderregler og håndbøker på det aktuelle språket.
- De fire kjernespråkene er alltid til stede og kan ikke avinstalleres — bare de 21 tilleggsspråkene er valgfrie.
- Dekningsgraden til en språkplugin kan ligge under 100 % dersom nye funksjoner ble lagt til før oversettelsen fulgte etter; manglende tekster faller da tilbake til engelsk, som er det tekniske fallback-språket for alle plugin-språk.
