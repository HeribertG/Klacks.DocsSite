# 垃圾郵件規則

在「設定 > 垃圾郵件規則」中，您可以定義篩選規則，讓收件匣模組自動將收到的電子郵件標記為垃圾郵件 — 這些規則不僅在持續擷取新郵件時生效（IMAP 輪詢），也適用於對已收到郵件的事後重新評估。

## 運作方式

**建立規則：** 選擇規則類型 — 寄件者包含、寄件者網域、主旨包含或內容包含 — 輸入要比對的樣式，然後點選新增。

**管理規則：** 每條規則都可以透過核取方塊啟用或停用，而不需刪除；也可以透過垃圾桶按鈕永久移除。

## 小提醒

- 停用的規則仍會保留，但不再套用於新郵件或重新評估的郵件 — 適合用來暫時停用某條規則，而不必直接刪除。
- Klacks 的 ERP 訂單匯入與一般電子郵件收件匣模組共用同一個信箱（單一信箱架構）。因此，範圍設定過寬的規則（例如整個寄件者網域）理論上也可能將傳入的 ERP 訂單郵件誤判為垃圾郵件 — 請在啟用新規則前仔細檢查。
- 建議使用寄件者網域規則來封鎖整個不受歡迎的寄件組織，而不是為每個地址單獨建立「寄件者包含」規則。
