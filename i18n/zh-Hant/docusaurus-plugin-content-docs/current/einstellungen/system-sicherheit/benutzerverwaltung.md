# 使用者管理

在「設定 > 使用者管理」中，您管理的是人員登入 Klacks 所使用的**登入帳戶**（系統使用者）— 這與員工或地址管理不同，因為一個登入帳戶可以、但不是必須與某位人員的主檔資料卡片建立關聯。

## 運作方式

**新增使用者：**「+ 新增使用者」對話方塊會根據名字與姓氏（各至少 2 個字元）自動產生一個使用者名稱建議。電子郵件為必填欄位，且必須包含 @ 符號；唯有所有欄位皆有效填寫後才能儲存。建立時會自動產生一組密碼 — 若要讓本人自行設定密碼，請於建立後在該使用者所在列使用「重設密碼」。

**編輯使用者：** 於姓名列上按兩下即可開啟編輯對話方塊，您可以在其中修改使用者名稱 — 新增時產生的名稱只是一個建議值。

**重設密碼：** 使用者所在列中的「重設密碼」連結，會寄送一封電子郵件到其登記的地址，本人可透過該郵件自行設定新密碼。

**刪除使用者：** 垃圾桶按鈕會在刪除對話方塊中確認後，永久移除該登入帳戶。

**指派權限：** 透過該列中的下拉選單指派**主管（Supervisor）**或**管理員（Admin）**角色；變更會立即生效，不需另外點選儲存。這兩個角色具體擁有哪些權限，詳見「快速入門」中的「角色與權限」一節 — 此處僅涉及帳戶本身的管理。

## 小提醒

- 目前登入的使用者也會顯示在清單中，但無法在此編輯自己的資料、變更自己的權限或刪除自己。
- 登入帳戶獨立於人員／員工管理之外 — 例如您可以為一家外部 IT 公司建立帳戶，而不需要為此建立員工資料卡。
- 管理員無法直接設定密碼：新帳戶會取得一組自動產生的密碼，之後任何變更都必須透過「重設密碼」的電子郵件連結完成。
