# 語言外掛

Klacks 前端固定內建四種核心語言 — 德文、英文、法文、義大利文 — 另外還有 21 種語言可作為可安裝的語言外掛（Language Packs）使用，其中包括阿拉伯文、捷克文、丹麥文、希臘文、西班牙文、芬蘭文、希伯來文、印尼文、日文、韓文、馬來文、挪威文（巴克摩爾書面語）、荷蘭文、波蘭文、葡萄牙文、羅馬尼亞文、瑞典文、泰文、越南文，以及簡體中文與繁體中文。

## 運作方式

**總覽：** 在「設定 > 語言外掛」中，您可以看到所有核心語言（永遠啟用，無法移除）以及已安裝的語言外掛。

**安裝：** 點擊放大鏡按鈕會開啟市集對話框，您可以在其中搜尋並安裝其他語言。

**解除安裝：** 透過該列中的解除安裝動作，即可移除已安裝的語言外掛；語言設定隨後會自動重新載入。

## 小提醒

- 語言外掛提供的不只是翻譯文字：還會一併帶來各國家與行政區特有的資料，以及以該語言翻譯的日曆規則與操作手冊。
- 四種核心語言永遠存在，無法解除安裝 — 只有另外 21 種語言是可選的。
- 當新功能上線但翻譯尚未跟上時，語言外掛的涵蓋率可能低於 100%；缺少的文字會回退到英文，這是所有外掛語言共用的技術性備援語言。
