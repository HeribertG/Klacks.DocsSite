# 功能外掛

功能外掛為 Klacks 擴充了不屬於核心產品的選用功能模組 — 例如通訊（透過 Telegram、Slack、Viber、LINE、Microsoft Teams、簡訊、Signal、Threema、KakaoTalk、微信與 Zalo 收發訊息）或平面圖（附工作站標記與班別指派的視覺化空間平面圖編輯器）。

## 運作方式

**管理已安裝的外掛：** 在「設定 > 功能外掛」中，您可以看到已安裝外掛的清單。每一列都可以透過開關切換該外掛的啟用／停用狀態，而不需將其解除安裝，也可以透過解除安裝功能將其徹底移除。

**尋找新外掛：** 點選放大鏡按鈕會開啟市集對話方塊，您可以在其中搜尋其他可用的外掛並直接安裝。

## 小提醒

- 停用的外掛仍保持已安裝狀態，但對所有使用者都會被鎖定：相關頁面無法存取，相關 API 端點會回傳 404 回應，如同它們根本不存在一樣。
- 啟用／停用或安裝／解除安裝後，Klacks 會自動重新載入翻譯內容，讓新的外掛文字立即可見，不需要您手動重新整理頁面。
- 有些外掛會自帶專屬的設定卡片（例如通訊服務提供者設定）— 這些卡片只有在安裝完成後，才會出現在設定頁面的其他位置。
