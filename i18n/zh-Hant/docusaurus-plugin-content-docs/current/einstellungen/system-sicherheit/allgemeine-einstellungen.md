# 一般設定

在「設定 > 一般」中，您可以設定 Klacks 安裝實例的基本識別身分：應用程式名稱、圖示與標誌 — 同一設定頁面上還有另一個獨立區塊，用於設定排班表格的配色方案。

## 運作方式

**應用程式名稱：** 在此輸入的名稱會立即成為應用程式的瀏覽器標題，並出現在介面的其他位置，例如自動寄送的電子郵件中。

**圖示：** 您可以透過拖放或選擇檔案的方式，上傳自訂的 Favicon（`.ico` 或 `.png`），它會取代瀏覽器分頁中的預設圖示。點選「移除」可將其重設為 `favicon.ico`。

**標誌：** 同樣可以透過拖放或選擇檔案的方式，上傳自訂標誌（`.jpeg`、`.png` 或 `.gif`），Klacks 會依比例縮放以供顯示。點選「移除」可將其刪除。

### 排班表格的配色方案

在同一設定頁面往下捲動，您可以設定排班表格為各個儲存格上色所用的顏色 — 例如星期六、星期日、假日、國定假日、已封存（已結算）的期間、換班、津貼，以及邊框與字型顏色。每種顏色都可以透過色彩選擇器單獨修改；清單中會分別顯示名稱與目前選定的顏色。

## 小提醒

- 對名稱、圖示與標誌的變更會立即生效，不需另外點選儲存 — 這裡沒有附取消選項的表單。
- 用心設計的配色方案，有助於在排班表格中快速取得視覺定位，例如一眼分辨週末、假日與工作日的差異 — 對列數眾多的大型團隊特別有幫助。
- 這些顏色只會影響排班表格本身的顯示，不會影響介面的其他區域。
