# 自主等級

Klacksy 在需要詢問您之前，可以自主行動到什麼程度？您可以在「設定 > Klacksy 自主性」中依個人喜好設定 — 這項設定僅適用於您自己的使用者帳戶，並非對所有使用者統一生效。

## 運作方式

**四個等級：**

- **建議** — Klacksy 只會提出建議，每一項變更都需要您確認。
- **輔助** — 可復原的操作會直接執行，其餘操作則需要確認。
- **自主**（預設）— 除特別敏感的操作外，所有操作均直接執行。
- **完全自主** — 即使是多步驟計畫，也無需中途確認即可執行完畢。

**分級如何生效：** Klacksy 可能執行的每一項操作，內部都依風險分類：純粹讀取的操作一律立即執行；可復原的操作，或與特定情境綁定的操作（例如在測試排班中）至少需要「輔助」等級；不可復原的操作至少需要「自主」等級。敏感操作 — 使用者管理、權限設定，以及對自主性設定本身的變更 — **無論選擇哪個等級**，一律都需要明確確認。

**確認流程：** 若某項操作需要您的核准，Klacksy 不會立即執行，而是先將它擱置 5 分鐘，並摘要說明將會發生什麼事。只有當您以自己的話明確表示同意後，Klacksy 才會觸發這項被擱置的操作 — 它絕不會代替您自行確認。

## 小提醒

- 自主等級是依使用者帳戶設定的個人選項 — 每位使用者都可以選擇自己覺得合適的等級。
- 等級調高並不代表完全不再詢問：敏感操作永遠都需要確認。
- 系統預設啟用「自主」等級 — 這與 Klacksy 在推出自主等級功能之前的行為一致。
