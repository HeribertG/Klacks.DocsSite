# Klacksy 人格設定

Klacksy 沒有僵化、寫死在程式裡的個性 — 您可以在「設定 > Klacksy 人格」中，用自己的話定義 Klacksy 的行為方式、說話方式，以及面對各種情況時的反應。

## 運作方式

**六個文字欄位：** 人格設定分為六個部分，每個部分都有各自的自由文字欄位：

- **身分** — Klacksy 是誰？角色、目的與情境脈絡。
- **性格與動機** — 性格特質、基本態度，以及驅動 Klacksy 的動力。
- **語氣** — Klacksy 聽起來如何？正式或輕鬆，簡短或詳盡。
- **界線** — Klacksy 不可以做什麼？資料隱私、禁忌話題、敏感領域。
- **溝通風格** — 回答的架構、如何處理不確定性、如何追問。
- **價值觀與延續性** — 核心價值觀，以及 Klacksy 如何隨時間演變。

**儲存：** 每個欄位在失去焦點（離開該欄位）時，都會個別自動儲存 — 不需要另外按下儲存按鈕。變更會立即反映在與 Klacksy 的下一次對話中。

## 小提醒

- 人格設定屬於共用的 Klacksy 預設代理人 — 變更會影響**所有**使用者，而不只是您自己的帳戶。
- 這些欄位都是純自由文字 — 沒有格式規定；經驗顯示，簡短清楚的句子通常比冗長的修飾更可靠。
- 欄位留空也沒問題：Klacksy 屆時會採用內建的基本態度。部分其他內部行為細節則由系統自行維護，無法在此直接設定。
