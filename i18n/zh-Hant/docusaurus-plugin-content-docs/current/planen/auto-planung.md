# 自動排班：數秒生成排班表

排班工作中最繁瑣的部分 — 將人員分配到班次 — Klacks 只需按一下按鈕就能完成。

## 運作方式

一鍵自動排班的背後，是一套**遺傳演算法**：它會產生數千種排班方案，逐一評分，並持續將最優的方案組合演化 — 直到得出一個均衡的結果為止。硬性規則永遠優先，其餘目標則其次考量：

- **規則** — 勞動法規、休息時間、資格要求與可用時間皆不可協商：無論結果多麼出色，都不能違反硬性規則
- **覆蓋率** — 每個班次都會配置到所需的人數
- **公平性** — 工作時數平均分配，並儘可能讓每個人的班次順序依循早班 → 晚班 → 夜班的模式
- **連續性** — 盡可能讓員工留在自己熟悉的工作地點

您可以直接在排班表中，針對所選的時間範圍啟動自動排班。結果會以一般排班草稿的形式呈現：所有內容都還能手動調整，任何項目都不會未經確認就被鎖定。

## 小提醒

- 資格要求是真正的硬性限制：不具備所需資格的人員不會被排入班次。
- 自動排班會尊重既有的排班條目 — 因此您可以先手動設定固定的班次，再讓其餘部分自動填滿。
- 若對結果不滿意，重新排班一次即可：演算法會找出同樣理想的替代方案。
