期間結算是您進行月結作業的工作平台：您可以放行個別日期、結束整個期間並將其封存 — 封存後的資料，就是報表分析與匯出的正式依據。

## 運作方式

此頁面分為三個頁籤：**期間**、**匯出**與**紀錄**。

1. **選擇期間：** 您可透過起始與結束日期界定範圍，並依需求篩選特定群組（或「所有群組」）。給付週期可選擇每週、每兩週、每月或自訂。
2. **封存：** 身為管理員，您可以點選*封存期間*（或*全部封存*）來結束這些條目。確認對話方塊會先顯示將有多少筆條目受到影響。
3. **匯出：** 您可以從已封存的期間產生匯出檔案 — 格式包括 CSV、JSON、XML、DATEV 或 BMD NTCS，語言與貨幣皆可選擇；如有需要，也可以針對特定訂單匯出。*匯出*頁籤會列出所有已產生的檔案，包含時間範圍、建立日期與建立者。
4. **重新開啟：** 身為管理員，您可以視需要重新開啟已封存的期間（*重新開啟期間*或*全部解除封存*）。此時**必須填寫理由**。
5. **紀錄：** 每一次封存與重新開啟都會記錄在*紀錄*頁籤中 — 包含動作、時間點與執行者。

## 小提醒

- 尚未結束的班次**不會**被匯出。若只有部分班次已結束，Klacks 會提出警告（「y 個班次中有 x 個已結束」）。
- 若某個時間範圍已產生過匯出檔案，之後再有異動時 Klacks 會提出警告：這可能會造成與會計帳目之間的落差。
- 重新開啟已封存的期間可能會影響薪資結算 — 因此 Klacks 會再次確認，並要求填寫理由。
- 您可以透過左側導覽列中的結算圖示進入期間結算。
