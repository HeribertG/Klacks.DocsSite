# 排班網格：您的互動式時間矩陣

這是每天工作的核心：一個效能強大的時間矩陣，列代表員工或客戶，欄代表日期。

## 運作方式

- **拖放操作：** 您可以直接用滑鼠把班次從一個人拖到另一個人身上；透過填滿控點，還能把某個條目複製到同一列的其他日期。
- **即時性：** 異動會立即顯示給所有開啟同一份排班表的人。兩位排班人員可以同時作業 — 系統會偵測衝突，而不會讓異動被悄悄覆蓋。
- **一目了然：** 資格要求與鎖定狀態會直接在網格中標示出來，衝突則由 Klacks 另外在專屬的總覽中提示 — 只要某項指派不被允許，您立刻就能看到。
- **快速：** 這個網格專為大型排班表打造；即使數百位員工、整整一個月的排班量，操作起來依然流暢。

## 小提醒

- 超出某人群組歸屬或合約期間的儲存格會顯示為灰色，無法排班 — 從根本上避免不小心排出錯誤的班次。
- 網格與自動排班可以搭配使用：先讓系統自動填入，再手動微調細節。
- 依各條目所處的核准階段不同，編輯功能可能會被鎖定 — 詳見[核准流程：從草稿到具約束力的排班表](../planen/freigabe-workflow.md)。
