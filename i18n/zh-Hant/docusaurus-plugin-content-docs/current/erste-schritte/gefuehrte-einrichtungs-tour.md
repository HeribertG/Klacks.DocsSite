# 引導式設定導覽

剛開始使用 Klacks 嗎？您不需要自己摸索：內建助手 Klacksy 會一步步帶您完成基礎設定。

## 運作方式

導覽可直接在與 Klacksy 的對話中啟動（例如輸入「帶我看看這個應用程式」，或點擊初次啟動時出現的建議）。接著 Klacksy 會逐站帶您走過最重要的功能區域：

- Klacksy 用簡短幾句話**說明每個區域** — 從公司基本設定與地址、日曆、使用者管理、群組，到員工、班次、可用時間、缺勤、假日、期間結算、電子郵件設定以及 Klacksy／外掛設定，總共 16 個站點。
- 對應的**導覽圖示會在側邊欄中閃爍**，讓您隨時清楚自己目前所在的位置。
- 節奏由您掌控：可以繼續、跳過某一站，或隨時結束導覽。

如此一來，您就能一步步建立起對整個應用程式的完整概觀，完全不需要翻閱操作手冊。

## 小提醒

- 這份導覽可以重複進行任意次數 — 也適合用來帶新加入的團隊成員上手。
- 在每一站您都可以向 Klacksy 提出問題；他會依據目前頁面的內容作答。
- 體驗導覽最快的方式就是在 Playground 中：登入、開啟 Klacksy、開始輸入。
