# 缺勤一目了然

休假、病假、進修、兵役 — 誰在什麼時候缺勤，在 Klacks 中會以清楚的甘特圖呈現，涵蓋整年度。

## 運作方式

- 每筆缺勤都會在該人員的時間軸上顯示為一段彩色長條 — 時間重疊與集中缺勤一眼就能看出。
- **自訂缺勤類型**由您自行定義，包含顏色與行為方式（支薪／不支薪、是否計入工時）。
- 一鍵即可匯出 **PDF** — 適合張貼公告或存檔留用。
- 缺勤會直接影響排班：缺勤人員不會被[自動排班：秒速產生班表](../planen/auto-planung.md)排入班次，而在[排班網格：您的互動式時間矩陣](../planen/plan-raster.md)中，衝突也會立即顯示出來。

## 小提醒

- 甘特圖總覽非常適合團隊的休假規劃：整個夏季的人力吃緊時段一目了然。
- 若有人臨時缺勤，Klacks 會立即顯示具備資格且有空的替補人選 — 不需要推翻整份排班表。
- 缺勤數值（例如一整天或半天的休假登記）會以總和的形式計入報表中的工時統計。
