萬年曆會自動計算每一年的假日 — 您只需定義一次規則，之後 Klacks 便會知曉未來所有年份的假日。

## 運作方式

每個假日都是以一條簡短的規則來描述。基本上分為兩種類型：

- **固定日期**，格式為 `MM/DD`：`01/01` 是元旦，`08/01` 是瑞士國慶日，`12/25` 是聖誕節。
- **與復活節相關的日期**，格式為 `EASTER+XX` 或 `EASTER-XX`：復活節日期會依高斯復活節公式自動計算，假日則以與該日期的間隔天數表示。例如：`EASTER-02` = 耶穌受難日，`EASTER+01` = 復活節星期一，`EASTER+39` = 耶穌升天節，`EASTER+50` = 聖靈降臨節星期一，`EASTER+60` = 聖體節。

此外還有**帶星期偏移的固定日期**（`MM/DD+XX+WW` 或 `MM/DD+XX-WW`），用於描述例如「十一月的第一個星期四」（`11/01+00+TH`）或「五月的最後一個星期一」（`05/25+00-MO`）這類假日。

透過 **SubRules（子規則）**，當某個假日恰好落在特定星期幾時，可以自動順延 — 例如 `SA+2;SU+1`：若該日落在星期六或星期日，就會順延到星期一。多條規則之間以 `;` 分隔。

每條規則除此之外還可輸入名稱與說明（多語言）、國家與州／邦，以及**法定假日**與**支薪**兩個核取方塊。

## 小提醒

- 星期縮寫使用英文：`SU`、`MO`、`TU`、`WE`、`TH`、`FR`、`SA`。
- SubRules 主要是用來把落在週末的假日順延到下一個工作日。
- **支薪**核取方塊與薪資計算相關。
- 透過國家與州／邦可以處理地區差異 — 例如耶穌受難日在瑞士並非所有州都適用（例外：VS、TI）。
- 預先安裝的國家與瑞士各州曆法套件皆已鎖定，無法刪除；您自行建立的日曆選擇則可隨時刪除。
