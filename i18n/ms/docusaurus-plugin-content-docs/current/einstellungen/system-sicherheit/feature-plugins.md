# Plugin Ciri

Plugin Ciri melanjutkan Klacks dengan kawasan fungsi pilihan yang bukan sebahagian daripada produk teras — contohnya Messaging (penghantaran dan penerimaan melalui Telegram, Slack, Viber, LINE, Microsoft Teams, SMS, Signal, Threema, KakaoTalk, WeChat dan Zalo) atau Floor Plan (editor pelan ruang visual dengan penanda tempat kerja dan pemetaan Syif).

## Cara ia berfungsi

**Mengurus Plugin Terpasang:** Di bawah Tetapan > Plugin Ciri anda melihat senarai plugin yang telah dipasang. Bagi setiap baris, anda boleh menghidupkan atau mematikan plugin melalui togol (diaktifkan/dinyahaktifkan) tanpa menyahpasangnya, atau mengalih keluarnya sepenuhnya melalui tindakan Nyahpasang.

**Mencari Plugin Baharu:** Butang kaca pembesar membuka dialog Marketplace, di mana anda boleh mencari plugin lain yang tersedia dan memasangnya terus.

## Perkara Berguna

- Plugin yang dinyahaktifkan kekal terpasang, tetapi disekat untuk semua pengguna: laman yang berkaitan tidak boleh diakses dan titik akhir API yang berkaitan memberikan respons 404, seolah-olah ia tidak wujud.
- Selepas mengaktifkan/menyahaktifkan atau memasang/menyahpasang, Klacks memuat semula terjemahan secara automatik supaya teks plugin baharu segera kelihatan tanpa anda perlu memuat semula halaman.
- Sesetengah plugin membawa kad tetapan tersendiri (contohnya konfigurasi Pembekal Messaging) — ini hanya muncul di tempat lain dalam Tetapan selepas pemasangan.
