# Selamat Datang ke Dokumentasi Klacks

**Klacks** ialah platform open-source untuk perancangan penggunaan tenaga kerja bagi operasi bersyif dan bertugas: rancang Tugasan secara automatik, optimumkan Perjalanan, kawal melalui Pembantu AI — On-Premise, dengan model AI pilihan anda. Untuk Spitex, hospital, syarikat keselamatan, pembersihan dan logistik.

Wiki ini menerangkan cara Klacks berfungsi dalam bahagian-bahagian kecil. Setiap halaman boleh dibaca dalam beberapa minit — dan semuanya boleh dicuba serta-merta dalam Playground awam.
